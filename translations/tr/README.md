# LangChain4j Yeni Başlayanlar İçin

LangChain4j ve Azure OpenAI GPT-5.2 ile AI uygulamaları geliştirmeye yönelik temel sohbetten AI ajanlarına kadar bir kurs.

**LangChain4j'ye yeni misiniz?** Temel terimler ve kavramlar için [Sözlük](docs/GLOSSARY.md)'e göz atın.

## İçindekiler

1. [Hızlı Başlangıç](00-quick-start/README.md) - LangChain4j ile başlayın
2. [Giriş](01-introduction/README.md) - LangChain4j temellerini öğrenin
3. [Prompt Mühendisliği](02-prompt-engineering/README.md) - Etkili prompt tasarımında ustalaşın
4. [RAG (Retrieval-Augmented Generation)](03-rag/README.md) - Akıllı bilgi tabanlı sistemler inşa edin
5. [Araçlar](04-tools/README.md) - Harici araçları ve basit asistanları entegre edin
6. [MCP (Model Context Protocol)](05-mcp/README.md) - Model Context Protocol (MCP) ve Agentic modüllerle çalışın
---

## Öğrenme Yolu

> **Hızlı Başlangıç**

1. Bu depoyu GitHub hesabınıza fork edin
2. **Code** → **Codespaces** sekmesine tıklayın → **...** → **Seçeneklerle yeni...** ye tıklayın
3. Varsayılan ayarları kullanın – bu kurs için oluşturulmuş Geliştirme konteynerini seçecektir
4. **Codespace oluştur** butonuna tıklayın
5. Ortamın hazır olması için 5-10 dakika bekleyin
6. Başlamak için doğrudan [Hızlı Başlangıç](./00-quick-start/README.md)’a geçin!

Modülleri tamamladıktan sonra, LangChain4j test konseptlerini uygulamalı olarak görmek için [Test Rehberi](docs/TESTING.md)’ni inceleyin.

> **Not:** Bu eğitim hem GitHub Modelleri hem de Azure OpenAI kullanır. [Hızlı Başlangıç](00-quick-start/README.md) modülü GitHub Modelleri kullanır (Azure aboneliği gerekmez), modüller 1-5 ise Azure OpenAI kullanır. Henüz yoksa bir [ÜCRETSİZ Azure hesap](https://aka.ms/azure-free-account) ile başlayın.


## GitHub Copilot ile Öğrenme

Hızlıca kodlamaya başlamak için, bu projeyi GitHub Codespace ya da sağlanan devcontainer ile yerel IDE'nizde açın. Bu derste kullanılan devcontainer, AI eşliğinde programlama için önceden yapılandırılmış GitHub Copilot içerir.

Her kod örneği, GitHub Copilot'a sorabileceğiniz önerilen soruları içerir, böylece anlayışınızı derinleştirebilirsiniz. 💡/🤖 işaretleri aşağıdaki yerlerde bulunur:

- **Java dosya başlıkları** - Her örneğe özel sorular
- **Modül README'leri** - Kod örneklerinden sonra keşif soruları

**Nasıl kullanılır:** Herhangi bir kod dosyasını açın ve Copilot'a önerilen soruları sorun. Kod tabanının tüm bağlamına sahiptir, açıklayabilir, genişletebilir ve alternatifler önerebilir.

Daha fazla öğrenmek ister misiniz? [AI Eşli Programlama için Copilot](https://aka.ms/GitHubCopilotAI)'a göz atın.
