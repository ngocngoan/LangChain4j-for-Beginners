# LangChain4j untuk Pemula

Satu kursus untuk membina aplikasi AI dengan LangChain4j dan Azure OpenAI GPT-5, dari sembang asas hingga agen AI.

**Baru kepada LangChain4j?** Lihat [Glosari](docs/GLOSSARY.md) untuk definisi istilah dan konsep utama.

## Jadual Kandungan

1. [Mula Cepat](00-quick-start/README.md) - Mula dengan LangChain4j
2. [Pengenalan](01-introduction/README.md) - Pelajari asas-asas LangChain4j
3. [Kejuruteraan Prompt](02-prompt-engineering/README.md) - Kuasai reka bentuk prompt yang berkesan
4. [RAG (Generation Augmented Retrieval)](03-rag/README.md) - Bina sistem berasaskan pengetahuan yang bijak
5. [Alat](04-tools/README.md) - Integrasi alat luaran dan pembantu ringkas
6. [MCP (Protokol Konteks Model)](05-mcp/README.md) - Bekerja dengan Protokol Konteks Model (MCP) dan modul Agens
---

## Laluan Pembelajaran

> **Mula Cepat**

1. Fork repositori ini ke akaun GitHub anda
2. Klik **Code** → tab **Codespaces** → **...** → **Baru dengan pilihan...**
3. Gunakan tetapan lalai – ini akan memilih kontena Pembangunan yang dibuat untuk kursus ini
4. Klik **Buat codespace**
5. Tunggu 5-10 minit untuk persekitaran bersedia
6. Terus ke [Mula Cepat](./00-quick-start/README.md) untuk mula!

Selepas menyelesaikan modul, terokai [Panduan Ujian](docs/TESTING.md) untuk melihat konsep ujian LangChain4j secara langsung.

> **Nota:** Latihan ini menggunakan kedua-dua Model GitHub dan Azure OpenAI. Modul [Mula Cepat](00-quick-start/README.md) menggunakan Model GitHub (tidak memerlukan langganan Azure), manakala modul 1-5 menggunakan Azure OpenAI. Mulakan dengan [akaun Azure PERCUMA](https://aka.ms/azure-free-account) jika anda belum memilikinya.


## Belajar dengan GitHub Copilot

Untuk memulakan pengaturcaraan dengan cepat, buka projek ini dalam GitHub Codespace atau IDE tempatan anda dengan devcontainer yang disediakan. Devcontainer yang digunakan dalam kursus ini telah dikonfigurasikan dengan GitHub Copilot untuk pengaturcaraan AI berpasukan.

Setiap contoh kod termasuk soalan yang disyorkan yang boleh anda ajukan kepada GitHub Copilot untuk mendalami pemahaman anda. Cari petunjuk 💡/🤖 dalam:

- **Header fail Java** - Soalan khusus untuk setiap contoh
- **Modul README** - Petunjuk penerokaan selepas contoh kod

**Cara menggunakan:** Buka mana-mana fail kod dan tanya Copilot soalan yang disyorkan. Ia mempunyai konteks penuh kod tersebut dan boleh menerangkan, mengembangkan, serta mencadangkan alternatif.

Ingin belajar lebih lanjut? Lihat [Copilot untuk Pengaturcaraan AI Berpasukan](https://aka.ms/GitHubCopilotAI).
