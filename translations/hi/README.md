# शुरुआती के लिए LangChain4j

LangChain4j और Azure OpenAI GPT-5.2 के साथ AI एप्लिकेशन बनाने के लिए एक कोर्स, बुनियादी चैट से लेकर AI एजेंट्स तक।

**LangChain4j में नए हैं?** प्रमुख शब्दों और अवधारणाओं के लिए [शब्दावली](docs/GLOSSARY.md) देखें।

## सामग्री सूची

1. [त्वरित शुरूआत](00-quick-start/README.md) - LangChain4j के साथ शुरुआत करें
2. [परिचय](01-introduction/README.md) - LangChain4j के मूल बातें जानें
3. [प्रॉम्प्ट इंजीनियरिंग](02-prompt-engineering/README.md) - प्रभावी प्रॉम्प्ट डिज़ाइन में महारत हासिल करें
4. [RAG (रिकवरी-अगमेंटेड जनरेशन)](03-rag/README.md) - बुद्धिमान ज्ञान आधारित सिस्टम बनाएं
5. [उपकरण](04-tools/README.md) - बाहरी उपकरण और सरल सहायक एकीकृत करें
6. [MCP (मॉडल संदर्भ प्रोटोकॉल)](05-mcp/README.md) - मॉडल संदर्भ प्रोटोकॉल (MCP) और एजेंटिक मॉड्यूल के साथ काम करें

## सीखने का मार्ग

> **त्वरित शुरूआत**

1. इस रिपॉजिटरी को अपने GitHub अकाउंट में Fork करें
2. क्लिक करें **Code** → **Codespaces** टैब → **...** → **New with options...**
3. डिफ़ॉल्ट का उपयोग करें – यह इस कोर्स के लिए बनाए गए विकास कंटेनर का चयन करेगा
4. क्लिक करें **Create codespace**
5. वातावरण के तैयार होने के लिए 5-10 मिनट प्रतीक्षा करें
6. सीधे [त्वरित शुरूआत](./00-quick-start/README.md) पर जाएं और शुरुआत करें!

मॉड्यूल पूरा करने के बाद, LangChain4j परीक्षण अवधारणाओं को क्रियान्वित में देखने के लिए [परीक्षण गाइड](docs/TESTING.md) को एक्सप्लोर करें।

> **ध्यान दें:** यह प्रशिक्षण दोनों GitHub मॉडल और Azure OpenAI का उपयोग करता है। [त्वरित शुरूआत](00-quick-start/README.md) मॉड्यूल GitHub मॉडल का उपयोग करता है (Azure सदस्यता आवश्यक नहीं), जबकि मॉड्यूल 1-5 Azure OpenAI का उपयोग करते हैं। यदि आपके पास नहीं है तो [मुफ्त Azure खाता](https://aka.ms/azure-free-account) के साथ शुरू करें।


## GitHub Copilot के साथ सीखना

त्वरित कोडिंग शुरू करने के लिए, इस प्रोजेक्ट को GitHub Codespace या अपने स्थानीय IDE में devcontainer के साथ खोलें। इस कोर्स में प्रयुक्त devcontainer पहले से GitHub Copilot के साथ AI पेयर प्रोग्रामिंग के लिए कॉन्फ़िगर्ड है।

प्रति कोड उदाहरण में GitHub Copilot से पूछने के लिए सुझाए गए प्रश्न शामिल हैं जिससे आपकी समझ गहरी हो सके। 💡/🤖 संकेत देखें:

- **Java फ़ाइल हेडर** - प्रत्येक उदाहरण के लिए विशिष्ट प्रश्न
- **मॉड्यूल README** - कोड उदाहरणों के बाद एक्सप्लोरेशन प्रॉम्प्ट

**कैसे उपयोग करें:** कोई भी कोड फ़ाइल खोलें और Copilot से सुझाए गए प्रश्न पूछें। इसे कोडबेस का पूरा संदर्भ है और यह समझा सकता है, विस्तार कर सकता है, और विकल्प सुझा सकता है।

और अधिक जानना चाहते हैं? देखें [AI पेयर प्रोग्रामिंग के लिए Copilot](https://aka.ms/GitHubCopilotAI)।
