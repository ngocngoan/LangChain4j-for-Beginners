# LangChain4j voor Beginners

Een cursus voor het bouwen van AI-toepassingen met LangChain4j en Azure OpenAI GPT-5.2, van basischat tot AI-agenten.

**Nieuw met LangChain4j?** Bekijk de [Glossary](docs/GLOSSARY.md) voor definities van belangrijke termen en concepten.

## Inhoudsopgave

1. [Quick Start](00-quick-start/README.md) - Begin met LangChain4j
2. [Introductie](01-introduction/README.md) - Leer de basisprincipes van LangChain4j
3. [Prompt Engineering](02-prompt-engineering/README.md) - Beheers effectieve promptontwerp
4. [RAG (Retrieval-Augmented Generation)](03-rag/README.md) - Bouw intelligente kennisgebaseerde systemen
5. [Tools](04-tools/README.md) - Integreer externe tools en eenvoudige assistenten
6. [MCP (Model Context Protocol)](05-mcp/README.md) - Werk met het Model Context Protocol (MCP) en agentmodules
---

## Leerpad

> **Quick Start**

1. Fork deze repository naar je GitHub-account
2. Klik op **Code** → tabblad **Codespaces** → **...** → **Nieuw met opties...**
3. Gebruik de standaardinstellingen – dit selecteert de ontwikkelcontainer die voor deze cursus is gemaakt
4. Klik op **Maak codespace aan**
5. Wacht 5-10 minuten totdat de omgeving klaar is
6. Ga direct naar [Quick Start](./00-quick-start/README.md) om te beginnen!

Na afronding van de modules, verken de [Testgids](docs/TESTING.md) om LangChain4j testconcepten in actie te zien.

> **Opmerking:** Deze training gebruikt zowel GitHub Models als Azure OpenAI. De [Quick Start](00-quick-start/README.md) module gebruikt GitHub Models (geen Azure-abonnement vereist), terwijl modules 1-5 Azure OpenAI gebruiken. Begin met een [GRATIS Azure-account](https://aka.ms/azure-free-account) als je dat nog niet hebt.


## Leren met GitHub Copilot

Om snel met coderen te beginnen, open dit project in een GitHub Codespace of je lokale IDE met de meegeleverde devcontainer. De devcontainer die in deze cursus wordt gebruikt, is vooraf geconfigureerd met GitHub Copilot voor AI-peerprogrammering.

Elk codevoorbeeld bevat voorgestelde vragen die je GitHub Copilot kunt stellen om je begrip te verdiepen. Let op de 💡/🤖 prompts in:

- **Java-bestand headers** - Vragen specifiek voor elk voorbeeld
- **Module READMEs** - Verkenningsprompts na codevoorbeelden

**Hoe te gebruiken:** Open elk codebestand en stel Copilot de voorgestelde vragen. Hij heeft volledige context van de codebasis en kan uitleg geven, uitbreiden en alternatieven voorstellen.

Wil je meer leren? Bekijk [Copilot voor AI Peerprogrammering](https://aka.ms/GitHubCopilotAI).
