## Leertraject

**Nieuw bij LangChain4j?** Bekijk de [Glossary](docs/GLOSSARY.md) voor definities van sleuteltermen en concepten.

> **Quick Start**

1. Fork deze repository naar je GitHub-account
2. Klik op **Code** → tabblad **Codespaces** → **...** → **Nieuw met opties...**
3. Gebruik de standaardinstellingen – hiermee wordt de ontwikkelcontainer geselecteerd die voor deze cursus is gemaakt
4. Klik op **Codespace aanmaken**
5. Wacht 5-10 minuten totdat de omgeving klaar is
6. Ga direct naar [Quick Start](./00-quick-start/README.md) om aan de slag te gaan!

Na het voltooien van de modules, bekijk de [Testing Guide](docs/TESTING.md) om LangChain4j-testconcepten in actie te zien.

> **Opmerking:** Deze training gebruikt zowel GitHub Models als Azure OpenAI. De [Quick Start](00-quick-start/README.md) module gebruikt GitHub Models (geen Azure-abonnement vereist), terwijl modules 1-5 Azure OpenAI gebruiken. Begin met een [GRATIS Azure-account](https://aka.ms/azure-free-account) als je er nog geen hebt.


## Leren met GitHub Copilot

Om snel te beginnen met coderen, open je dit project in een GitHub Codespace of je lokale IDE met de meegeleverde devcontainer. De devcontainer die in deze cursus wordt gebruikt, is vooraf geconfigureerd met GitHub Copilot voor AI-gepaarde programmering.

Elke codevoorbeeld bevat voorgestelde vragen die je GitHub Copilot kunt stellen om je begrip te verdiepen. Zoek naar de 💡/🤖 prompts in:

- **Java-bestandsheaders** - Specifieke vragen voor elk voorbeeld
- **Module READMEs** - Verkenningsprompts na codevoorbeelden

**Hoe te gebruiken:** Open elk codebestand en stel Copilot de voorgestelde vragen. Het heeft volledige context van de codebase en kan uitleggen, uitbreiden en suggesties geven voor alternatieven.

Wil je meer leren? Bekijk [Copilot voor AI-gepaarde programmering](https://aka.ms/GitHubCopilotAI).
