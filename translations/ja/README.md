## 学習パス

> **クイックスタート**

1. このリポジトリをGitHubアカウントにフォークする
2. **Code** → **Codespaces** タブ → **...** → **New with options...** をクリック
3. デフォルトを使用 – このコース用に作成された開発コンテナが選択されます
4. **Create codespace** をクリック
5. 環境が整うまで5〜10分待つ
6. すぐに[クイックスタート](./00-quick-start/README.md)に進んで始めましょう！

モジュールを修了後、[テストガイド](docs/TESTING.md)でLangChain4jのテスト概念を確認しましょう。

> **注意:** 本トレーニングではGitHub ModelsとAzure OpenAIの両方を使用します。[クイックスタート](00-quick-start/README.md)モジュールはGitHub Models（Azureサブスクリプション不要）を使用し、モジュール1−5はAzure OpenAIを使用します。アカウントをお持ちでなければ[無料のAzureアカウント](https://aka.ms/azure-free-account)で始めましょう。


## GitHub Copilotを使った学習

すぐにコーディングを始めるには、このプロジェクトをGitHub Codespaceまたは提供されたdevcontainer付きのローカルIDEで開いてください。このコースで使用するdevcontainerは、AIペアプログラミング用にGitHub Copilotが事前設定されています。

各コード例には、GitHub Copilotに質問して理解を深めるための推奨質問が含まれています。💡/🤖のプロンプトを探してください：

- **Javaファイルのヘッダー** - 各例に特化した質問
- **モジュールのREADME** - コード例の後の探求用プロンプト

**使い方:** 任意のコードファイルを開き、推奨質問をCopilotに聞いてみてください。コードベース全体のコンテキストを把握しており、説明、拡張案、代案提案が可能です。

さらに知りたい方は[AIペアプログラミング向けCopilot](https://aka.ms/GitHubCopilotAI)をご覧ください。
