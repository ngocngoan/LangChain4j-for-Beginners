# نئے صارفین کے لیے LangChain4j

LangChain4j اور Azure OpenAI GPT-5.2 کے ساتھ AI ایپلیکیشنز بنانے کا کورس، بنیادی چیٹ سے لے کر AI ایجنٹس تک۔

**LangChain4j میں نئے ہیں؟** کلیدی اصطلاحات اور تصورات کی تعریف کے لیے [گلوسری](docs/GLOSSARY.md) دیکھیں۔

## فہرست مضامین

1. [جلدی آغاز](00-quick-start/README.md) - LangChain4j کے ساتھ شروع کریں
2. [تعارف](01-introduction/README.md) - LangChain4j کے بنیادی اصول سیکھیں
3. [پرومپٹ انجینئرنگ](02-prompt-engineering/README.md) - مؤثر پرومپٹ ڈیزائن میں مہارت حاصل کریں
4. [RAG (ریٹریول-آگمینٹڈ جنریشن)](03-rag/README.md) - ذہین علم پر مبنی نظام بنائیں
5. [آلات](04-tools/README.md) - بیرونی آلات اور آسان معاونین کو مربوط کریں
6. [MCP (ماڈل کانٹیکسٹ پروٹوکول)](05-mcp/README.md) - ماڈل کانٹیکسٹ پروٹوکول (MCP) اور ایجنٹک ماڈیولز کے ساتھ کام کریں

## تعلیمی راستہ

> **جلدی آغاز**

1. اس ریپوزیٹری کو اپنے GitHub اکاؤنٹ میں فورک کریں
2. کلک کریں **کوڈ** → **کوڈ اسپیسز** ٹیب → **...** → **نیا اختیارات کے ساتھ...**
3. ڈیفالٹس استعمال کریں – یہ اس کورس کے لیے بنایا گیا ڈویلپمنٹ کنٹینر منتخب کرے گا
4. کلک کریں **کوڈ اسپیس بنائیں**
5. ماحول کے تیار ہونے تک 5-10 منٹ انتظار کریں
6. شروع کرنے کے لیے سیدھے [جلدی آغاز](./00-quick-start/README.md) پر جائیں!

ماڈیولز مکمل کرنے کے بعد، LangChain4j ٹیسٹنگ تصورات کو عملی طور پر دیکھنے کے لیے [ٹیسٹنگ گائیڈ](docs/TESTING.md) دریافت کریں۔

> **نوٹ:** یہ تربیت GitHub ماڈلز اور Azure OpenAI دونوں استعمال کرتی ہے۔ [جلدی آغاز](00-quick-start/README.md) ماڈیول GitHub ماڈلز استعمال کرتا ہے (کوئی Azure سبسکرپشن درکار نہیں)، جب کہ ماڈیولز 1-5 Azure OpenAI استعمال کرتے ہیں۔ اگر آپ کے پاس Azure اکاؤنٹ نہیں ہے تو [مفت Azure اکاؤنٹ](https://aka.ms/azure-free-account) کے ساتھ شروع کریں۔


## GitHub Copilot کے ساتھ تعلیم

جلدی کوڈنگ شروع کرنے کے لیے، اس پروجیکٹ کو GitHub Codespace میں یا اپنے مقامی آئی ڈی ای میں فراہم کردہ devcontainer کے ساتھ کھولیں۔ اس کورس میں استعمال ہونے والا devcontainer GitHub Copilot کے ساتھ AI مشترکہ پروگرامنگ کے لیے پہلے سے تشکیل شدہ ہے۔

ہر کوڈ مثال میں وہ سوالات شامل ہیں جو آپ GitHub Copilot سے پوچھ سکتے ہیں تاکہ اپنی سمجھ کو گہرا کریں۔ 💡/🤖 اشارے تلاش کریں:

- **جاوا فائل ہیڈرز** - ہر مثال کے مخصوص سوالات
- **ماڈیول ریڈمز** - کوڈ مثالوں کے بعد تلاش کے اشارے

**استعمال کا طریقہ:** کوئی بھی کوڈ فائل کھولیں اور Copilot سے تجویز کردہ سوالات پوچھیں۔ اسے کوڈ بیس کا مکمل سیاق و سباق معلوم ہے اور یہ وضاحت کر سکتا ہے، بڑھا سکتا ہے، اور متبادل تجویز کر سکتا ہے۔

مزید سیکھنا چاہتے ہیں؟ دیکھیں [AI مشترکہ پروگرامنگ کے لیے Copilot](https://aka.ms/GitHubCopilotAI)۔
