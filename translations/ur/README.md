# نو آموزوں کے لیے LangChain4j

LangChain4j اور Azure OpenAI GPT-5.2 کے ساتھ AI ایپلیکیشنز بنانے کا ایک کورس، بنیادی چیٹ سے لے کر AI ایجنٹس تک۔

**LangChain4j میں نئے ہیں؟** کلیدی اصطلاحات اور تصورات کی تعریفات کے لیے [گلاسری](docs/GLOSSARY.md) دیکھیں۔

## فہرست مضامین

1. [فوری آغاز](00-quick-start/README.md) – LangChain4j کے ساتھ شروع کریں
2. [تعارف](01-introduction/README.md) – LangChain4j کی بنیادی باتیں سیکھیں
3. [پرومپٹ انجینئرنگ](02-prompt-engineering/README.md) – مؤثر پرومپٹ ڈیزائن میں مہارت حاصل کریں
4. [RAG (ریکویل آگمینٹڈ جنریشن)](03-rag/README.md) – ذہین علم پر مبنی نظام تیار کریں
5. [ٹولز](04-tools/README.md) – بیرونی اوزار اور سادہ معاونات کو شامل کریں
6. [MCP (ماڈل کانٹیکسٹ پروٹوکول)](05-mcp/README.md) – ماڈل کانٹیکسٹ پروٹوکول (MCP) اور ایجنٹک ماڈیولز کے ساتھ کام کریں

## تعلیمی راستہ

> **فوری آغاز**

1. اس ریپوزٹری کو اپنے GitHub اکاؤنٹ پر فورک کریں
2. **Code** → **Codespaces** ٹیب → **...** → **New with options...** پر کلک کریں
3. ڈیفالٹس استعمال کریں – یہ اس کورس کے لیے بنایا گیا ڈیویلپمنٹ کنٹینر منتخب کرے گا
4. **Create codespace** پر کلک کریں
5. ماحول تیار ہونے کے لیے 5-10 منٹ انتظار کریں
6. شروع کرنے کے لیے سیدھا [فوری آغاز](./00-quick-start/README.md) پر جائیں!

ماڈیولز مکمل کرنے کے بعد، LangChain4j کی جانچ کے تصورات کو عملی طور پر دیکھنے کے لیے [ٹیسٹنگ گائیڈ](docs/TESTING.md) دریافت کریں۔

> **نوٹ:** یہ تربیت GitHub ماڈلز اور Azure OpenAI دونوں استعمال کرتی ہے۔ [فوری آغاز](00-quick-start/README.md) ماڈیول GitHub ماڈلز استعمال کرتا ہے (Azure سبسکرپشن ضروری نہیں)، جبکہ ماڈیولز 1-5 Azure OpenAI استعمال کرتے ہیں۔ اگر آپ کے پاس اکاؤنٹ نہیں ہے تو [مفت Azure اکاؤنٹ](https://aka.ms/azure-free-account) کے ساتھ شروع کریں۔


## GitHub کوپائلٹ کے ساتھ سیکھنا

تیزی سے کوڈنگ شروع کرنے کے لیے اس پروجیکٹ کو GitHub Codespace یا آپ کے مقامی IDE میں مہیا کردہ ڈیول کنٹینر کے ساتھ کھولیں۔ اس کورس میں استعمال ہونے والا ڈیول کنٹینر پہلے سے GitHub کوپائلٹ کے لیے کنفیگرڈ ہے جو AI کے ساتھ جوڑی پروگرامنگ کے لیے ہے۔

ہر کوڈ مثال میں وہ سوالات شامل ہیں جو آپ GitHub کوپائلٹ سے پوچھ سکتے ہیں تاکہ آپ کی سمجھ میں گہرائی آئے۔ 💡/🤖 پرامپٹس کو تلاش کریں:

- **جاوا فائل ہیڈرز** - ہر مثال کے مخصوص سوالات
- **ماڈیول READMEs** - کوڈ مثالوں کے بعد تلاش کے اشارے

**استعمال کا طریقہ:** کسی بھی کوڈ فائل کو کھولیں اور کوپائلٹ سے تجویز کیے گئے سوالات پوچھیں۔ کوپائلٹ کوڈبیس کا مکمل سیاق و سباق رکھتا ہے اور وضاحت، توسیع، اور متبادل تجویز کر سکتا ہے۔

مزید سیکھنا چاہتے ہیں؟ [AI کے ساتھ جوڑی پروگرامنگ کے لیے کوپائلٹ](https://aka.ms/GitHubCopilotAI) دیکھیں۔
