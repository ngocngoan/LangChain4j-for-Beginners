# новичков کے لیے LangChain4j

LangChain4j اور Azure OpenAI GPT-5 کے ساتھ AI ایپلیکیشنز بنانے کے لیے ایک کورس، بنیادی چیٹ سے لے کر AI ایجنٹس تک۔

**LangChain4j میں نئے ہیں؟** کلیدی اصطلاحات اور تصورات کی تعریف کے لیے [گلاسری](docs/GLOSSARY.md) دیکھیں۔

## فہرست مضامین

1. [فوری آغاز](00-quick-start/README.md) - LangChain4j کے ساتھ شروع کریں
2. [تعارف](01-introduction/README.md) - LangChain4j کی بنیادیں سیکھیں
3. [پرامپٹ انجینئرنگ](02-prompt-engineering/README.md) - مؤثر پرامپٹ ڈیزائن میں مہارت حاصل کریں
4. [RAG (ریٹریول اگمینٹڈ جنریشن)](03-rag/README.md) - ذہین علمی نظام بنائیں
5. [ٹولز](04-tools/README.md) - خارجی ٹولز اور آسان اسسٹنٹس کو ضم کریں
6. [MCP (ماڈل کانٹیکسٹ پروٹوکول)](05-mcp/README.md) - ماڈل کانٹیکسٹ پروٹوکول (MCP) اور ایجنٹک ماڈیولز کے ساتھ کام کریں

## تعلیمی راستہ

> **فوری آغاز**

1. اس ریپوزیٹری کو اپنے GitHub اکاؤنٹ میں فورک کریں
2. **Code** پر کلک کریں → **Codespaces** ٹیب → **...** → **New with options...**
3. ڈیفالٹس استعمال کریں – یہ کورس کے لیے تیار کردہ ڈویلپمنٹ کنٹینر کو منتخب کرے گا
4. **Create codespace** پر کلک کریں
5. ماحول کے تیار ہونے کے لیے 5-10 منٹ انتظار کریں
6. شروع کرنے کے لیے سیدھے [Quick Start](./00-quick-start/README.md) پر جائیں!

ماڈیولز مکمل کرنے کے بعد، LangChain4j کے ٹیسٹنگ تصورات کو عملی طور پر دیکھنے کے لیے [Testing Guide](docs/TESTING.md) کو دیکھیں۔

> **نوٹ:** یہ تربیت GitHub ماڈلز اور Azure OpenAI دونوں کا استعمال کرتی ہے۔ [Quick Start](00-quick-start/README.md) ماڈیول GitHub ماڈلز استعمال کرتا ہے (Azure سبسکرپشن کی ضرورت نہیں)، جبکہ ماڈیولز 1-5 Azure OpenAI استعمال کرتے ہیں۔ اگر آپ کے پاس نہیں ہے تو ایک [مفت Azure اکاؤنٹ](https://aka.ms/azure-free-account) کے ساتھ شروع کریں۔

## GitHub Copilot کے ساتھ تعلیم

تیزی سے کوڈنگ شروع کرنے کے لیے، اس منصوبے کو GitHub Codespace یا اپنے مقامی IDE میں مہیا کردہ devcontainer کے ساتھ کھولیں۔ اس کورس میں استعمال ہونے والا devcontainer GitHub Copilot کے ساتھ پری-کنفیگر ہے جو AI کے لیے جوڑے ہوئے پروگرامنگ مہیا کرتا ہے۔

ہر کوڈ مثال میں ایسے سوالات شامل ہیں جو آپ GitHub Copilot سے پوچھ سکتے ہیں تاکہ آپ کی سمجھ کو گہرا کیا جا سکے۔ تلاش کریں 💡/🤖 کے پرامپٹس کو:

- **Java فائل ہیڈرز** - ہر مثال کے مخصوص سوالات
- **ماڈیول کے README** - کوڈ مثالوں کے بعد دریافت کے پرامپٹس

**استعمال کیسے کریں:** کسی بھی کوڈ فائل کو کھولیں اور Copilot کو تجویز کردہ سوالات پوچھیں۔ اس کے پاس پورا کوڈ بیس کا سیاق موجود ہے اور یہ وضاحت، توسیع، اور متبادل تجویز کر سکتا ہے۔

مزید جاننا چاہتے ہیں؟ دیکھیں [AI جوڑے ہوئے پروگرامنگ کے لیے Copilot](https://aka.ms/GitHubCopilotAI)۔
