# LangChain4j برائے مبتدی

LangChain4j اور Azure OpenAI GPT-5.2 کے ساتھ AI ایپلیکیشنز بنانے کا کورس، بنیادی چیٹ سے لے کر AI ایجنٹس تک۔

**LangChain4j میں نئے ہیں؟** اہم اصطلاحات اور تصورات کی تعریف کے لیے [گلاسری](docs/GLOSSARY.md) دیکھیں۔

## فہرست مضامین

1. [فوری آغاز](00-quick-start/README.md) - LangChain4j کے ساتھ شروع کریں
2. [تعارف](01-introduction/README.md) - LangChain4j کے بنیادی اصول سیکھیں
3. [پرومپٹ انجینئرنگ](02-prompt-engineering/README.md) - مؤثر پرومپٹ ڈیزائن میں مہارت حاصل کریں
4. [RAG (ریٹریول-آگمینٹڈ جنریشن)](03-rag/README.md) - ذہین علم پر مبنی نظام بنائیں
5. [اؤزار](04-tools/README.md) - بیرونی اوزار اور آسان اسسٹنٹس کو مربوط کریں
6. [MCP (ماڈل کانٹیکسٹ پروٹوکول)](05-mcp/README.md) - ماڈل کانٹیکسٹ پروٹوکول (MCP) اور ایجنٹک ماڈیولز کے ساتھ کام کریں
---

## تعلیمی راستہ

> **فوری آغاز**

1. اس ذخیرے کو اپنے GitHub اکاؤنٹ میں فورک کریں
2. **Code** → **Codespaces** ٹیب → **...** → **New with options...** پر کلک کریں
3. ڈیفالٹس استعمال کریں – یہ اس کورس کے لیے بنائے گئے ڈیولپمنٹ کنٹینر کو منتخب کرے گا
4. **Create codespace** پر کلک کریں
5. ماحول کے تیار ہونے کے لیے 5-10 منٹ انتظار کریں
6. شروع کرنے کے لیے سیدھا [Quick Start](./00-quick-start/README.md) پر جائیں!

ماڈیول مکمل کرنے کے بعد، LangChain4j کے ٹیسٹنگ تصورات کو عملی طور پر دیکھنے کے لیے [Testing Guide](docs/TESTING.md) دریافت کریں۔

> **نوٹ:** یہ تربیت GitHub ماڈلز اور Azure OpenAI دونوں کو استعمال کرتی ہے۔ [Quick Start](00-quick-start/README.md) ماڈیول GitHub ماڈلز استعمال کرتا ہے (Azure سبسکرپشن کی ضرورت نہیں)، جبکہ ماڈیولز 1-5 Azure OpenAI استعمال کرتے ہیں۔ اگر آپ کے پاس نہیں ہے تو [مفت Azure اکاؤنٹ](https://aka.ms/azure-free-account) کے ساتھ شروع کریں۔


## GitHub Copilot کے ساتھ سیکھنا

کوڈنگ جلد شروع کرنے کے لیے، اس پروجیکٹ کو GitHub Codespace یا آپ کے مقامی IDE میں فراہم کردہ devcontainer کے ساتھ کھولیں۔ اس کورس میں استعمال ہونے والا devcontainer پہلے سے GitHub Copilot کے ساتھ ترتیب دیا گیا ہے تاکہ AI پیئر پروگرامنگ کی سہولت دے۔

ہر کوڈ مثال میں وہ سوالات شامل ہیں جو آپ GitHub Copilot سے پوچھ کر اپنی سمجھ کو گہرا کر سکتے ہیں۔ درج ذیل جگہوں پر 💡/🤖 پرومپٹس دیکھیں:

- **Java فائل ہیڈرز** - ہر مثال کے مخصوص سوالات
- **ماڈیول READMEs** - کوڈ مثالوں کے بعد دریافت کے پرومپٹس

**استعمال کا طریقہ:** کسی بھی کوڈ فائل کو کھولیں اور تجویز کردہ سوالات Copilot سے پوچھیں۔ اسے کوڈ بیس کا مکمل سیاق و سباق معلوم ہے اور یہ وضاحت، توسیع، اور متبادل تجویز کر سکتا ہے۔

مزید سیکھنا چاہتے ہیں؟ دیکھیں [Copilot for AI Paired Programming](https://aka.ms/GitHubCopilotAI)۔
