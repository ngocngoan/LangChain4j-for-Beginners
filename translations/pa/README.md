# LangChain4j ਨਵੇਂ ਸਿੱਖਣ ਵਾਲਿਆਂ ਲਈ

LangChain4j ਅਤੇ Azure OpenAI GPT-5.2 ਨਾਲ AI ਐਪਲੀਕੇਸ਼ਨ ਬਣਾਉਣ ਲਈ ਇੱਕ ਕੋਰਸ, ਆਧਾਰਭੂਤ ਚੈੱਟ ਤੋਂ ਲੈ ਕੇ AI ਏਜੰਟ ਤੱਕ।

**LangChain4j ਵਿੱਚ ਨਵਾਂ ਹੋ?** ਮੁੱਖ ਸ਼ਬਦਾਂ ਅਤੇ ਧਾਰਣਾਵਾਂ ਦੀ ਪਰਿਭਾਸ਼ਾ ਲਈ [Glossary](docs/GLOSSARY.md) ਦੇਖੋ।

## ਸਿੱਖਣ ਦਾ ਰਸਤਾ

> **Quick Start**

1. ਇਸ ਰਿਪੋਜ਼ਟਰੀ ਨੂੰ ਆਪਣੇ GitHub ਅਕਾਉਂਟ ਤੇ ਫੋਰਕ ਕਰੋ
2. **Code** → **Codespaces** ਟੈਬ → **...** → **New with options...** 'ਤੇ ਕਲਿੱਕ ਕਰੋ
3. ਡਿਫੌਲਟਸ ਦੀ ਵਰਤੋਂ ਕਰੋ – ਇਹ ਇਸ ਕੋਰਸ ਲਈ ਬਣਾਇਆ ਗਿਆ Development ਕੰਟੇਨਰ ਚੁਣੇਗਾ
4. **Create codespace** 'ਤੇ ਕਲਿੱਕ ਕਰੋ
5. ਮਾਹੌਲ ਤੈਯਾਰ ਹੋਣ ਲਈ 5-10 ਮਿੰਟ ਦੀ ਉਡੀਕ ਕਰੋ
6. ਤੁਰੰਤ [Quick Start](./00-quick-start/README.md) 'ਤੇ ਜਾਕੇ ਸ਼ੁਰੂ ਕਰੋ!

ਮੋਡੀਊਲ ਮੁਕੰਮਲ ਕਰਨ ਤੋਂ ਬਾਅਦ, LangChain4j ਟੈਸਟਿੰਗ ਸੰਕਲਪਾਂ ਨੂੰ ਵੇਖਣ ਲਈ [Testing Guide](docs/TESTING.md) ਦੀ ਖੋਜ ਕਰੋ।

> **ਨੋਟ:** ਇਹ ਟ੍ਰੇਨਿੰਗ ਦੋਨੋਂ GitHub ਮਾਡਲਾਂ ਅਤੇ Azure OpenAI ਦੀ ਵਰਤੋਂ ਕਰਦੀ ਹੈ। [Quick Start](00-quick-start/README.md) ਮੋਡੀਊਲ GitHub ਮਾਡਲਾਂ ਨਾਲ ਹੈ (Azure ਸਬਸਕ੍ਰਿਪਸ਼ਨ ਲੋੜੀਂਦਾ ਨਹੀਂ), ਜਦਕਿ ਮੋਡੀਊਲ 1-5 Azure OpenAI ਦੀ ਵਰਤੋਂ ਕਰਦੇ ਹਨ। ਜੇ ਤੁਹਾਡੇ ਕੋਲ ਹੈ ਨਹੀਂ, ਤਾਂ ਫ੍ਰੀ Azure ਖਾਤਾ ਲਓ: [FREE Azure account](https://aka.ms/azure-free-account)।

## GitHub Copilot ਨਾਲ ਸਿੱਖਣਾ

ਤੁਰੰਤ ਕੋਡਿੰਗ ਸ਼ੁਰੂ ਕਰਨ ਲਈ, ਇਸ ਪ੍ਰੋਜੈਕਟ ਨੂੰ GitHub Codespace ਜਾਂ ਆਪਣੇ ਲੋਕਲ IDE ਵਿੱਚ devcontainer ਨਾਲ ਖੋਲ੍ਹੋ। ਇਸ ਕੋਰਸ ਦਾ devcontainer GitHub Copilot ਨਾਲ ਪਹਿਲਾਂ ਤੋਂ ਸੰਰਚਿਤ ਹੈ ਜੋ AI ਜੋੜੀਵਾਰ ਪ੍ਰੋਗ੍ਰਾਮਿੰਗ ਲਈ ਹੈ।

ਹਰ ਕੋਡ ਉਦਾਹਰਨ ਵਿੱਚ ਕੁਝ ਸੁਝਾਏ ਗਏ ਸਵਾਲ ਹੁੰਦੇ ਹਨ ਜੋ ਤੁਸੀਂ GitHub Copilot ਨੂੰ ਪੁੱਛ ਕੇ ਆਪਣੀ ਸਮਝ ਨੂੰ গভਿਰ ਕਰ ਸਕਦੇ ਹੋ। 💡/🤖 ਪ੍ਰਾਂਪਟ ਲੱਭੋ:

- **Java ਫਾਈਲ ਹੈਡਰਜ਼** - ਹਰ ਉਦਾਹਰਨ ਲਈ ਵਿਸ਼ੇਸ਼ ਸਵਾਲ
- **ਮੋਡੀਊਲ README** - ਕੋਡ ਉਦਾਹਰਣਾਂ ਦੇ ਬਾਅਦ ਖੋਜ ਸਵਾਲ

**ਕਿਵੇਂ ਵਰਤਣਾ ਹੈ:** ਕੋਈ ਵੀ ਕੋਡ ਫਾਈਲ ਖੋਲ੍ਹੋ ਅਤੇ Copilot ਨੂੰ ਸੁਝਾਏ ਸਵਾਲ ਪੁੱਛੋ। ਇਸ ਕੋਲ ਕੋਡਬੇਸ ਦਾ ਪੂਰਾ ਸੰਦਰਭ ਹੈ ਅਤੇ ਇਹ ਵਿਆਖਿਆ ਕਰ ਸਕਦਾ ਹੈ, ਵਧਾ ਸਕਦਾ ਹੈ, ਅਤੇ ਵਿਕਲਪ ਸੁਝਾ ਸਕਦਾ ਹੈ।

ਹੋਰ ਸਿੱਖਣਾ ਚਾਹੁੰਦੇ ਹੋ? ਦੇਖੋ [Copilot for AI Paired Programming](https://aka.ms/GitHubCopilotAI)।
