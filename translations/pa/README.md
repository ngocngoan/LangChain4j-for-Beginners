# LangChain4j ਸ਼ੁਰੂਆਤੀ ਦੇ ਲਈ

LangChain4j ਅਤੇ Azure OpenAI GPT-5 ਨਾਲ AI ਐਪਲੀਕੇਸ਼ਨਾਂ ਬਣਾਉਣ ਲਈ ਇੱਕ ਕੋਰਸ, ਮੂਲ ਚੈਟ ਤੋਂ AI ਏਜੰਟਜ਼ ਤੱਕ।

**LangChain4j ਵਿੱਚ ਨਵਾਂ ਹੋ?** ਕੁੰਜੀ ਸ਼ਬਦ ਅਤੇ ਸਿਧਾਂਤਾਂ ਦੀਆਂ ਵਿਆਖਿਆਵਾਂ ਲਈ [ਐਲਫਾਬੇਟਿਕ ਸੂਚੀ](docs/GLOSSARY.md) ਵੇਖੋ।

## ਸਮੱਗਰੀ ਸੂਚੀ

1. [ਤੁਰੰਤ ਸ਼ੁਰੂਆਤ](00-quick-start/README.md) - LangChain4j ਨਾਲ ਸ਼ੁਰੂ ਕਰੋ
2. [ਪ੍ਰਸਤਾਵਨਾ](01-introduction/README.md) - LangChain4j ਦੇ ਮੂਲ ਸਿਧਾਂਤ ਸਿੱਖੋ
3. [ਪ੍ਰਾਂਪਟ ਇੰਜੀਨੀਅਰਿੰਗ](02-prompt-engineering/README.md) - ਪ੍ਰਭਾਵਸ਼ਾਲੀ ਪ੍ਰਾਂਪਟ ਡਿਜ਼ਾਈਨ ਵਿੱਚ ਮਾਹਿਰ ਬਣੋ
4. [RAG (ਰੀਟਰੀਵਲ-ਓਗਮੈਂਟਡ ਜੇਨੇਰੇਸ਼ਨ)](03-rag/README.md) - ਸਮਝਦਾਰ ਗਿਆਨ-ਅਧਾਰਤ ਪ੍ਰਣਾਲੀਆਂ ਬਣਾਓ
5. [ਟੂਲਜ਼](04-tools/README.md) - ਬਾਹਰੀ ਸਾਧਨਾਂ ਅਤੇ ਸਰਲ ਸਹਾਇਕਾਂ ਨਾਲ ਸੰਚਾਲਨ ਕਰੋ
6. [MCP (ਮਾਡਲ ਸੰਦਰਭ ਪ੍ਰੋਟੋਕੋਲ)](05-mcp/README.md) - ਮਾਡਲ ਸੰਦਰਭ ਪ੍ਰੋਟੋਕੋਲ (MCP) ਅਤੇ ਏਜੰਟਿਕ ਮੋਡੀਊਲਾਂ ਨਾਲ ਕੰਮ ਕਰੋ

##  ਸਿੱਖਣ ਦਾ ਰਸਤਾ

> **ਤੁਰੰਤ ਸ਼ੁਰੂਆਤ**

1. ਇਸ ਰਿਪੋਜਿਟਰੀ ਨੂੰ ਆਪਣੇ GitHub ਖਾਤੇ 'ਤੇ Fork ਕਰੋ
2. **Code** → **Codespaces** ਟੈਬ → **...** → **New with options...** 'ਤੇ ਕਲਿਕ ਕਰੋ
3. ਡਿਫਾਲਟ ਸੈਟਿੰਗਜ਼ ਵਰਤੋਂ – ਇਹ ਕੋਰਸ ਲਈ ਬਣਾਈ ਗਈ ਡਿਵੈਲਪਮੈਂਟ ਕੰਟੇਨਰ ਚੁਣੇਗਾ
4. **Create codespace** 'ਤੇ ਕਲਿਕ ਕਰੋ
5. ਮਾਹੌਲ ਤਿਆਰ ਹੋਣ ਲਈ 5-10 ਮਿੰਟ ਇੰਤਜ਼ਾਰ ਕਰੋ
6. ਸ਼ੁਰੂ ਕਰਨ ਲਈ ਸਿੱਧਾ [Quick Start](./00-quick-start/README.md) 'ਤੇ ਜਾਓ!

ਮੋਡੀਊਲਾਂ ਨੂੰ ਪੂਰਾ ਕਰਨ ਤੋਂ ਬਾਦ, LangChain4j ਟੈਸਟਿੰਗ ਸਿਧਾਂਤਾਂ ਨੂੰ ਦੇਖਣ ਲਈ [Testing Guide](docs/TESTING.md) ਨੂੰ ਵੇਖੋ।

> **ਨੋਟ:** ਇਹ ਟਰੇਨਿੰਗ ਗਿਟਹਬ ਮਾਡਲਜ਼ ਅਤੇ Azure OpenAI ਦੋਹਾਂ ਦਾ ਉਪਯੋਗ ਕਰਦੀ ਹੈ। [Quick Start](00-quick-start/README.md) ਮੋਡੀਊਲ ਗਿਟਹਬ ਮਾਡਲਜ਼ ਦਾ ਉਪਯੋਗ ਕਰਦਾ ਹੈ (ਕੋਈ Azure ਸਬਸਕ੍ਰਿਪਸ਼ਨ ਨਹੀਂ ਚਾਹੀਦਾ), ਜਦਕਿ ਮੋਡੀਊਲ 1-5 Azure OpenAI ਦਾ ਉਪਯੋਗ ਕਰਦੇ ਹਨ। ਜੇ ਤੁਹਾਡੇ ਕੋਲ Azure ਖਾਤਾ ਨਹੀਂ ਹੈ ਤਾਂ [ਮੁਫ਼ਤ Azure ਖਾਤਾ](https://aka.ms/azure-free-account) ਨਾਲ ਸ਼ੁਰੂ ਕਰੋ।

## GitHub ਕੋਪਾਇਲਟ ਨਾਲ ਸਿੱਖਣਾ

ਕੋਡ ਲਿਖਣ ਬਹੁਤ ਤੇਜ਼ੀ ਨਾਲ ਸ਼ੁਰੂ ਕਰਨ ਲਈ, ਇਸ ਪ੍ਰੋਜੈਕਟ ਨੂੰ ਗਿਟਹਬ ਕੋਡਸਪੇਸ ਜਾਂ ਆਪਣੀ ਸਥਾਨਕ IDE ਵਿੱਚ ਦਿੱਤੇ ਹੋਏ devcontainer ਨਾਲ ਖੋਲ੍ਹੋ। ਇਸ ਕੋਰਸ ਵਿੱਚ ਵਰਤਿਆ ਗਇਆ devcontainer GitHub Copilot ਨਾਲ ਪਹਿਲਾਂ ਤੋਂ ਸੰਰਚਿਤ ਹੈ ਜੋ AI ਜੋੜ-ਕੀਤੇ ਪਰੋਗ੍ਰਾਮਿੰਗ ਲਈ ਹੈ।

ਹਰ ਕੋਡ ਉਦਾਹਰਨ ਵਿੱਚ ਸੁਝਾਏ ਗਏ ਪ੍ਰਸ਼ਨਾਂ ਨੂੰ GitHub Copilot ਤੋਂ ਪੁੱਛੋ ਤਾਂ ਜੋ ਤੁਹਾਡੀ ਸਮਝ ਖੁਲ੍ਹਕੇ ਹੋ ਸਕੇ। ਇਹ ਪ੍ਰੋੰਪਟਾਂ 💡/🤖 ਹੇਠਾਂ ਲੱਭੋ:

- **ਜਾਵਾ ਫਾਈਲ ਹੈਡਰਾਂ** - ਹਰ ਉਦਾਹਰਨ ਨਾਲ ਸੰਬੰਧਿਤ ਪ੍ਰਸ਼ਨ
- **ਮੋਡੀਊਲ README ਫਾਈਲਾਂ** - ਕੋਡ ਉਦਾਹਰਣਾਂ ਦੇ ਬਾਅਦ ਅਨੁਸੰਧਾਨ ਪ੍ਰੋੰਪਟ

**ਵਰਤੋਂ ਕਰਨ ਦਾ ਤਰੀਕਾ:** ਕੋਈ ਵੀ ਕੋਡ ਫਾਈਲ ਖੋਲ੍ਹੋ ਅਤੇ Copilot ਨੂੰ ਸੁਝਾਏ ਗਏ ਪ੍ਰਸ਼ਨ ਪੁੱਛੋ। ਇਸ ਕੋਡਬੇਸ ਦਾ ਪੂਰਾ ਸੰਦਰਭ ਹੈ ਅਤੇ ਇਹ ਸਮਝਾ ਸਕਦਾ ਹੈ, ਵਧਾ ਸਕਦਾ ਹੈ ਅਤੇ ਵਿਕਲਪ ਸੁਝਾ ਸਕਦਾ ਹੈ।

ਵਧੇਰੇ ਜਾਣਨਾ ਚਾਹੁੰਦੇ ਹੋ? [AI ਜੋੜ-ਕੀਤੇ ਪਰੋਗ੍ਰਾਮਿੰਗ ਲਈ Copilot](https://aka.ms/GitHubCopilotAI) ਦੇਖੋ।
