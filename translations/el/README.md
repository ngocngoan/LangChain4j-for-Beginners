# LangChain4j για Αρχαρίους

Ένα μάθημα για την κατασκευή εφαρμογών AI με LangChain4j και Azure OpenAI GPT-5.2, από βασική συνομιλία έως AI agents.

**Νέος στο LangChain4j;** Δείτε το [Λεξιλόγιο](docs/GLOSSARY.md) για ορισμούς βασικών όρων και εννοιών.

## Πίνακας Περιεχομένων

1. [Γρήγορη Εκκίνηση](00-quick-start/README.md) - Ξεκινήστε με το LangChain4j  
2. [Εισαγωγή](01-introduction/README.md) - Μάθετε τα βασικά του LangChain4j  
3. [Μηχανική Προτροπής](02-prompt-engineering/README.md) - Κατακτήστε τον αποτελεσματικό σχεδιασμό προτροπών  
4. [RAG (Ανάκτηση-Ενισχυμένη Γενιά)](03-rag/README.md) - Κατασκευάστε έξυπνα συστήματα βασισμένα στη γνώση  
5. [Εργαλεία](04-tools/README.md) - Ενσωματώστε εξωτερικά εργαλεία και απλούς βοηθούς  
6. [MCP (Πρωτόκολλο Πλαισίου Μοντέλου)](05-mcp/README.md) - Εργαστείτε με το Πρωτόκολλο Πλαισίου Μοντέλου (MCP) και μονάδες Agentic

## Μονοπάτι Μάθησης

> **Γρήγορη Εκκίνηση**

1. Κάντε fork αυτό το αποθετήριο στον λογαριασμό σας στο GitHub  
2. Κάντε κλικ στο **Code** → καρτέλα **Codespaces** → **...** → **Νέο με επιλογές...**  
3. Χρησιμοποιήστε τις προεπιλογές – αυτό θα επιλέξει το Development container που δημιουργήθηκε για αυτό το μάθημα  
4. Κάντε κλικ στο **Create codespace**  
5. Περιμένετε 5-10 λεπτά για να είναι έτοιμο το περιβάλλον  
6. Μεταβείτε απευθείας στο [Γρήγορη Εκκίνηση](./00-quick-start/README.md) για να ξεκινήσετε!

Μετά την ολοκλήρωση των μονάδων, εξερευνήστε τον [Οδηγό Δοκιμών](docs/TESTING.md) για να δείτε τις έννοιες δοκιμών του LangChain4j σε δράση.

> **Σημείωση:** Αυτή η εκπαίδευση χρησιμοποιεί τόσο GitHub Models όσο και Azure OpenAI. Η ενότητα [Γρήγορη Εκκίνηση](00-quick-start/README.md) χρησιμοποιεί τα GitHub Models (δεν απαιτείται συνδρομή Azure), ενώ οι ενότητες 1-5 χρησιμοποιούν Azure OpenAI. Ξεκινήστε με έναν [ΔΩΡΕΑΝ Λογαριασμό Azure](https://aka.ms/azure-free-account) αν δεν έχετε.


## Μάθηση με το GitHub Copilot

Για να ξεκινήσετε γρήγορα τον προγραμματισμό, ανοίξτε αυτό το έργο σε GitHub Codespace ή στον τοπικό σας IDE με τον παρεχόμενο devcontainer. Το devcontainer που χρησιμοποιείται σε αυτό το μάθημα είναι προδιαμορφωμένο με GitHub Copilot για προγραμματισμό σε ζεύγη AI.

Κάθε παράδειγμα κώδικα περιλαμβάνει προτεινόμενες ερωτήσεις που μπορείτε να κάνετε στο GitHub Copilot για να εμβαθύνετε την κατανόησή σας. Αναζητήστε τις ενδείξεις 💡/🤖 σε:

- **Επικεφαλίδες αρχείων Java** - Ερωτήσεις ειδικές για κάθε παράδειγμα  
- **Αρχεία README των μονάδων** - Εξερευνητικές ερωτήσεις μετά τα παραδείγματα κώδικα  

**Πώς να το χρησιμοποιήσετε:** Ανοίξτε οποιοδήποτε αρχείο κώδικα και ζητήστε από το Copilot τις προτεινόμενες ερωτήσεις. Έχει πλήρες πλαίσιο για τον κώδικα και μπορεί να εξηγήσει, να επεκτείνει και να προτείνει εναλλακτικές λύσεις.

Θέλετε να μάθετε περισσότερα; Δείτε το [Copilot για AI προγραμματισμό σε ζεύγη](https://aka.ms/GitHubCopilotAI).
