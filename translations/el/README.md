# LangChain4j για Αρχάριους

Ένα μάθημα για την κατασκευή εφαρμογών AI με τα LangChain4j και Azure OpenAI GPT-5.2, από βασική συνομιλία έως πράκτορες AI.

**Νέος στα LangChain4j;** Ρίξτε μια ματιά στο [Λεξιλόγιο](docs/GLOSSARY.md) για ορισμούς βασικών όρων και εννοιών.

## Περιεχόμενα

1. [Γρήγορη Εκκίνηση](00-quick-start/README.md) - Ξεκινήστε με LangChain4j
2. [Εισαγωγή](01-introduction/README.md) - Μάθετε τα βασικά των LangChain4j
3. [Σχεδιασμός Προτροπών](02-prompt-engineering/README.md) - Μάθετε να σχεδιάζετε αποτελεσματικές προτροπές
4. [RAG (Ανάκτηση-Ενισχυμένη Γεννήτρια)](03-rag/README.md) - Δημιουργήστε έξυπνα συστήματα βασισμένα στη γνώση
5. [Εργαλεία](04-tools/README.md) - Ενσωματώστε εξωτερικά εργαλεία και απλούς βοηθούς
6. [MCP (Πρωτόκολλο Συμφραζομένων Μοντέλου)](05-mcp/README.md) - Εργαστείτε με το Πρωτόκολλο Συμφραζομένων Μοντέλου (MCP) και Πρακτορικά μονοπάτια

## Μονοπάτι Μάθησης

> **Γρήγορη Εκκίνηση**

1. Κάντε fork αυτό το αποθετήριο στον λογαριασμό σας στο GitHub
2. Κάντε κλικ στο **Code** → καρτέλα **Codespaces** → **...** → **Νέο με επιλογές...**
3. Χρησιμοποιήστε τις προεπιλογές – αυτό θα επιλέξει το περιβάλλον ανάπτυξης που δημιουργήθηκε για αυτό το μάθημα
4. Κάντε κλικ στο **Δημιουργία codespace**
5. Περιμένετε 5-10 λεπτά για να είναι έτοιμο το περιβάλλον
6. Πηγαίνετε απευθείας στο [Γρήγορη Εκκίνηση](./00-quick-start/README.md) για να ξεκινήσετε!

Μετά την ολοκλήρωση των μονάδων, εξερευνήστε τον [Οδηγό Δοκιμών](docs/TESTING.md) για να δείτε τις έννοιες δοκιμών LangChain4j στην πράξη.

> **Σημείωση:** Αυτή η εκπαίδευση χρησιμοποιεί τόσο τα GitHub Models όσο και το Azure OpenAI. Η ενότητα [Γρήγορη Εκκίνηση](00-quick-start/README.md) χρησιμοποιεί GitHub Models (δεν απαιτείται συνδρομή Azure), ενώ οι ενότητες 1-5 χρησιμοποιούν Azure OpenAI. Ξεκινήστε με έναν [ΔΩΡΕΑΝ λογαριασμό Azure](https://aka.ms/azure-free-account) αν δεν έχετε ήδη.


## Μάθηση με GitHub Copilot

Για να ξεκινήσετε γρήγορα τον κώδικα, ανοίξτε αυτό το έργο σε ένα GitHub Codespace ή στον τοπικό IDE με το παρεχόμενο devcontainer. Το devcontainer που χρησιμοποιείται σε αυτό το μάθημα έρχεται προδιαμορφωμένο με το GitHub Copilot για προγραμματισμό ζευγαριών AI.

Κάθε παράδειγμα κώδικα περιλαμβάνει προτεινόμενες ερωτήσεις που μπορείτε να κάνετε στο GitHub Copilot για να εμβαθύνετε την κατανόησή σας. Αναζητήστε τα 💡/🤖 ερωτήματα στα:

- **Κεφαλίδες αρχείων Java** - Ερωτήσεις συγκεκριμένες για κάθε παράδειγμα
- **README των μονάδων** - Ερωτήματα εξερεύνησης μετά από παραδείγματα κώδικα

**Πώς να χρησιμοποιήσετε:** Ανοίξτε οποιοδήποτε αρχείο κώδικα και κάντε τις προτεινόμενες ερωτήσεις στο Copilot. Έχει πλήρες πλαίσιο του κώδικα και μπορεί να εξηγήσει, να επεκτείνει και να προτείνει εναλλακτικές.

Θέλετε να μάθετε περισσότερα; Ρίξτε μια ματιά στο [Copilot για Προγραμματισμό Ζευγαριών AI](https://aka.ms/GitHubCopilotAI).
