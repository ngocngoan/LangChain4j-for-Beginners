# LangChain4j pro začátečníky

Kurz pro vytváření AI aplikací s LangChain4j a Azure OpenAI GPT-5.2, od základního chatu až po AI agenty.

**Nový v LangChain4j?** Podívejte se na [Glosář](docs/GLOSSARY.md) pro definice klíčových termínů a pojmů.

## Obsah

1. [Rychlý start](00-quick-start/README.md) - Začněte s LangChain4j
2. [Úvod](01-introduction/README.md) - Naučte se základy LangChain4j
3. [Návrh promptů](02-prompt-engineering/README.md) - Osvojte si efektivní návrh promptů
4. [RAG (Retrieval-Augmented Generation)](03-rag/README.md) - Vytvářejte inteligentní znalostní systémy
5. [Nástroje](04-tools/README.md) - Integrace externích nástrojů a jednoduchých asistentů
6. [MCP (Model Context Protocol)](05-mcp/README.md) - Práce s Model Context Protocol (MCP) a agentními moduly

## Výuková cesta

> **Rychlý start**

1. Vytvořte fork tohoto repozitáře do svého účtu na GitHubu
2. Klikněte na **Code** → záložka **Codespaces** → **...** → **Nový s možnostmi...**
3. Použijte výchozí hodnoty – vybere se vývojové prostředí vytvořené pro tento kurz
4. Klikněte na **Create codespace**
5. Počkejte 5-10 minut, než bude prostředí připravené
6. Přejděte přímo na [Rychlý start](./00-quick-start/README.md) a začněte!

Po dokončení modulů prozkoumejte [Průvodce testováním](docs/TESTING.md), kde uvidíte praktické využití testovacích konceptů LangChain4j.

> **Poznámka:** Toto školení používá jak GitHub Models, tak Azure OpenAI. Modul [Rychlý start](00-quick-start/README.md) používá GitHub Models (není potřeba předplatné Azure), zatímco moduly 1–5 využívají Azure OpenAI. Pokud ještě nemáte, založte si [ZDARMA účet Azure](https://aka.ms/azure-free-account).

## Výuka s GitHub Copilot

Pro rychlý začátek s kódováním otevřete tento projekt v GitHub Codespace nebo ve svém lokálním IDE s dodaným devcontainerem. Devcontainer použitý v tomto kurzu je přednastavený s GitHub Copilotem pro AI párové programování.

Každý příklad kódu obsahuje doporučené otázky, které můžete položit GitHub Copilotovi pro hlubší pochopení. Hledejte výzvy 💡/🤖 ve:

- **Hlavičkách Java souborů** – Otázky specifické pro každý příklad
- **README modulů** – Nápady na průzkum po příkladech kódu

**Jak používat:** Otevřete jakýkoli kódový soubor a zeptejte se Copilota na doporučené otázky. Má plný kontext kódu a může vysvětlovat, rozšiřovat a navrhovat alternativy.

Chcete se dozvědět více? Podívejte se na [Copilot pro AI párové programování](https://aka.ms/GitHubCopilotAI).
