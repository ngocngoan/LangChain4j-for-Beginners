# LangChain4j dành cho Người Mới Bắt Đầu

Một khóa học xây dựng ứng dụng AI với LangChain4j và Azure OpenAI GPT-5.2, từ trò chuyện cơ bản đến các tác nhân AI.

**Mới làm quen với LangChain4j?** Xem [Thuật ngữ](docs/GLOSSARY.md) để biết định nghĩa các thuật ngữ và khái niệm chính.

## Mục Lục

1. [Bắt Đầu Nhanh](00-quick-start/README.md) - Bắt đầu với LangChain4j
2. [Giới Thiệu](01-introduction/README.md) - Tìm hiểu các nguyên lý cơ bản của LangChain4j
3. [Kỹ Thuật Prompt](02-prompt-engineering/README.md) - Làm chủ thiết kế prompt hiệu quả
4. [RAG (Sinh Tạo Hỗ Trợ Truy Xuất)](03-rag/README.md) - Xây dựng hệ thống kiến thức thông minh
5. [Công Cụ](04-tools/README.md) - Tích hợp công cụ bên ngoài và trợ lý đơn giản
6. [MCP (Giao Thức Ngữ Cảnh Mô Hình)](05-mcp/README.md) - Làm việc với Giao Thức Ngữ Cảnh Mô Hình (MCP) và các mô-đun Agentic

## Lộ Trình Học

> **Bắt Đầu Nhanh**

1. Fork kho lưu trữ này vào tài khoản GitHub của bạn
2. Nhấn **Code** → tab **Codespaces** → **...** → **New with options...**
3. Sử dụng mặc định – thao tác này sẽ chọn Container Phát triển được tạo sẵn cho khóa học này
4. Nhấn **Create codespace**
5. Đợi 5-10 phút cho môi trường sẵn sàng
6. Vào thẳng phần [Bắt Đầu Nhanh](./00-quick-start/README.md) để bắt đầu!

Sau khi hoàn thành các module, khám phá [Hướng Dẫn Kiểm Thử](docs/TESTING.md) để thấy các khái niệm kiểm thử LangChain4j trong thực tế.

> **Lưu ý:** Khóa đào tạo này sử dụng cả Mô hình GitHub và Azure OpenAI. Module [Bắt Đầu Nhanh](00-quick-start/README.md) dùng Mô hình GitHub (không cần đăng ký Azure), còn các module 1-5 dùng Azure OpenAI. Bắt đầu với [Tài khoản Azure MIỄN PHÍ](https://aka.ms/azure-free-account) nếu bạn chưa có.


## Học cùng GitHub Copilot

Để nhanh chóng bắt đầu viết mã, mở dự án này trong GitHub Codespace hoặc IDE cục bộ của bạn với devcontainer được cung cấp. Devcontainer sử dụng trong khóa học này được cấu hình sẵn với GitHub Copilot cho lập trình cặp AI.

Mỗi ví dụ mã đều kèm theo câu hỏi gợi ý bạn có thể hỏi GitHub Copilot để nâng cao hiểu biết. Tìm các dấu 💡/🤖 trong:

- **Tiêu đề file Java** - Câu hỏi riêng cho mỗi ví dụ
- **README các module** - Câu hỏi khám phá sau ví dụ mã

**Cách sử dụng:** Mở bất kỳ file mã nào và hỏi Copilot theo các câu hỏi gợi ý. Copilot có toàn bộ ngữ cảnh mã nguồn và có thể giải thích, mở rộng, cũng như đề xuất các phương án thay thế.

Muốn tìm hiểu thêm? Xem [Copilot for AI Paired Programming](https://aka.ms/GitHubCopilotAI).
