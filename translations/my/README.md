## သင်ယူမှု လမ်းကြောင်း

**LangChain4j အသစ်လား?** အဓိပ္ပါယ်နှင့် အယူအဆများအတွက် [အသုံးအနှုန်းစာရင်း](docs/GLOSSARY.md) ကို ကြည့်ပါ။

> **အရှိန်အဟုန် စတင်ခြင်း**

1. ဒီ repository ကို သင့် GitHub အကောင့်သို့ Fork ပြုလုပ်ပါ
2. **Code** → **Codespaces** tab → **...** → **New with options...** ကိုနှိပ်ပါ
3. မရပ်တန့်ပဲ ရွေးချယ်မှုများကို သုံးပါ – ဒီသင်တန်းအတွက် ဖန်တီးထားသော Development container ကို ရွေးယူပါလိမ့်မယ်
4. **Create codespace** ကိုနှိပ်ပါ
5. ပတ်ဝန်းကျင် ပြင်ဆင်ပြီးရန် ၅-၁၀ မိနစ် စောင့်ပါ
6. တိုက်ရိုက် [အရှိန်အဟုန် စတင်ခြင်း](./00-quick-start/README.md) သို့ ရောက်ရှိပါ!

အပိုင်းများပြီးပါက [စမ်းသပ်မှုလမ်းညွှန်](docs/TESTING.md) ကို ရှာဖွေကြည့်ပါ၊ LangChain4j စမ်းသပ်ဖော်ပြချက်များကိုကြည့်ရှုနိုင်သည်။

> **မွတ္ခ်က္။** ဒီသင်ခန်းစာမှာ GitHub Models နှင့် Azure OpenAI နှစ်မျိုးကို အသုံးပြုသည်။ [အရှိန်အဟုန် စတင်ခြင်း](00-quick-start/README.md) အပိုင်းမှာ GitHub Models ကိုသုံးပြီး (Azure subscription မလိုအပ်ပါ), ၁ မှ ၅ အပိုင်းများမှာ Azure OpenAI ကို သုံးသည်။ Azure အကောင့် မရှိသေးပါက [အခမဲ့ Azure အကောင့်](https://aka.ms/azure-free-account) ဖြင့် စတင်ပါ။

## GitHub Copilot နှင့် သင်ယူခြင်း

အမြန်ဆုံး ကုဒ်ရေးရန် ဤ project ကို GitHub Codespace သို့မဟုတ် သင့်ရဲ့ ဒေသတွင်း IDE တွင် devcontainer ဖြင့် ဖွင့်ပါ။ ဒီ သင်တန်းအတွက် အသုံးပြုထားသော devcontainer သည် AI တွဲဖက်ရေးသားရေးအတွက် GitHub Copilot နဲ့ ကြိုတင်ပြင်ဆင်ထားပါသည်။

ကုဒ်မူလပုံစံ တစ်ခုချင်းစီတွင် GitHub Copilot ကို မေးနိုင်သော အကြံပြုမေးခွန်းများ ပါဝင်သည်။ 💡/🤖 ခြင်းများကို အောက်ပါနေရာများတွင် ကြည့်ရှုပါ-

- **Java ဖိုင်ခေါင်းစီးများ** - ပုံစံတိုင်းအတွက် မေးခွန်းများ
- **Module README များ** - ကုဒ်နမူနာပြီးနောက် စူးစမ်းမေးမြန်းမှုများ

**ဘယ်လိုအသုံးပြုမလဲ။** ဖိုင် ဖွင့်ပြီး Copilot ကို အကြံပြုမေးခွန်းများကို မေးပါ။ ၎င်းတွင် ကုဒ်အခြေခံအားလုံးနားလည်မှုရှိပြီး ရှင်းပြပေးနိုင်၊ ဖွံ့ဖြိုးတိုးချဲ့နှင့် အခြားရွေးချယ်စရာများကို အကြံပြုနိုင်သည်။

ပိုမိုသိလိုပါသလား? [AI တွဲဖက်ရေးသားရေးအတွက် Copilot](https://aka.ms/GitHubCopilotAI) ကိုကြည့်ပါ။
