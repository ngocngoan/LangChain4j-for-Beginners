# LangChain4j 入門指南

透過 LangChain4j 和 Azure OpenAI GPT-5.2 建立 AI 應用程式的課程，包含從基礎聊天到 AI 代理人的內容。

**LangChain4j 新手？** 請查看 [術語彙編](docs/GLOSSARY.md) 以了解關鍵詞彙與概念定義。

## 目錄

1. [快速開始](00-quick-start/README.md) - 開始使用 LangChain4j
2. [介紹](01-introduction/README.md) - 學習 LangChain4j 的基礎知識
3. [提示工程](02-prompt-engineering/README.md) - 精通有效的提示設計
4. [RAG（檢索增強生成）](03-rag/README.md) - 建立智慧的知識型系統
5. [工具](04-tools/README.md) - 整合外部工具和簡易助理
6. [MCP（模型上下文協議）](05-mcp/README.md) - 使用模型上下文協議（MCP）與智能代理模組
---

## 學習路徑

> **快速開始**

1. 分叉此倉庫到您的 GitHub 帳號
2. 點選 **Code** → **Codespaces** 索引標籤 → **...** → **New with options...**
3. 保持預設設定 — 這會選擇本課程所建立的開發容器
4. 點擊 **Create codespace**
5. 等待 5-10 分鐘環境準備完成
6. 直接進入 [快速開始](./00-quick-start/README.md) 開始學習！

完成模組後，可參考 [測試指南](docs/TESTING.md) 了解 LangChain4j 測試概念。

> **注意：** 本課程同時使用了 GitHub Models 與 Azure OpenAI。[快速開始](00-quick-start/README.md) 模組使用 GitHub Models（無需 Azure 訂閱），而模組 1-5 則使用 Azure OpenAI。若尚無帳號，請先申請 [免費 Azure 帳號](https://aka.ms/azure-free-account)。

## 使用 GitHub Copilot 進行學習

要快速開始寫程式，請在 GitHub Codespace 或本機 IDE 中打開此專案，並使用本課程提供的 devcontainer。本課程中的 devcontainer 預先設定了 GitHub Copilot，以進行 AI 配對程式設計。

每個程式範例都包含建議您可向 GitHub Copilot 詢問的問題，幫助加深理解。請留意以下位置的 💡/🤖 提示：

- **Java 程式檔標頭** - 各範例特定問題
- **模組 README** - 程式示例後的探索提示

**使用方式：** 打開任意程式檔後，向 Copilot 詢問建議問題。它具有完整的程式碼庫上下文，能夠解釋、擴充及建議其他方案。

想了解更多？請參閱 [AI 配對程式設計的 Copilot](https://aka.ms/GitHubCopilotAI)。
