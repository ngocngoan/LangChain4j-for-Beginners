## 學習路徑

**是 LangChain4j 新手嗎？** 請瀏覽 [詞彙表](docs/GLOSSARY.md) 了解關鍵術語與概念定義。

> **快速開始**

1. Fork 此存放庫至你的 GitHub 帳號
2. 點擊 **Code** → **Codespaces** 標籤 → **...** → **New with options...**
3. 使用預設設定 — 此選擇為本課程建立的開發容器
4. 點擊 **Create codespace**
5. 等待 5-10 分鐘環境準備完成
6. 直接跳至 [快速開始](./00-quick-start/README.md) 展開學習！

完成模組後，探索 [測試指南](docs/TESTING.md) 觀察 LangChain4j 測試概念如何實施。

> **注意：** 本訓練同時使用 GitHub 模型和 Azure OpenAI。[快速開始](00-quick-start/README.md) 模組使用 GitHub 模型（無需 Azure 訂閱），而第 1-5 模組則使用 Azure OpenAI。若尚無 Azure 帳戶，歡迎以 [免費 Azure 帳戶](https://aka.ms/azure-free-account) 開始。

## 使用 GitHub Copilot 學習

要快速開始編碼，把此專案用 GitHub Codespace 或本地 IDE 配合提供的 devcontainer 開啟。本課程所用的 devcontainer 預先設定包含 GitHub Copilot，支持 AI 配對編程。

每個程式碼範例中包含你可以問 GitHub Copilot 的建議問題，加深理解。請尋找以下位置的 💡/🤖 提示：

- **Java 檔案標頭** — 針對每個範例的特定問題
- **模組 README** — 程式碼範例後的探索提示

**使用方式：** 開啟任何程式碼檔案，向 Copilot 詢問建議問題。它擁有完整程式碼庫上下文，能解釋、擴充並建議替代方案。

想學得更多？請參考 [Copilot AI 配對編程](https://aka.ms/GitHubCopilotAI)。
