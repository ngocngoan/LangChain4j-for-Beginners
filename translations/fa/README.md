# LangChain4j برای مبتدیان

دوره‌ای برای ساخت برنامه‌های هوش مصنوعی با LangChain4j و Azure OpenAI GPT-5.2، از چت پایه تا عوامل هوش مصنوعی.

**تازه‌وارد به LangChain4j هستید؟** برای تعاریف واژگان کلیدی و مفاهیم به [واژه‌نامه](docs/GLOSSARY.md) مراجعه کنید.

## مسیر یادگیری

> **شروع سریع**

1. این مخزن را به حساب GitHub خود فورک کنید  
2. روی **Code** کلیک کنید → زبانه **Codespaces** → **...** → **New with options...**  
3. تنظیمات پیش‌فرض را استفاده کنید – این کانتینر توسعه‌ای ساخته شده برای این دوره را انتخاب می‌کند  
4. روی **Create codespace** کلیک کنید  
5. ۵ تا ۱۰ دقیقه صبر کنید تا محیط آماده شود  
6. مستقیماً به [شروع سریع](./00-quick-start/README.md) بروید و کار را آغاز کنید!

پس از اتمام ماژول‌ها، راهنمای [آزمون](docs/TESTING.md) را بررسی کنید تا مفاهیم آزمون LangChain4j را در عمل ببینید.

> **توجه:** این آموزش هم از GitHub Models و هم از Azure OpenAI استفاده می‌کند. ماژول [شروع سریع](00-quick-start/README.md) از GitHub Models استفاده می‌کند (نیاز به اشتراک Azure ندارد)، در حالی که ماژول‌های ۱ تا ۵ از Azure OpenAI بهره می‌برند. اگر حساب Azure ندارید، با یک [حساب رایگان Azure](https://aka.ms/azure-free-account) شروع کنید.


## یادگیری با GitHub Copilot

برای شروع سریع کدنویسی، این پروژه را در GitHub Codespace یا IDE محلی خود با devcontainer ارائه شده باز کنید. devcontainer استفاده‌شده در این دوره از پیش با GitHub Copilot برای برنامه‌نویسی جفتی AI تنظیم شده است.

هر مثال کدی شامل سوالات پیشنهادی است که می‌توانید از GitHub Copilot بپرسید تا فهم خود را عمیق‌تر کنید. به دنبال نشانه‌های 💡/🤖 در:

- **هدرهای فایل جاوا** - سوالات مخصوص هر مثال  
- **README ماژول‌ها** - سوالات اکتشافی بعد از مثال‌های کد

**چگونه استفاده کنیم:** هر فایل کدی را باز کنید و سوالات پیشنهادی را از Copilot بپرسید. Copilot کامل از کد مطلع است و می‌تواند توضیح دهد، گسترش دهد، و جایگزین‌های پیشنهادی بدهد.

می‌خواهید بیشتر بیاموزید؟ [Copilot برای برنامه‌نویسی جفتی AI](https://aka.ms/GitHubCopilotAI) را ببینید.
