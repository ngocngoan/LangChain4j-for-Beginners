# LangChain4j 入门

## 学习路径

> **快速开始**

1. 将本仓库 Fork 到你的 GitHub 账号
2. 点击 **Code** → **Codespaces** 标签 → **...** → **New with options...**
3. 使用默认设置 — 这将选择为本课程创建的开发容器
4. 点击 **Create codespace**
5. 等待 5-10 分钟环境准备完毕
6. 直接跳转到 [快速开始](./00-quick-start/README.md) 开始学习！

完成所有模块后，查看 [测试指南](docs/TESTING.md) 了解 LangChain4j 测试概念的实际演示。

> **注意：** 本训练使用 GitHub 模型和 Azure OpenAI。 [快速开始](00-quick-start/README.md) 模块使用 GitHub 模型（无需 Azure 订阅），而模块1-5使用 Azure OpenAI。若没有，请开启一个 [免费 Azure 账户](https://aka.ms/azure-free-account)。

## 使用 GitHub Copilot 学习

为了快速开始编程，可在 GitHub Codespace 或本地 IDE 中打开本项目，使用本课程提供的 devcontainer。本课程使用的 devcontainer 预配置了 GitHub Copilot 以辅助 AI 配对编程。

每个代码示例都包含一些建议提问，你可以询问 GitHub Copilot 以加深理解。请留意以下位置的 💡/🤖 提示：

- **Java 文件头部** - 针对每个示例的问题
- **模块 README** - 代码示例后的探索提示

**使用方法：** 打开任意代码文件，向 Copilot 提出建议问题。它能理解整个代码库，提供解释、扩展及替代方案。

想了解更多？请访问 [AI 配对编程的 Copilot](https://aka.ms/GitHubCopilotAI)。
