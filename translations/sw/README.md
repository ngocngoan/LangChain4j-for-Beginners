# LangChain4j kwa Waanzilishi

Kozi ya kujenga programu za AI kwa LangChain4j na Azure OpenAI GPT-5.2, kutoka mazungumzo ya msingi hadi mawakala wa AI.

**Mpya kwa LangChain4j?** Angalia [Kamusi](docs/GLOSSARY.md) kwa ufafanuzi wa maneno na dhana muhimu.

## Jedwali la Yaliyomo

1. [Anza Haraka](00-quick-start/README.md) - Anza na LangChain4j
2. [Utangulizi](01-introduction/README.md) - Jifunze misingi ya LangChain4j
3. [Uhandisi wa Prompt](02-prompt-engineering/README.md) - Jifunze kubuni maombi yenye ufanisi
4. [RAG (Uzalishaji Ulioboreshwa kwa Kupata Taarifa)](03-rag/README.md) - Jenga mifumo ya maarifa yenye akili
5. [Vifaa](04-tools/README.md) - Unganisha zana za nje na wasaidizi rahisi
6. [MCP (Itifaki ya Muktadha wa Modeli)](05-mcp/README.md) - Fanya kazi na Itifaki ya Muktadha wa Modeli (MCP) na moduli za Kiwakala
---

## Njia ya Kujifunza

> **Anza Haraka**

1. Tafadhali tengeneza kopia ya repositi hii kwa akaunti yako ya GitHub
2. Bonyeza **Code** → kichupo cha **Codespaces** → **...** → **Mpya na chaguzi...**
3. Tumia chaguo asili – hii itachagua chombo cha Maendeleo kilichotengenezwa kwa kozi hii
4. Bonyeza **Unda codespace**
5. Subiri dakika 5-10 mazingira yaendelee kuwa tayari
6. Nenda moja kwa moja kwa [Anza Haraka](./00-quick-start/README.md) kuanza!

Baada ya kumaliza moduli, chunguza [Mwongozo wa Upimaji](docs/TESTING.md) kuona dhana za upimaji wa LangChain4j zikifanyakazi.

> **Kumbuka:** Mafunzo haya yanatumia GitHub Models na Azure OpenAI. Moduli ya [Anza Haraka](00-quick-start/README.md) inatumia GitHub Models (hakuna usajili wa Azure unaohitajika), wakati moduli 1-5 zinatumia Azure OpenAI. Anza na akaunti ya [AZURE BURE](https://aka.ms/azure-free-account) kama huna.


## Kujifunza kwa kutumia GitHub Copilot

Ili kuanza kuandika programu haraka, fungua mradi huu katika GitHub Codespace au IDE yako ya mtaa ukitumia devcontainer iliyoandaliwa. Devcontainer inayotumika katika kozi hii imesanifishwa awali na GitHub Copilot kwa usaidizi wa utengenezaji programu kwa AI.

Kila mfano wa msimbo una maswali yaliyopendekezwa ambayo unaweza kumuuliza GitHub Copilot ili kuongeza uelewa wako. Tazama vidokezo vya 💡/🤖 katika:

- **Kichwa cha Faili za Java** - Maswali maalum kwa kila mfano
- **README za Moduli** - Vidokezo vya uchunguzi baada ya mifano ya msimbo

**Jinsi ya kutumia:** Fungua faili yoyote ya msimbo na muulize GitHub Copilot maswali yaliyopendekezwa. Ina muktadha kamili wa msimbo na inaweza kuelezea, kupanua, na kupendekeza mbadala.

Unataka kujifunza zaidi? Angalia [Copilot kwa Usanifu wa Programu ya AI](https://aka.ms/GitHubCopilotAI).
