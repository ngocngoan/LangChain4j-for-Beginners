# LangChain4j ആരംഭക്കാർക്കായി

LangChain4j-ഉം Azure OpenAI GPT-5.2-ഉം ഉപയോഗിച്ച് ആഗ്യമായ പ്രോഗ്രാമുകൾ നിർമ്മിക്കുന്ന ഒരു കോഴ്‌സ്, അടിസ്ഥാന ചാറ്റിൽ നിന്ന് AI ഏജന്റുകൾ വരെ.

**LangChain4j-യിൽ പുതിയവരാ?** സുപ്രധാന പദങ്ങളും ആശയങ്ങളും നിർവചിക്കുന്നതിനായി [Glossary](docs/GLOSSARY.md) നോക്കൂ.

## ഉള്ളടക്ക പട്ടിക

1. [ത്വരിത പ്രാരംഭം](00-quick-start/README.md) - LangChain4j-നൊപ്പം തുടങ്ങുക
2. [പരിചയം](01-introduction/README.md) - LangChain4j-യുടെ അടിസ്ഥാനങ്ങൾ പഠിക്കുക
3. [പ്രോമ്പ്റ്റ് എഞ്ചിനീയറിംഗ്](02-prompt-engineering/README.md) - ഫലപ്രദമായ പ്രേംപ്റ്റ് രൂപകൽപ്പന നേടുക
4. [RAG (Retrieval-Augmented Generation)](03-rag/README.md) - ബുദ്ധിമുട്ടുള്ള നോളജ്-ബെയസ്ഡ് സിസ്റ്റങ്ങൾ നിർമ്മിക്കുക
5. [സാധനങ്ങൾ](04-tools/README.md) - ബാഹ്യ ഉപകരണങ്ങളും ലളിത സഹായകരും സംയോജിപ്പിക്കുക
6. [MCP (Model Context Protocol)](05-mcp/README.md) - Model Context Protocol (MCP)യും Agentic മോഡ്യൂളുകളും ഉപയോഗിക്കുക

## പഠന പാത

> **ത്വരിത പ്രാരംഭം**

1. ഈ റിപ്പോസിറ്ററി നിങ്ങളുടെ GitHub അക്കൗണ്ടിലേക്ക് ഫോർക്കുചെയ്യുക
2. **Code** → **Codespaces** ടാബ് → **...** → **New with options...** തിരഞ്ഞെടുക്കുക
3. ഡിഫോൾട്ടുകൾ ഉപയോഗിക്കുക – കോഴ്‌സിനായി സൃഷ്ടിച്ച ഡെവലപ്പ്‌മെന്റ് കണ്ടെയ്നർ തിരഞ്ഞെടുക്കും
4. **Create codespace** ക്ലിക്കുചെയ്യുക
5. പരിപാലനത്തിന് 5-10 മിനിറ്റ് കാത്തിരിക്കുക
6. തുറന്നിട്ട് [Quick Start](./00-quick-start/README.md) സന്ദർശിച്ച് തുടക്കം കുറിക്കൂ!

മോഡ്യൂളുകൾ പൂർത്തീകരിച്ചതിനു ശേഷം, LangChain4j ടെസ്റ്റിംഗ് ആശയങ്ങൾ പ്രയോഗത്തിൽ കാണാൻ [Testing Guide](docs/TESTING.md) പരിശോധിക്കുക.

> **കുറിപ്പ്:** ഈ പരിശീലനം GitHub മോഡലുകളും Azure OpenAI ഉം രണ്ടും ഉപയോഗിക്കുന്നു. [Quick Start](00-quick-start/README.md) മോഡ്യൂൾ GitHub മോഡലുകൾ മാത്രം ഉപയോഗിക്കുന്നു (Azure സബ്‌സ്‌ക്രിപ്ഷൻ ആവശ്യമില്ല), എന്നാൽ 1-5 മോട്യൂളുകൾ Azure OpenAI ഉപയോഗിക്കുന്നു. നിങ്ങൾക്കു Azure അക്കൗണ്ട് ഇല്ലെങ്കിൽ [FREE Azure account](https://aka.ms/azure-free-account) ഉപയോഗിച്ച് ആരംഭിക്കുക.


## GitHub Copilot-ഉടെയുള്ള പഠനം

തുമ്പ്രേരിയോട്ർപ്പുഴുതകന്യാ കോഡിംഗ് തുടങ്ങിയാൽ, ഈ പ്രോജക്ട് GitHub Codespace-ൽ അല്ലെങ്കിൽ നൽകപ്പെട്ട devcontainer ഉപയോഗിച്ച് നിങ്ങളുടെ പ്രാദേശിക IDE-യിൽ തുറക്കൂ. ഈ കോഴ്സിൽ ഉപയോഗിക്കുന്ന devcontainer-ൽ GitHub Copilot മുൻകൂർ ക്രമീകരണം ചെയ്തിട്ടുണ്ട്, AI പൊരുത്തം പ്രോഗ്രാമിങ്ങിനായി.

ഓരോ കോഡ് ഉദാഹരണത്തിലും GitHub Copilot-നെ ചോദിക്കാൻ കഴിയും എന്നാണ് നിർദേശിക്കുന്ന ചോദ്യങ്ങൾ ഉൾക്കൊള്ളുന്നു. ഇവ 💡/🤖 പ്രോംപ്റ്റുകളായി കണ്ടുപിടിക്കുക:

- **ജാവ ഫയൽ ഹെഡറുകൾ** - ഓരോ ഉദാഹരണത്തിനുള്ള പ്രത്യേക ചോദ്യങ്ങൾ
- **മോഡ്യൂൾ README-കൾ** - കോഡ് ഉദാഹരണങ്ങൾക്ക് ശേഷം ശോധനക്കുറിപ്പുകൾ

**ഉപയോഗം:** ഏതൊരു കോഡ് ഫയലും തുറന്ന് Copilot-നോട് നിർദേശിച്ച ചോദ്യങ്ങൾ ചോദിക്കുക. കോഡ്ബേസ് പൂർണമായി അറിയാമെന്നും വിശദീകരിക്കാൻ, വിപുലീകരിക്കാൻ, ഓപ്ഷനുകൾ നിർദ്ദേശിക്കാൻ കഴിയും.

കൂടുതൽ അറിയാൻ ആഗ്രഹിക്കുന്നുവോ? [Copilot for AI Paired Programming](https://aka.ms/GitHubCopilotAI) നോക്കുക.
