# LangChain4j ആദ്യപടികള്‍ക്കുള്ളത്

LangChain4jയും Azure OpenAI GPT-5.2-ഉം ഉപയോഗിച്ച് പ്രാഥമിക ചാറ്റിൽ നിന്നു കൃത്രിമ ബുദ്ധി ഏജൻറുകളിലേക്ക് AI ആപ്ലിക്കേഷനുകൾ നിർമ്മിക്കുന്ന കോഴ്സ്.

**LangChain4j-ലെ പുതിയവരോക?** പ്രധാന പദങ്ങളും ആശയങ്ങളും മനസ്സിലാക്കുന്നതിനായി [ഗ്ലോസറി](docs/GLOSSARY.md) നോക്കുക.

## ഉള്ളടക്ക പട്ടിക

5. [ടൂളുകൾ](04-tools/README.md) - പുറം ടൂളുകളും ലളിത സഹായികളും ഇന്റഗ്രേറ്റ് ചെയ്യുക  
6. [MCP (മോഡൽ കോൺടകസ്റ്റ് പ്രോട്ടോക്കോൾ)](05-mcp/README.md) - മോഡൽ കോൺടകസ്റ്റ് പ്രോട്ടോക്കോൾ (MCP)യും ഏജൻറിക് മോഡ്യൂളുകളും കൈകാര്യം ചെയ്യുക

## പഠന പാത

> **ഹരിതാരംഭം**

1. ഈ റെപ്പോസിറ്ററി നിങ്ങളുടെ GitHub അക്കൗണ്ടിലേക്ക് ഫോർക്കുചെയ്യുക  
2. **Code** → **Codespaces** ടാബ് → **...** → **New with options...** ക്ലിക്ക് ചെയ്യുക  
3. ഡിഫോൾട്ട് സെറ്റിംഗുകൾ ഉപയോഗിക്കുക – ഇത് ഈ കോഴ്സിന് സൃഷ്ടിച്ച ഡെവലപ്പ്മെന്റ് കണ്ടെയ്‌നർ തിരഞ്ഞെടുക്കും  
4. **Create codespace** ക്ലിക്ക് ചെയ്യുക  
5. പരിസരം സജ്ജമാകാൻ 5-10 മിനിറ്റ് കാത്തിരിക്കുക  
6. തുടക്കമാക്കാൻ നേരിട്ട് [ഹരിതാരംഭം](./00-quick-start/README.md) സന്ദർശിക്കുക!

മോഡ്യൂളുകൾ പൂർത്തിയാക്കിയതേസമയം, [ടെസ്റ്റിംഗ്ഗ് ഗൈഡ്](docs/TESTING.md) പരിശോധിച്ച് LangChain4j ടെസ്റ്റിംഗിന്റെ ആശയങ്ങൾ പ്രായോഗികമായി കാണുക.

> **ഗమనിക്കുക:** ഈ പരിശീലനം GitHub മോഡലുകളും Azure OpenAIയും ഇരുവരും ഉപയോഗിക്കുന്നു. [ഹരിതാരംഭം](00-quick-start/README.md) GitHub മോഡലുകൾ ഉപയോഗിക്കുന്നു (Azure സബ്സ്ക്രിപ്ഷൻ ആവശ്യമില്ല), എന്നാൽ 1-5 മോഡ്യൂളുകൾ Azure OpenAI ഉപയോഗിക്കുന്നു. നിങ്ങൾക്കു ഇല്ലെങ്കിൽ [ഫ്രീ Azure അക്കൗണ്ട്](https://aka.ms/azure-free-account) ആരംഭിക്കുക.


## GitHub Copilot ഉപയോഗിച്ചുള്ള പഠനം

വേഗത്തിൽ കോഡിംഗ് ആരംഭിക്കാൻ, GitHub Codespace-ഇൽ അല്ലെങ്കിൽ നിങ്ങളുടെ പ്രാദേശിക IDE-യിൽ ഈ പ്രോജക്ട് devcontainer-ഉം ഉപയോഗിച്ച് തുറക്കുക. ഈ കോഴ്സിൽ ഉപയോഗിക്കുന്ന devcontainer GitHub Copilot കൃത്രിമ ബുദ്ധി കൂട്ടുകാർ പ്രോഗ്രാമിംഗിനായി മുൻകൂട്ടി കോൺഫിഗർ ചെയ്തിരിക്കുന്നു.

ഓരോ കോഡ് ഉദാഹരണത്തിനും GitHub Copilot-നെ ചോദിച്ചറിയാനാകുന്ന നിർദ്ദേശിച്ച ചോദ്യങ്ങൾ ഉൾപ്പെടുന്നു. 💡/🤖 പ്രോംപ്റ്റുകൾ താഴെ കാണാം:

- **ജാവ ഫയല് ഹെഡറുകൾ** - ഓരോ ഉദാഹരണത്തിനും പ്രത്യേക ചോദ്യങ്ങൾ  
- **മോഡ്യൂൾ READMEകൾ** - കോഡ് ഉദാഹരണങ്ങൾക്കു ശേഷം അവലോകന പ്രോംപ്റ്റുകൾ  

**ഉപയോഗ മാർഗം:** ഏതെങ്കിലും കോഡ് ഫയൽ തുറന്ന് നിർദ്ദേശിച്ച ചോദ്യങ്ങൾ Copilot-നോട് ചോദിക്കുക. കോഡ് ബേസിന്റെ പൂർണ്ണ സാന്ദർഭ്യം അറിയുന്ന Copilot വിശദീകരിക്കുകയും വീതപ്പെടുത്തുകയും മറ്റു മാർഗങ്ങൾ നിർദ്ദേശിക്കുകയും ചെയ്യാം.

കൂടുതൽ അറിയണമോ? [AI കൂട്ടുകാർ പ്രോഗ്രാമിംഗിനായി Copilot](https://aka.ms/GitHubCopilotAI) നോക്കുക.
