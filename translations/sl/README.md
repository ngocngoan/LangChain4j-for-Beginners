# LangChain4j za začetnike

Tečaj za izdelavo AI aplikacij z LangChain4j in Azure OpenAI GPT-5.2, od osnovnega klepeta do AI agentov.

**Nov v LangChain4j?** Oglejte si [Slovar](docs/GLOSSARY.md) za definicije ključnih izrazov in konceptov.

## Vsebina

1. [Hiter začetek](00-quick-start/README.md) - Začnite z LangChain4j
2. [Uvod](01-introduction/README.md) - Spoznajte osnove LangChain4j
3. [Inženiring pozivov](02-prompt-engineering/README.md) - Obvladajte učinkovito oblikovanje pozivov
4. [RAG (generiranje z dodatnim iskanjem informacij)](03-rag/README.md) - Zgradite pametne sisteme na osnovi znanja
5. [Orodja](04-tools/README.md) - Integrirajte zunanja orodja in preproste asistente
6. [MCP (Model Context Protocol)](05-mcp/README.md) - Delo z Model Context Protocol (MCP) in agentnimi moduli
---

## Učni načrt

> **Hiter začetek**

1. Forkajte ta repozitorij na svoj GitHub račun
2. Kliknite **Code** → zavihek **Codespaces** → **...** → **New with options...**
3. Uporabite privzete nastavitve – izbrana bo razvojna vsebina, ustvarjena za ta tečaj
4. Kliknite **Create codespace**
5. Počakajte 5-10 minut, da bo okolje pripravljeno
6. Takoj začnite z [Hitrim začetkom](./00-quick-start/README.md)!

Po zaključku modulov si oglejte [Vodnik za testiranje](docs/TESTING.md), kjer boste videli koncepte testiranja LangChain4j v praksi.

> **Opomba:** Ta izobraževalni tečaj uporablja tako GitHub modele kot Azure OpenAI. Modul [Hiter začetek](00-quick-start/README.md) uporablja GitHub modele (ni potrebna naročnina na Azure), medtem ko moduli 1-5 uporabljajo Azure OpenAI. Začnite z [BREZPLAČNIM Azure računom](https://aka.ms/azure-free-account), če ga še nimate.


## Učenje z GitHub Copilot

Za hiter začetek s kodiranjem odprite ta projekt v GitHub Codespace ali v svojem lokalnem IDE z ustrezno devcontainer konfiguracijo. Devcontainer, uporabljen v tem tečaju, je vnaprej nastavljen s GitHub Copilot za AI programsko sodelovanje.

Vsak primer kode vključuje predlagana vprašanja, ki jih lahko zastavite GitHub Copilotu za poglobitev svojega razumevanja. Iščite namige 💡/🤖 v:

- **Glave posameznih Java datotek** - vprašanja, specifična za vsak primer
- **README moduli** - raziskovalni namigi po primerih kode

**Kako uporabljati:** Odprite katero koli datoteko s kodo in Copilotu zastavite predlagana vprašanja. Ima popoln kontekst celotne kode in lahko pojasni, razširi ali predlaga alternative.

Želite izvedeti več? Oglejte si [Copilot za AI programsko sodelovanje](https://aka.ms/GitHubCopilotAI).
