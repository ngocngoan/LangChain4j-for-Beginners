# LangChain4j ಆರಂಭಿಕರಿಗಾಗಿ

ಬೆಸಿಕ್ ಚಾಟ್‌ನಿಂದ ಗೆ AI ಏಜೆಂಟ್‌ಗಳವರೆಗೆ LangChain4j ಮತ್ತು Azure OpenAI GPT-5.2 ಬಳಸಿ AI ಅಪ್ಲಿಕೇಶನ್‌ಗಳನ್ನು ನಿರ್ಮಿಸುವ ಕೋರ್ಸ್.

**LangChain4j ಅನ್ನು ಹೊಸದಾಗಿ ಬಳಸುತ್ತಿದ್ದೀರಾ?** ಪ್ರಮುಖ ಪದಗಳು ಮತ್ತು ಧಾರಣೆಗಳ ವ್ಯಾಖ್ಯಾನಗಳಿಗೆ [Glossary](docs/GLOSSARY.md) ನೋಡಿಕೊಳ್ಳಿ.

## ವಿಷಯಗಳ ಪಟ್ಟಿಕೆ

1. [ತ್ವರಿತ ಪ್ರಾರಂಭ](00-quick-start/README.md) - LangChain4j ಪ್ರಾರಂಭಿಸಿ
2. [ಪರಿಚಯ](01-introduction/README.md) - LangChain4j ಮೂಲಭೂತಗಳನ್ನು ಕಲಿಯಿರಿ
3. [ಪ್ರಾಂಪ್ಟ್ ಎಂಜಿನಿಯರಿಂಗ್](02-prompt-engineering/README.md) - ಪರಿಣಾಮಕಾರಿ ಪ್ರಾಂಪ್ಟ್ ವಿನ್ಯಾಸದಲ್ಲಿ ನಿಪುಣರಾಗಿ
4. [RAG (Retrieval-Augmented Generation)](03-rag/README.md) - ಬುದ್ಧಿವಂತ ಜ್ಞಾನ ಆಧಾರಿತ ಸಿಸ್ಟಮ್‌ಗಳನ್ನು ನಿರ್ಮಿಸಿ

## ಕಲಿಕಾ ಮಾರ್ಗ

> **ತ್ವರಿತ ಪ್ರಾರಂಭ**

1. ಈ ಸಂಗ್ರಹಿಕೆಯನ್ನು ನಿಮ್ಮ GitHub ಖಾತೆಗೆ Fork ಮಾಡಿ
2. **Code** → **Codespaces** ಟ್ಯಾಬ್ → **...** → **New with options...** ಕ್ಲಿಕ್ ಮಾಡಿ
3. ಡೀಫಾಲ್ಟ್ ಆಯ್ಕೆಗಳನ್ನು ಬಳಸಿರಿ – ಇದು ಈ ಕೋರ್ಸ್‌ಗೆ ಸೃಷ್ಟಿಸಲಾದ ಡೆವಲಪ್‌ಮೆಂಟ್ ಕಂಟೈನರ್ ಅನ್ನು ಆರಿಸುತ್ತದೆ
4. **Create codespace** ಕ್ಲಿಕ್ ಮಾಡಿ
5. ಪರಿಸರ ಸಿದ್ಧವಾಗಲು 5-10 ನಿಮಿಷಗಳು ಕಾಯಿರಿ
6. ತಕ್ಷಣವೇ [ತ್ವರಿತ ಪ್ರಾರಂಭ](./00-quick-start/README.md) ಗೆ ಹೋಗಿ ಶುರುಮಾಡಿ!

ಮೊಡ್ಯೂಲ್‌ಗಳನ್ನು ಪೂರ್ಣಗೊಳಿಸಿದ ನಂತರ, LangChain4j ಪರೀಕ್ಷಾ ಕನ್ಸೆಪ್ಟ್‌ಗಳನ್ನು ಕಾರ್ಯರೂಪದಲ್ಲಿ ನೋಡಲು [Testing Guide](docs/TESTING.md) ಅನ್ನು ಅನ್ವೇಷಿಸಿ.

> **ಗಮನಿಸಿ:** ಈ ತರಬೇತಿಯಲ್ಲಿದೆ GitHub ಮಾದರಿಗಳನ್ನು ಮತ್ತು Azure OpenAI ಎರಡನ್ನೂ ಬಳಸುತ್ತದೆ. [ತ್ವರಿತ ಪ್ರಾರಂಭ](00-quick-start/README.md) ಮೋಡ್ಯೂಲ್ GitHub ಮಾದರಿಗಳನ್ನು ಬಳಕೆಮಾಡುತ್ತದೆ (Azure ಸಬ್‌ಸ್ರಿಪ್ಷನ್ ಅಗತ್ಯವಿಲ್ಲ), ಆದರೆ 1-5 ಮೋಡ್ಯೂಲ್‌ಗಳು Azure OpenAI ಅನ್ನು ಬಳಕೆಮಾಡುತ್ತವೆ. ಒಂದು [FREE Azure ಖಾತೆ](https://aka.ms/azure-free-account) ಇಲ್ಲದಿದ್ದರೆ ಆರಂಭಿಸಿ.

## GitHub Copilot ಜೊತೆಗೆ ಕಲಿಕೆ

ತ್ವರಿತವಾಗಿ ಕೋಡಿಂಗ್ ಪ್ರಾರಂಭಿಸಲು, ಈ ಪ್ರಾಜೆಕ್ಟ್ ಅನ್ನು GitHub Codespace ಅಥವಾ ಒದಗಿಸಲಾದ devcontainer ಇರುವ ಸ್ಥಳೀಯ IDE ನಲ್ಲಿ ತೆರವಿರಿ. ಈ ಕೋರ್ಸ್‌ನಲ್ಲಿ ಉಪಯೋಗಿಸಲಾಗುವ devcontainer GitHub Copilot ಸಜ್ಜುಗೊಳಿಸಿರುವುದರಿಂದ AI ಜೋಡಿ ಪ್ರೋಗ್ರಾಮಿಂಗ್ ಸಾಧ್ಯ.

ಪ್ರತೀ ಕೋಡ್ ಉದಾಹರಣೆಯಲ್ಲಿ GitHub Copilot ಗೆ ಕೇಳಬಹುದಾದ ಪ್ರಸ್ತಾಪಿತ ಪ್ರಶ್ನೆಗಳು ಒಳಗೊಂಡಿವೆ, ಅವು ನಿಮ್ಮ ಅರ್ಥೈಸುವಿಕೆಯನ್ನು ಗಾಢಗೊಳಿಸುತ್ತವೆ. ಕೆಳಗಿನವುಗಳಲ್ಲಿ 💡/🤖 ಸೂಚನೆಗಳನ್ನು ನೋಡಿ:

- **Java ಫೈಲ್ ಹೆಡರ್‌ಗಳು** - ಪ್ರತಿ ಉದಾಹರಣೆಗೆ ವಿಶಿಷ್ಟ ಪ್ರಶ್ನೆಗಳು
- **ಮಾಡ್ಯೂಲ್ READMEs** - ಕೋಡ್ ಉದಾಹರಣೆಗಳ ನಂತರ ಅನ್ವೇಷಣೆ ಪ್ರಾಂಪ್ಟ್‌ಗಳು

**ಬಳಸದ ವಿಧಾನ:** ಯಾವುದಾದರೂ ಕೋಡ್ ಫೈಲ್ ತೆರೆಯಿರಿ ಮತ್ತು Copilot ಗೆ ಪ್ರಸ್ತಾಪಿತ ಪ್ರಶ್ನೆಗಳನ್ನು ಕೇಳಿ. ಇದಕ್ಕೆ ಕೋಡ್ ಬೇಸ್ ಪೂರ್ಣ ಸಂಧರ್ಭ ಇದೆ ಮತ್ತು ಇದು ವಿವರಿಸಬಹುದು, ವಿಸ್ತರಿಸಬಹುದು ಮತ್ತು ಪರ್ಯಾಯಗಳನ್ನು ಸೂಚಿಸಬಹುದು.

ಹೆಚ್ಚು ತಿಳಿದುಕೊಳ್ಳಬೇಕೆ? [AI ಜೋಡಿ ಪ್ರೋಗ್ರಾಮಿಂಗ್ Copilot](https://aka.ms/GitHubCopilotAI) ನೋಡಿರಿ.
