## ಕಲಿಕಾ ಮಾರ್ಗ

**LangChain4j ಹೊಸದಿದ್ದೀರಾ?** ಪ್ರಮುಖ ಪದಗಳು ಮತ್ತು ತತ್ವಗಳುಗಾಗಿ [ಶಬ್ದಕೋಶ](docs/GLOSSARY.md) ನೋಡಿ.

> **ತ್ವರಿತ ಪ್ರಾರಂಭ**

1. ಈ ರೆಪೊ ರೈಲನ್ನು ನಿಮ್ಮ GitHub ಖಾತೆಗೆ ಫೋರ್ಕ್ ಮಾಡಿಕೊಳ್ಳಿ
2. **Code** → **Codespaces** ಟ್ಯಾಬ್ → **...** → **New with options...** ಕ್ಲಿಕ್ ಮಾಡಿ
3. ಡೀಫಾಲ್ಟ್ ಗಳನ್ನು ಬಳಸಿ – ಇದು ಈ ಕೋರ್ಸ್ ಗಾಗಿ ಸೃಷ್ಟಿಸಲಾದ ಡೆವಲಪ್‌ಮೆಂಟ್ ಕಂಟೈನರ್ ಆಯ್ಕೆ ಮಾಡುತ್ತದೆ
4. **Create codespace** ಕ್ಲಿಕ್ ಮಾಡಿ
5. ಪರಿಸರ ಸಿದ್ಧವಾಗಲು 5-10 ನಿಮಿಷ ಕಾಯಿರಿ
6. ನೇರವಾಗಿ [ತ್ವರಿತ ಪ್ರಾರಂಭ](./00-quick-start/README.md) ಗೆ જતા ಪ್ರಾರಂಭಿಸಿ!

ಘಟಕಗಳನ್ನು ಪೂರ್ಣಗೊಳಿಸಿದ ಬಳಿಕ, LangChain4j ಪರೀಕ್ಷಾ ತತ್ವಗಳನ್ನು ಅನುಭವಿಸಲು [ಪರೀಕ್ಷಾ ಮಾರ್ಗದರ್ಶಿ](docs/TESTING.md) ಪರಿಶೀಲಿಸಿ.

> **ಗಮನಿಸಿ:** ಈ ತರಬೇತಿಯಲ್ಲಿ GitHub ಮಾದರಿಗಳು ಮತ್ತು Azure OpenAI ಎರಡೂ ಉಪಯೋಗಿಸುತ್ತವೆ. [ತ್ವರಿತ ಪ್ರಾರಂಭ](00-quick-start/README.md) ಘಟಕ GitHub ಮಾದರಿಗಳನ್ನು ಬಳಸುತ್ತದೆ (Azure ಸಬ್ಸ್ಕ್ರಿಪ್ಷನ್ ಬೇಕಾಗಿಲ್ಲ), ಆದರೆ 1-5 ಘಟಕಗಳು Azure OpenAI ಬಳಕೆ ಮಾಡುತ್ತವೆ. ನಿಮ್ಮ ಬಳಿ ಇಲ್ಲದಿದ್ದರೆ [ಉಚಿತ Azure ಖಾತೆಯನ್ನು](https://aka.ms/azure-free-account) ಪಡೆಯಿರಿ.

## GitHub Copilot ಜೊತೆಗೆ ಕಲಿಕೆ

ತ್ವರಿತವಾಗಿ ಕೋಡಿಂಗ್ ಪ್ರಾರಂಭಿಸಲು, GitHub Codespace ಅಥವಾ ನಿಮ್ಮ ಸ್ಥಳೀಯ IDEನಲ್ಲಿ ಈ ಪ್ರಾಜೆಕ್ಟ್‍ನ್ನು devcontainer ಜೊತೆಗೆ ತೆರೆಯಿರಿ. ಈ ಕೋರ್ಸ್‍ನಲ್ಲಿ ಬಳಸಿದ devcontainer GitHub Copilot ಜೊತೆಗೆ ಪೂರ್ವರೂಪಗೊಂಡಿದೆ, ಇದು AI ಜೊತೆ ಸಂಯೋಜಿತ ಪ್ರೋಗ್ರಾಮಿಂಗ್ ಅನುಭವವನ್ನು ನೀಡುತ್ತದೆ.

ಪ್ರತಿ ಕೋಡ್ ಉದಾಹರಣೆ GitHub Copilot‍ನಿಂದ ಕೇಳಬಹುದಾದ ಸಲಹೆ ಪ್ರಶ್ನೆಗಳನ್ನು ಒಳಗೊಂಡಿದೆ. ನಿಮ್ಮ ಜ್ಞಾನವನ್ನು ಹೆಚ್ಚು ಗಾಢಗೊಳಿಸಲು ಅವುಗಳನ್ನು ಹುಡುಕಿ:

- **Java ಫೈಲ್ ಶೀರ್ಷಿಕೆಗಳು** - ಪ್ರತಿಯೊಂದು ಉದಾಹರಣೆಗೆ ಸಂಬಂಧಿಸಿದ ಪ್ರಶ್ನೆಗಳು
- **ಘಟಕ READMEಗಳು** - ಕೋಡ್ ಉದಾಹರಣೆಗಳ ನಂತರ ಅನ್ವೇಷಣಾ ಪ್ರಶ್ನೆಗಳು

**ಬಳಸದ ಮுறை:** ಯಾವುದೇ ಕೋಡ್ ಫೈಲ್ ತೆರೆಯಿರಿ ಮತ್ತು Copilotಗೆ ಸೂಚಿಸಿದ ಪ್ರಶ್ನೆಗಳನ್ನು ಕೇಳಿರಿ. ಇದು ಕೋಡ್‍ನ ಸಂಪೂರ್ಣ ಸಂದರ್ಭ ಹೊಂದಿದ್ದು ವಿವರಿಸಲು, ವಿಸ್ತರಿಸಲು ಮತ್ತು ಪರ್ಯಾಯಗಳನ್ನು ಸಲಹೆ ಮಾಡಲು ಸಾಧ್ಯ.

ಇನ್ನಷ್ಟು ತಿಳಿಯಲು [AI ಜೋಡಣೆ ಪ್ರೋಗ್ರಾಮಿಂಗ್ ಗಾಗಿ Copilot](https://aka.ms/GitHubCopilotAI) ನೋಡಿ.
