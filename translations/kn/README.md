# LangChain4j ಪ್ರಾರಂಭಿಕರಿಗಾಗಿ

ಸುಳಿವು ಡೋಲಿಕೆಯಿಂದ ಕೃತಕ ಬುದ್ಧಿಮತ್ತೆ ಅನ್ವಯಗಳನ್ನು ನಿರ್ಮಿಸಲು LangChain4j ಮತ್ತು Azure OpenAI GPT-5 ಬಳಸಿ ಕೋರ್ಸ್.

**LangChain4j ಬಗ್ಗೆ ಹೊಸದಾಗಿದ್ದೀರಾ?** ಪ್ರಮುಖ ಪದಗಳ ಮತ್ತು ಧಾರಣೆಗಳ ವ್ಯಾಖ್ಯಾನಗಳಿಗೆ [ಪದಕೋಶ](docs/GLOSSARY.md) ನೋಡಿ.

## ವಿಷಯಾನುಕ್ರಮಣಿಕೆ

1. [ತ್ವರಿತ ಪ್ರಾರಂಭ](00-quick-start/README.md) - LangChain4j ಆಯುಕ್ತರಾಗಿ ಪ್ರಾರಂಭಿಸಿ
2. [ಪರಿಚಯ](01-introduction/README.md) - LangChain4j ಮೂಲಭೂತಗಳನ್ನು ಕಲಿಯಿರಿ
3. [ಪ್ರಾಂಪ್ಟ್ ಇಂಜಿನಿಯರಿಂಗ್](02-prompt-engineering/README.md) - ಪರಿಣಾಮಕಾರಿಯಾದ ಪ್ರಾಂಪ್ಟ್ ವಿನ್ಯಾಸದಲ್ಲಿ ಪರಿಣತಿ ಹೊಂದಿರಿ
4. [RAG (ರಿಟ್ರಿವಲ್-ಆಗ್ಮೆಂಟೆಡ್ ಜನರೇಷನ್)](03-rag/README.md) - ಬುದ್ಧಿವಂತ ಸರಕಾರ-ಆಧಾರಿತ ವ್ಯವಸ್ಥೆಗಳನ್ನು ನಿರ್ಮಿಸಿ
5. [ಟೂಲ್‌ಗಳು](04-tools/README.md) - ಬಾಹ್ಯ ಸಾಧನಗಳು ಮತ್ತು ಸರಳ ಸಹಾಯಕರನ್ನು ಜೋಡಿಸಿ
6. [MCP (ಮಾದರಿ ಸಂದರ್ಭ ಪ್ರೋಟೋಕಾಲ್)](05-mcp/README.md) - ಮಾದರಿ ಸಂದರ್ಭ ಪ್ರೋಟೋಕಾಲ್ (MCP) ಮತ್ತು ಏಜೆಂಟಿಕ್ ಮೋಡ್ಯೂಲ್‌ಗಳೊಂದಿಗೆ ಕೆಲಸ ಮಾಡಿರಿ

## ಅಧ್ಯಯನ ಮಾರ್ಗ

> **ತ್ವರಿತ ಪ್ರಾರಂಭ**

1. ಈ ರೆಪೊಸ್ಟೋರಿಯನ್ನು ನಿಮ್ಮ GitHub ಖಾತೆಗೆ ಫೋರ್ಕ್ ಮಾಡಿ
2. **Code** → **Codespaces** ಟ್ಯಾಬ್ → **...** → **New with options...** ಕ್ಲಿಕ್ ಮಾಡಿ
3. ಡೀಫಾಲ್ಟ್‌ಗಳನ್ನು ಬಳಸಿ – ಇದು ಈ ಕೋರ್ಸ್‌ಗಾಗಿ ನಿರ್ಮಿಸಿದ ಡೆವಲಪ್ಮೆಂಟ್ ಕಂಟೇನರ್ ಆಯ್ಕೆ ಮಾಡುತ್ತದೆ
4. **Create codespace** ಕ್ಲಿಕ್ ಮಾಡಿ
5. ವಾತಾವರಣ ಸಿದ್ಧವಾಗಲು 5-10 ನಿಮಿಷಗಳಷ್ಟು ಕಾಯಿರಿ
6. ತಕ್ಷಣವೇ ಪ್ರಾರಂಭಿಸಲು [ತ್ವರಿತ ಪ್ರಾರಂಭ](./00-quick-start/README.md) ಗೆ ಹಾರಿಸಿ!

ಮಾಡ್ಯೂಲ್‌ಗಳನ್ನು ಪೂರ್ಣಗೊಳಿಸಿದ ನಂತರ, LangChain4j ಪರೀಕ್ಷಾ ತತ್ವಗಳನ್ನು ನೇರವಾಗಿ ಅನುಭವಿಸಲು [ಪರೀಕ್ಷಾ ಮಾರ್ಗದರ್ಶಿ](docs/TESTING.md) ತಪಾಸಿಸಿ.

> **ಸೂಚನೆ:** ಈ ತರಬೇತಿಯಲ್ಲಿ GitHub Models ಮತ್ತು Azure OpenAI ಎರಡು ಕೂಡ ಬಳಸಲಾಗುತ್ತವೆ. [ತ್ವರಿತ ಪ್ರಾರಂಭ](00-quick-start/README.md) ಮಾಡ್ಯೂಲ್ GitHub Models ಬಳಕೆ ಮಾಡುತ್ತದೆ (Azure ಸಬ್ಸ್ಕ್ರಿಪ್ಶನ್ ಅಗತ್ಯವಿಲ್ಲ), ಆದರೆ ಮಾಡ್ಯೂಲ್ 1-5 Azure OpenAI ಬಳಕೆ ಮಾಡುತ್ತವೆ. ನಿಮ್ಮ ಬಳಿ ಇಲ್ಲದಿದ್ದರೆ [ಮುಕ್ತ Azure ಖಾತೆಯೊಂದಿಗೆ](https://aka.ms/azure-free-account) ಪ್ರಾರಂಭಿಸಿ.


## GitHub Copilot ಜೊತೆಗೆ ಅಧ್ಯಯನ

ತ್ವರಿತವಾಗಿ ಕೋಡಿಂಗ್ ಪ್ರಾರಂಭಿಸಲು, ಈ ಯೋಜನೆಯನ್ನು GitHub Codespace ನಲ್ಲಿ ಅಥವಾ ನಿಮ್ಮ ಸ್ಥಳೀಯ IDE ನಲ್ಲಿ ನೀಡಲಾದ ಡೆವ್‌ಕಂಟೇನರ್ ಬಳಸಿ ತೆರೆಯಿರಿ. ಈ ಕೋರ್ಸ್‌ನಲ್ಲಿ ಬಳಕೆಯ ಡೆವ್‌ಕಂಟೇನರ್ GitHub Copilot ಸಹಿತ ಪೂರ್ವ-ಕಾನ್ಫಿಗರ್ ಮಾಡಲ್ಪಟ್ಟಿದೆ, ಇದು AI ಜೋಡಣ برنامه ನಿಮಗೆ ಸಹಾಯ ಮಾಡುತ್ತದೆ.

ಪ್ರತಿ ಕೋಡ್ ಉದಾಹರಣೆಯಲ್ಲೂ GitHub Copilot ನಿಂದ ಕೇಳಬಹುದಾದ ಪ್ರಶ್ನೆಗಳಿವೆ, ಅದು ನಿಮ್ಮ ಅರ್ಥಮಾಡಿಕೊಳ್ಳುವಿಕೆಯನ್ನು ಗಾಢಗೊಳಿಸಲು ಸಹಾಯ ಮಾಡುತ್ತದೆ. ಕೆಳಗಿನ 💡/🤖 ಸೂಚನೆಗಳಿಗಾಗಿ ಹುಡುಕಿ:

- **ಜावा ಫೈಲ್ ಶೀರ್ಷಿಕೆಗಳು** - ಪ್ರತಿಯೊಂದು ಉದಾಹರಣೆಗೆ ವಿಶೇಷವಾದ ಪ್ರಶ್ನೆಗಳು
- **ಮಾಡ್ಯೂಲ್ README ಗಳು** - ಕೋಡ್ ಉದಾಹರಣೆಗಳ ನಂತರ ಅನ್ವೇಷಣೆ ಸೂಚನೆಗಳು

**ಬಳಸುವುದು ಹೇಗೆ:** ಯಾವುದೇ ಕೋಡ್ ಫೈಲ್ ತೆರೆಯಿರಿ ಮತ್ತು Copilot ಗೆ ಸೂಚಿಸಲಾದ ಪ್ರಶ್ನೆಗಳನ್ನು ಕೇಳಿ. ಅದು ಸಂಪೂರ್ಣ ಕೋಡ್‌ಬೇಸ್ ಹಿನ್ನೆಲೆಯನ್ನು ಹೊಂದಿದೆ ಮತ್ತು ವಿವರಣೆ, ವಿಸ್ತರಣೆ ಮತ್ತು ಪರ್ಯಾಯಗಳನ್ನು ಸೂಚಿಸಬಹುದು.

ಇನ್ನಷ್ಟು ತಿಳಿಯೋಣವೇ? [AI ಜೋಡಣ 프로그램ಿಗಾಗಿ Copilot](https://aka.ms/GitHubCopilotAI) ನೋಡಿರಿ.
