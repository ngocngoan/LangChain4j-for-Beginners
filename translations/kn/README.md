# ಪ್ರಾರಂಭಿಕರಿಗಾಗಿ LangChain4j

LangChain4j ಮತ್ತು Azure OpenAI GPT-5.2 ಬಳಸಿ AI ಆಪ್ಲಿಕೇಷನ್‌ಗಳನ್ನು ನಿರ್ಮಿಸುವ ಕೋರ್ಸ್, ಮೂಲ ಚಾಟ್‌ನಿಂದ AI 에ಜೆಂಟ್‌ಗಳ ವರೆಗೆ.

**LangChain4j ಹೊಸದಾಗಿದ್ದೀರಾ?** ಪ್ರಮುಖ ಪದಗಳು ಮತ್ತು ಸಂಪ್ರದಾಯಗಳ ವ್ಯಾಖ್ಯಾನಗಳಿಗೆ [ಶಬ್ದಕೋಶ](docs/GLOSSARY.md) ಅನ್ನು ಪರಿಶೀಲಿಸಿ.

## ಕಲಿಕೆಯ ಮಾರ್ಗ

> **ತ್ವರಿತ ಪ್ರಾರಂಭ**

1. ಈ ರಿಪೊಸಿಟರಿಯನ್ನು ನಿಮ್ಮ GitHub ಖಾತೆಗೆ Fork ಮಾಡಿ
2. **Code** → **Codespaces** ಟ್ಯಾಬ್ → **...** → **New with options...** ಕ್ಲಿಕ್ ಮಾಡಿ
3. ಡೀಫಾಲ್ಟ್‌ಗಳನ್ನು ಬಳಸಿ – ಇದು ಈ ಕೋರ್ಸ್‌ಗೆ ರಚಿಸಲಾದ ಡೆವಲಪ್‌ಮೆಂಟ್ ಕಂಟೈನರ್ ಆಯ್ಕೆಮಾಡುತ್ತದೆ
4. **Create codespace** ಕ್ಲಿಕ್ ಮಾಡಿ
5. ಪರಿಸರ ಸಿದ್ದವಾಗಲು 5-10 ನಿಮಿಷ ಕಾಯಿರಿ
6. ತಕ್ಷಣವೇ [ತ್ವರಿತ ಪ್ರಾರಂಭ](./00-quick-start/README.md) ಗೆ ಹೋಗಿ ಪ್ರಾರಂಭಿಸಿ!

ಘಟಕಗಳನ್ನು ಪೂರ್ಣಗೊಳಿಸಿದ ನಂತರ, LangChain4j ಪರೀಕ್ಷಾ ಆಲೋಚನೆಗಳನ್ನು عملಗೊಳಿಸಲು [ಪರೀಕ್ಷಾ ಮಾರ್ಗದರ್ಶಿ](docs/TESTING.md) ಅನ್ನು ಅನ್ವೇಷಿಸಿ.

> **ಗಮನಿಸಿ:** ಈ ತರಬೇತಿಯಲ್ಲಿ GitHub ಮಾದರಿಗಳು ಮತ್ತು Azure OpenAI ಎರಡೂ ಬಳಸಲಾಗುತ್ತವೆ. [ತ್ವರಿತ ಪ್ರಾರಂಭ](00-quick-start/README.md) ಘಟಕವು GitHub ಮಾದರಿಗಳನ್ನು ಬಳಸುತ್ತದೆ (Azure ಸಬ್ಸ್ಕ್ರಿಪ್ಷನ್ ಅಗತ್ಯವಿಲ್ಲ), ಆದರೆ ಅಂಶಗಳು 1-5 Azure OpenAI ಉಪಯೋಗಿಸುತ್ತವೆ. ನೀವು ಇರದಿದ್ದರೆ [ ಉಚಿತ Azure ಖಾತೆಯನ್ನು](https://aka.ms/azure-free-account) ಪ್ರಾರಂಭಿಸಿ.


## GitHub Copilot ಜೊತೆ ಕಲಿಕೆ

ತ್ವರಿತವಾಗಿ ಕೋಡಿಂಗ್ ಪ್ರಾರಂಭಿಸಲು, ಈ ಪ್ರಾಜೆಕ್ಟ್ ಅನ್ನು GitHub Codespace ಅಥವಾ ನಿಮ್ಮ ಸ್ಥಳೀಯ IDE ನಲ್ಲಿ devcontainer ಜೊತೆ ತೆರೆಯಿರಿ. ಈ ಕೋರ್ಸ್‌ನಲ್ಲಿ ಬಳಸಲಾಗುವ devcontainer GitHub Copilot ನೊಂದಿಗೆ ಪೂರ್ವ ಸಂರಚಿತವಾಗಿದೆ AI ಜೋಡಣೆ ಕಾರ್ಯಕ್ರಮಗಳಿಗೆ.

ಪ್ರತಿ ಕೋಡ್ ಉದಾಹರಣೆಯಲ್ಲಿ GitHub Copilot ನೊಂದಿಗೆ ನಿಮ್ಮ ಅರ್ಥವನ್ನು ಗಾಢಗೊಳಿಸಲು ಕೇಳಬಹುದಾದ ಸಲಹೆ ನಮೂನೆಗಳಿವೆ. 💡/🤖 ಸೂಚನೆಗಳನ್ನು ಹುಡುಕಿರಿ:

- **Java ಫೈಲ್ ಹೆಡರ್‌ಗಳು** - ಪ್ರತಿ ಉದಾಹರಣೆಗೆ ವಿಶಿಷ್ಟ ಪ್ರಶ್ನೆಗಳು
- **ಮಾಡ್ಯೂಲ್ README ಗಳು** - ಕೋಡ್ ಉದಾಹರಣೆಗಳ ನಂತರ ಅನ್ವೇಷಣಾ ಸೂಚನೆಗಳು

**ಬಳಸುವ ವಿಧಾನ:** ಯಾವುದೇ ಕೋಡ್ ಫೈಲ್ ತೆರೆಯಿರಿ ಮತ್ತು Copilot ಗೆ ಸೂಚಿಸಿದ ಪ್ರಶ್ನೆಗಳನ್ನು ಕೇಳಿ. ಇದರ ಬಳಿ ಸಂಪೂರ್ಣ ಕೋಡ್‌ಬೇಸ್ ಪ್ರಾಸಂಗಿಕತೆ ಇರುತ್ತದೆ ಮತ್ತು ಇದು ವಿವರಣೆ, ವಿಸ್ತರಣೆ ಹಾಗೂ ಪರ್ಯಾಯ ಸಲಹೆಗಳನ್ನು ನೀಡಬಹುದು.

ಹೆಚ್ಚು ತಿಳಿಯಲು ಬಯಸುತ್ತೀರಾ? [AI ಜೊತೆಗೆ Copilot](https://aka.ms/GitHubCopilotAI) ಅನ್ನು ಪರಿಶೀಲಿಸಿ.
