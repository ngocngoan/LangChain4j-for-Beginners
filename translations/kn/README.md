## ಅಧ್ಯಯನ ಮಾರ್ಗ

**LangChain4jನಲ್ಲಿ ಹೊಸವರು?** ಮುಖ್ಯ ಪದಗಳು ಮತ್ತು ಸಂಜ್ಞೆಗಳ ವ್ಯಾಖ್ಯಾನಕ್ಕಾಗಿ [ಕೋಶ](docs/GLOSSARY.md) ನೋಡಿ.

> **ತ್ವರಿತ ಪ್ರಾರಂಭ**

1. ಈ ರೆಪೊವನ್ನು ನಿಮ್ಮ GitHub ಖಾತೆಗೆ Fork ಮಾಡಿ
2. **Code** → **Codespaces** ಟ್ಯಾಬ್ → **...** → **ಹೊಸ ಆಯ್ಕೆಯೊಂದಿಗೆ...** ಮೇಲೆ ಕ್ಲಿಕ್ ಮಾಡಿ
3. ಪೂರ್ವನಿಗದಿತವನ್ನು ಬಳಸಿ – ಇದು ಈ ಕೋರ್ಸಿಗೆ ತಯಾರಿಸಿದ ಡೆವಲಪ್ಮೆಂಟ್ ಕಾಂಟೇನರ್ ಆಯ್ಕೆ ಮಾಡುತ್ತದೆ
4. **Codespace ರಚಿಸಿ** ಕ್ಲಿಕ್ ಮಾಡಿ
5. ಪರಿಸರ ಸಿದ್ಧವಾಗಲು 5-10 ನಿಮಿಷ ಕಾಯಿರಿ
6. ತಕ್ಷಣವೇ ಪ್ರಾರಂಭಿಸಲು [ತ್ವರಿತ ಪ್ರಾರಂಭ](./00-quick-start/README.md) ಗೆ ಜಂಪ್ ಮಾಡಿ!

ಮಡ್ಯೂಲ್‌ಗಳನ್ನು ಪೂರ್ಣಗೊಳಿಸಿದ ನಂತರ, LangChain4j ಟೆಸ್ಟಿಂಗ್ ತತ್ತ್ವಗಳನ್ನು ಕ್ರಿಯಾಶೀಲವಾಗಿ ನೋಡಲು [ಟೆಸ್ಟಿಂಗ್ ಗೈಡ್](docs/TESTING.md) ಅನ್ನು ಅನ್ವೇಷಿಸಿ.

> **ಗಮನಿಸಿ:** ಈ ತರಬೇತಿ GitHub ಮಾದರಿಗಳ ಮತ್ತು Azure OpenAI ಎರಡನ್ನೂ ಬಳಸುತ್ತದೆ. [ತ್ವರಿತ ಪ್ರಾರಂಭ](00-quick-start/README.md) ಮಡ್ಯೂಲ್ GitHub ಮಾದರಿಗಳನ್ನು ಬಳಸುತ್ತದೆ (Azure ಸಬ್ಸ್ಕ್ರಿಪ್ಷನ್ ಅಗತ್ಯವಿಲ್ಲ), ಆದರೆ 1-5 ಮಡ್ಯೂಲ್‌ಗಳು Azure OpenAI ಬಳಕೆ ಮಾಡುತ್ತವೆ. ನೀವು ಹೊಂದಿಲ್ಲದಿದ್ದರೆ [ಉಚಿತ Azure ಖಾತೆ](https://aka.ms/azure-free-account) ಜೊತೆಗೆ ಪ್ರಾರಂಭಿಸಿ.


## GitHub Copilot ಜೊತೆಗೆ ಅಧ್ಯಯನ

ತ್ವರಿತವಾಗಿ ಕೊಂಡಿಡಿಂಗ್ ಮಾಡಲು, ಈ ಯೋಜನೆಯನ್ನು GitHub Codespace ಅಥವಾ ಸ್ಥಳೀಯ IDE ನಲ್ಲಿ devcontainer ಜೊತೆಗೆ ತೆರೆಯಿರಿ. ಈ ಕೋರ್ಸ್‌ನಲ್ಲಿ ಬಳಸಲಾಗುವ devcontainer GitHub Copilot ನೊಂದಿಗೆ ಪೂರ್ವ-ಕಾನ್ಫಿಗರ್ ಆಗಿದೆ AI ಜೋಡಿನ ಪ್ರೋಗ್ರಾಮಿಂಗ್‌ಗೆ.

ಪ್ರತಿ ಕೋಡ್ ಉದಾಹರಣೆಯಲ್ಲಿ GitHub Copilot ಗೆ ಕೇಳಬಹುದಾದ ಪ್ರಸ್ತಾಪಿತ ಪ್ರಶ್ನೆಗಳಿವೆ, ಇದು ನಿಮ್ಮ ಅರ್ಥೈಸಿಕೆಯನ್ನು ಗಾಢವಾಗಿಸುತ್ತದೆ. ಕೆಳಗಿನ ಕಡೆ 💡/🤖 ಸೂಚನೆಗಳನ್ನು ನೋಡಿ:

- **ಜಾವಾ ಫೈಲ್ ಶೀರ್ಷಿಕೆಗಳು** - ಪ್ರತಿ ಉದಾಹರಣೆಗೆ ವಿಶೇಷವಾದ ಪ್ರಶ್ನೆಗಳು
- **ಮಡ್ಯೂಲ್ READMEಗಳು** - ಕೋಡ್ ಉದಾಹರಣೆಯ ನಂತರ ಅನ್ವೇಷಣೆಯ ಪ್ರಸ್ತಾಪಗಳು

**ಹೇಗೆ ಬಳಕೆ ಮಾಡುವುದು:** ಯಾವುದೇ ಕೋಡ್ ಫೈಲ್ ತೆರೆಯಿರಿ ಮತ್ತು ಪ್ರಸ್ತಾಪಿತ ಪ್ರಶ್ನೆಗಳು Copilot ಗೆ ಕೇಳಿ. ಈ ಹೆಸರು ಹಾಗೂ ಕೋಡ್‌ಬೇಸ್‌ನ ಸಂಪೂರ್ಣ ಪೋಷಣೆಯೊಂದಿಗೆ ಇದೆ ಮತ್ತು ವಿವರಣೆ, ವಿಸ್ತರಣೆ ಮತ್ತು ಪರ್ಯಾಯ ತಿಳಿಸಬಹುದು.

ಇನ್ನಷ್ಟು ತಿಳಿಯಲು ಇಚ್ಛೆಯಿದೆಯೇ? ನೋಡಿ [AI ಜೋಡಿನ ಪ್ರೋಗ್ರಾಮಿಂಗ್ Copilot](https://aka.ms/GitHubCopilotAI).
