## ಕಲಿಕೆ ಮಾರ್ಗ

**LangChain4j‌ನಲ್ಲಿ ಹೊಸ ಹೆಜ್ಜೆ ಹಾಕುತ್ತಿದ್ದೀರಾ?** ಪ್ರಮುಖ ಪದಗಳು ಮತ್ತು ಆಲೋಚನೆಗಳ ವ್ಯಾಖ್ಯಾನಕ್ಕಾಗಿ [ವಾಕ್ಯಕೋಶ](docs/GLOSSARY.md) ನೋಡಿ.

> **ತ್ವರಿತ ಪ್ರಾರಂಭ**

1. ಈ ಸಂಗ್ರಹವನ್ನು ನಿಮ್ಮ GitHub ಖಾತೆಗೆ ಫೋರ್ಕ್ ಮಾಡಿ
2. **Code** → **Codespaces** ಟ್ಯಾಬ್ → **...** → **New with options...** ಕ್ಲಿಕ್ ಮಾಡಿ
3. ಡೀಫಾಲ್ಟ್‌ಗಳನ್ನು ಬಳಸಿ – ಇದು ಈ ಕೋರ್ಸ್‌ಗೆ ನಿರ್ಮಿಸಲಾಗಿರುವ ಡೆವಲಪ್‌ಮೆಂಟ್ ಕಂಟೈನರ್ ಆಯ್ಕೆ ಮಾಡುತ್ತದೆ
4. **Create codespace** ಕ್ಲಿಕ್ ಮಾಡಿ
5. ಪರಿಸರ ಸಿದ್ಧವಾಗಲು 5-10 ನಿಮಿಷ ಕಾಯಿರಿ
6. ತಕ್ಷಣವೇ ಪ್ರಾರಂಭಿಸಲು [ತ್ವರಿತ ಪ್ರಾರಂಭ](./00-quick-start/README.md)ಕ್ಕೆ ನೇರವಾಗಿ ಹೋಗಿ!

ಮಾಡ್ಯೂಲ್‌ಗಳನ್ನು ಮುಗಿಸಿದ ನಂತರ, LangChain4j ಪರೀಕ್ಷಾ ಆಲೋಚನೆಗಳನ್ನು ಕ್ರಿಯೆಯಲ್ಲಿ ನೋಡಲು [ಟೆಸ್ಟಿಂಗ್ ಗೈಡ್](docs/TESTING.md) ಅನ್ವೇಷಿಸಿ.

> **ಗಮನಿಸಿ:** ಈ ತರಬೇತಿಯಲ್ಲಿ GitHub ಮಾದರಿಗಳು ಮತ್ತು Azure OpenAI ಇಬ್ಬರೂ ಬಳಸಿ ಮಾಡಲಾಗಿದೆ. [ತ್ವರಿತ ಪ್ರಾರಂಭ](00-quick-start/README.md) ಮಾಡ್ಯೂಲ್ GitHub ಮಾದರಿಗಳನ್ನು (Azure ಸಬ್ಸ್ಕ್ರಿಪ್ಷನ್ ಅಗತ್ಯವಿಲ್ಲ) ಬಳಸುತ್ತದೆ, ಆದರೆ 1-5 ಮಾಡ್ಯೂಲ್‌ಗಳು Azure OpenAI ಬಳಕೆ ಮಾಡುತ್ತವೆ. ನಿಮ್ಮ ಬಳಿ ಖಾತೆ ಇಲ್ಲದಿದ್ದರೆ [ಉಚಿತ Azure ಖಾತೆ](https://aka.ms/azure-free-account) ಮೂಲಕ ಪ್ರಾರಂಭಿಸಿ.


## GitHub Copilot ಬಳಸಿ ಕಲಿಕೆ

ತ್ವರಿತವಾಗಿ ಕೇಾಡಿಂಗ್ ಪ್ರಾರಂಭಿಸಲು, ಈ ಪ್ರಾಜೆಕ್ಟ್ ಅನ್ನು GitHub Codespace ಅಥವಾ ನೀಡಲಾದ devcontainer ಜೊತೆಗೆ ನಿಮ್ಮ ಸ್ಥಳೀಯ IDE ನಲ್ಲಿ ತೆರೆಯಿರಿ. ಈ ಕೋರ್ಸ್‌ಗೆ ಬಳಸಲಾದ devcontainer GitHub Copilot ಅನ್ನು AI ಸಾಮೂಹಿಕ ಪ್ರೋಗ್ರಾಮಿಂಗ್‌ಗೆ ಪೂರ್ವ-ಪುಟಕ್ಕೇರಿಸಿದೆ.

ಪ್ರತಿ ಕೇಾಡು ಉದಾಹರಣೆ GitHub Copilot ಗೆ ಕೇಳಬಹುದಾದ ಸಂಶೋಧನಾ ಪ್ರಶ್ನೆಗಳನ್ನು ಒಳಗೊಂಡಿದೆ, ಇದರಿಂದ ನಿಮ್ಮ ಅರ್ಥಮಾಡಿಕೊಳ್ಳುವಿಕೆಯನ್ನು ಗಾಢಗೊಳಿಸಬಹುದು. ಕೆಳಗಿನ ಸ್ಥಳದಲ್ಲಿ 💡/🤖 ಸೂಚನಗಳನ್ನು ಗಮನಿಸಿ:

- **Java ಫೈಲ್ ಹೆಡರ್ಗಳು** - ಪ್ರತಿ ಉದಾಹರಣೆಗೆ ವಿಶೇಷವಾದ ಪ್ರಶ್ನೆಗಳು
- **ಮಾಡ್ಯೂಲ್ READMEಗಳು** - ಕೇಾಡು ಉದಾಹರಣೆಗಳ ನಂತರ ಅನ್ವೇಷಣಾ ಸೂಚನೆಗಳು

**ಬಳಸುವ ವಿಧಾನ:** ಯಾವುದೇ ಕೇಾಡು ಕಡತವನ್ನು ತೆರೆಯಿರಿ ಮತ್ತು Copilot ಗೆ ಸೂಚಿಸಲಾದ ಪ್ರಶ್ನೆಗಳನ್ನು ಕೇಳಿ. ಇದಕ್ಕೆ ಸಂಪೂರ್ಣ ಕೇಾಡ್‌ಬೇಸ್ ಸನ್ನಿವೇಶ ಇದೆ ಮತ್ತು ಇದು ವಿವರಣೆ ಕೊಡಬಹುದು, ವಿಸ್ತರಿಸಬಹುದು ಮತ್ತು ಬದಲಾಯಣೆಗಳನ್ನು ಸಲಹೆ ಮಾಡಬಹುದು.

ಹೆಚ್ಚು ಕಲಿಯಲು ಇಚ್ಛಿಸುವಿರಾ? [AI ಸಮನ್ವಿತ ಪ್ರೋಗ್ರಾಮಿಂಗ್ Copilot](https://aka.ms/GitHubCopilotAI) ನೋಡಿ.
