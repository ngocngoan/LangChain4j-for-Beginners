# LangChain4j per Principianti

Un corso per costruire applicazioni AI con LangChain4j e Azure OpenAI GPT-5.2, dal chat di base agli agenti AI.

**Nuovo a LangChain4j?** Dai un'occhiata al [Glossario](docs/GLOSSARY.md) per definizioni di termini e concetti chiave.

## Indice

1. [Avvio Rapido](00-quick-start/README.md) - Inizia con LangChain4j
2. [Introduzione](01-introduction/README.md) - Impara le basi di LangChain4j
3. [Prompt Engineering](02-prompt-engineering/README.md) - Padroneggia la progettazione efficace dei prompt
4. [RAG (Retrieval-Augmented Generation)](03-rag/README.md) - Costruisci sistemi intelligenti basati sulla conoscenza
5. [Strumenti](04-tools/README.md) - Integra strumenti esterni e assistenti semplici
6. [MCP (Model Context Protocol)](05-mcp/README.md) - Lavora con il Model Context Protocol (MCP) e i moduli agentici
---

## Percorso di Apprendimento

> **Avvio Rapido**

1. Fai fork di questo repository nel tuo account GitHub
2. Clicca **Code** → **Scheda Codespaces** → **...** → **Nuovo con opzioni...**
3. Usa le impostazioni predefinite – questo selezionerà il container di sviluppo creato per questo corso
4. Clicca **Crea codespace**
5. Attendi 5-10 minuti che l'ambiente sia pronto
6. Vai direttamente a [Avvio Rapido](./00-quick-start/README.md) per iniziare!

Dopo aver completato i moduli, esplora la [Guida al Testing](docs/TESTING.md) per vedere in azione i concetti di testing di LangChain4j.

> **Nota:** Questo corso usa sia GitHub Models che Azure OpenAI. Il modulo [Avvio Rapido](00-quick-start/README.md) utilizza GitHub Models (non è richiesta una sottoscrizione Azure), mentre i moduli 1-5 usano Azure OpenAI. Inizia con un [account Azure GRATUITO](https://aka.ms/azure-free-account) se non ne hai uno.


## Apprendimento con GitHub Copilot

Per iniziare rapidamente a programmare, apri questo progetto in un GitHub Codespace o nel tuo IDE locale con il devcontainer fornito. Il devcontainer usato in questo corso viene preconfigurato con GitHub Copilot per la programmazione affiancata dall'AI.

Ogni esempio di codice include domande suggerite che puoi fare a GitHub Copilot per approfondire la comprensione. Cerca i suggerimenti 💡/🤖 in:

- **Intestazioni dei file Java** - Domande specifiche per ogni esempio
- **README dei moduli** - Suggerimenti di esplorazione dopo gli esempi di codice

**Come usarlo:** Apri qualsiasi file di codice e poni le domande suggerite a Copilot. Ha pieno contesto del codice e può spiegare, estendere e suggerire alternative.

Vuoi saperne di più? Dai un'occhiata a [Copilot per la programmazione affiancata dall'AI](https://aka.ms/GitHubCopilotAI).
