# LangChain4j за почетнике

Курс за прављење AI апликација уз LangChain4j и Azure OpenAI GPT-5.2, од основног чата до AI агената.

**Нови сте у LangChain4j?** Погледајте [Глосар](docs/GLOSSARY.md) за дефиниције кључних термина и појмова.

## Садржај

1. [Брзи почетак](00-quick-start/README.md) - Започните са LangChain4j
2. [Увод](01-introduction/README.md) - Савладајте основе LangChain4j
3. [Дизајн упита](02-prompt-engineering/README.md) - Мастер дизајн ефикасних упита
4. [RAG (Retrieval-Augmented Generation)](03-rag/README.md) - Направите интелигентне системе засноване на знању
5. [Алатке](04-tools/README.md) - Интегришите спољашње алатке и једноставне асистенте
6. [MCP (Model Context Protocol)](05-mcp/README.md) - Радите са Model Context Protocol (MCP) и агенцијским модулима
---

## Пут учења

> **Брзи почетак**

1. Форкујте ово складиште на свој GitHub налог
2. Кликните **Code** → таб **Codespaces** → **...** → **New with options...**
3. Користите подразумевана подешавања – одабраће се Development контејнер креиран за овај курс
4. Кликните **Create codespace**
5. Сачекајте 5-10 минута да окружење буде спремно
6. Одмах започните са [Брзим почетком](./00-quick-start/README.md) и крените!

Након завршетка модула, истражите [Водич за тестирање](docs/TESTING.md) да видите концепте тестирања LangChain4j у пракси.

> **Напомена:** Ова обука користи и GitHub моделе и Azure OpenAI. Модул [Брзи почетак](00-quick-start/README.md) користи GitHub моделе (није потребна Azure претплата), док модули 1-5 користе Azure OpenAI. Почните са [БЕСПЛАТНИМ Azure налогом](https://aka.ms/azure-free-account) ако га немате.


## Учење са GitHub Copilot-ом

Да бисте брзо почели са кодирањем, отворите овај пројекат у GitHub Codespace-у или у свом локалном IDE-ју са обезбеђеним devcontainer-ом. Devcontainer коришћен на овом курсу долази претходно конфигурисан са GitHub Copilot-ом за AI парно програмирање.

Сваки пример кода садржи предлоге питања која можете поставити GitHub Copilot-у да боље разумете. Потражите 💡/🤖 позиве у:

- **Java заглављима фајлова** - Питања специфична за сваки пример
- **Реадме-има модула** - Позиви за истраживање након примера кода

**Како користити:** Отворите било који фајл са кодом и поставите Copilot-у предложена питања. Он има пун контекст базе кода и може објаснити, проширити и предложити алтернативе.

Желите да сазнате више? Погледајте [Copilot за AI парно програмирање](https://aka.ms/GitHubCopilotAI).
