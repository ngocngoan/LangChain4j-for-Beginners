# LangChain4j สำหรับผู้เริ่มต้น

หลักสูตรการสร้างแอปพลิเคชัน AI ด้วย LangChain4j และ Azure OpenAI GPT-5.2 ตั้งแต่การแชทพื้นฐานไปจนถึงตัวแทน AI

**ใหม่กับ LangChain4j?** ตรวจสอบ [อภิธานศัพท์](docs/GLOSSARY.md) สำหรับคำจำกัดความของคำสำคัญและแนวคิดต่างๆ

## สารบัญ

1. [เริ่มต้นอย่างรวดเร็ว](00-quick-start/README.md) - เริ่มต้นใช้งาน LangChain4j
2. [บทนำ](01-introduction/README.md) - เรียนรู้พื้นฐานของ LangChain4j
3. [วิศวกรรมคำสั่ง](02-prompt-engineering/README.md) - เชี่ยวชาญการออกแบบคำสั่งอย่างมีประสิทธิภาพ
4. [RAG (Retrieval-Augmented Generation)](03-rag/README.md) - สร้างระบบฐานความรู้ที่ชาญฉลาด
5. [เครื่องมือ](04-tools/README.md) - รวมเครื่องมือภายนอกและผู้ช่วยง่ายๆ
6. [MCP (Model Context Protocol)](05-mcp/README.md) - ทำงานกับ Model Context Protocol (MCP) และโมดูล Agentic
---

## เส้นทางการเรียนรู้

> **เริ่มต้นอย่างรวดเร็ว**

1. Fork โครงการนี้ไปยังบัญชี GitHub ของคุณ
2. คลิก **Code** → แท็บ **Codespaces** → **...** → **New with options...**
3. ใช้ค่าปริยาย – จะเลือก Development container ที่ถูกสร้างสำหรับหลักสูตรนี้
4. คลิก **Create codespace**
5. รอ 5-10 นาทีจนกว่าสภาพแวดล้อมจะพร้อมใช้งาน
6. ไปที่ [เริ่มต้นอย่างรวดเร็ว](./00-quick-start/README.md) เพื่อเริ่มต้น!

หลังจากทำโมดูลเสร็จแล้ว ให้สำรวจ [คู่มือการทดสอบ](docs/TESTING.md) เพื่อดูแนวคิดการทดสอบ LangChain4j ในการปฏิบัติ

> **หมายเหตุ:** การฝึกอบรมนี้ใช้ทั้ง GitHub Models และ Azure OpenAI โมดูล [เริ่มต้นอย่างรวดเร็ว](00-quick-start/README.md) ใช้ GitHub Models (ไม่ต้องสมัครสมาชิก Azure) ในขณะที่โมดูล 1-5 ใช้ Azure OpenAI เริ่มต้นใช้งานได้ฟรีด้วย [บัญชี Azure ฟรี](https://aka.ms/azure-free-account) หากคุณยังไม่มี


## การเรียนรู้ด้วย GitHub Copilot

เพื่อเริ่มเขียนโค้ดอย่างรวดเร็ว ให้เปิดโครงการนี้ใน GitHub Codespace หรือ IDE ท้องถิ่นของคุณด้วย devcontainer ที่เตรียมไว้แล้ว Devcontainer ที่ใช้ในหลักสูตรนี้ได้ติดตั้ง GitHub Copilot สำหรับการเขียนโปรแกรมร่วมกับ AI แล้ว

ตัวอย่างโค้ดแต่ละอันมีคำถามแนะนำที่คุณสามารถถาม GitHub Copilot เพื่อเข้าใจลึกขึ้น ค้นหาสัญลักษณ์ 💡/🤖 ใน:

- **หัวไฟล์ Java** - คำถามเฉพาะสำหรับแต่ละตัวอย่าง
- **README ของโมดูล** - คำถามสำรวจหลังตัวอย่างโค้ด

**วิธีใช้งาน:** เปิดไฟล์โค้ดใดก็ได้และถามคำถามแนะนำกับ Copilot มันมีบริบทเต็มของฐานโค้ดและสามารถอธิบาย ขยายความ และแนะนำทางเลือกได้

ต้องการเรียนรู้เพิ่มเติมหรือไม่? ดูที่ [Copilot สำหรับการเขียนโปรแกรมร่วมกับ AI](https://aka.ms/GitHubCopilotAI)
