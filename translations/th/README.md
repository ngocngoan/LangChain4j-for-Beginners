# LangChain4j สำหรับผู้เริ่มต้น

คอร์สสำหรับสร้างแอปพลิเคชัน AI ด้วย LangChain4j และ Azure OpenAI GPT-5.2 ตั้งแต่แชทพื้นฐานถึงเอเจนต์ AI

**ใหม่กับ LangChain4j?** ดูที่ [อภิธานศัพท์](docs/GLOSSARY.md) สำหรับคำจำกัดความของคำสำคัญและแนวคิด

## สารบัญ

1. [เริ่มต้นอย่างรวดเร็ว](00-quick-start/README.md) - เริ่มใช้งาน LangChain4j
2. [แนะนำ](01-introduction/README.md) - เรียนรู้พื้นฐานของ LangChain4j
3. [วิศวกรรมพรอมต์](02-prompt-engineering/README.md) - เชี่ยวชาญการออกแบบพรอมต์อย่างมีประสิทธิภาพ
4. [RAG (การสร้างโดยเสริมการดึงข้อมูล)](03-rag/README.md) - สร้างระบบฐานความรู้ที่ชาญฉลาด
5. [เครื่องมือ](04-tools/README.md) - ผสมผสานเครื่องมือภายนอกและผู้ช่วยง่ายๆ
6. [MCP (โพรโตคอลบริบทโมเดล)](05-mcp/README.md) - ทำงานกับโพรโตคอลบริบทโมเดล (MCP) และโมดูล Agentic
---

## เส้นทางการเรียนรู้

> **เริ่มต้นอย่างรวดเร็ว**

1. Fork คลังนี้ไปยังบัญชี GitHub ของคุณ
2. คลิกที่ **Code** → แท็บ **Codespaces** → **...** → **สร้างใหม่พร้อมตัวเลือก...**
3. ใช้ค่าเริ่มต้น – จะเลือก Development container ที่สร้างสำหรับคอร์สนี้
4. คลิก **Create codespace**
5. รอ 5-10 นาทีเพื่อเตรียมสภาพแวดล้อมให้พร้อม
6. ข้ามไปยัง [เริ่มต้นอย่างรวดเร็ว](./00-quick-start/README.md) เพื่อเริ่มเรียน!

หลังจากทำโมดูลเสร็จแล้ว สำรวจ [คู่มือการทดสอบ](docs/TESTING.md) เพื่อดูแนวคิดการทดสอบ LangChain4j ในการปฏิบัติ

> **หมายเหตุ:** การฝึกอบรมนี้ใช้ทั้ง GitHub Models และ Azure OpenAI โมดูล [เริ่มต้นอย่างรวดเร็ว](00-quick-start/README.md) ใช้ GitHub Models (ไม่ต้องสมัคร Azure) ขณะที่โมดูล 1-5 ใช้ Azure OpenAI เริ่มต้นด้วย [บัญชี Azure ฟรี](https://aka.ms/azure-free-account) หากคุณยังไม่มี

## การเรียนรู้กับ GitHub Copilot

เพื่อเริ่มโค้ดอย่างรวดเร็ว เปิดโปรเจกต์นี้ใน GitHub Codespace หรือ IDE ในเครื่องด้วย devcontainer ที่ให้มา devcontainer ที่ใช้ในคอร์สนี้ตั้งค่า GitHub Copilot สำหรับการเขียนโปรแกรมคู่ AI มาให้เรียบร้อย

ตัวอย่างโค้ดแต่ละอันมีคำถามแนะนำที่คุณสามารถถาม GitHub Copilot เพื่อเสริมความเข้าใจ มองหา 💡/🤖 ใน:

- **ส่วนหัวไฟล์ Java** - คำถามเฉพาะสำหรับแต่ละตัวอย่าง
- **README โมดูล** - คำถามสำรวจหลังตัวอย่างโค้ด

**วิธีใช้:** เปิดไฟล์โค้ดใดก็ได้แล้วถาม Copilot ด้วยคำถามเหล่านี้ มันมีบริบททั้งหมดของโค้ดและสามารถอธิบาย ขยายความ และเสนอทางเลือกได้

ต้องการรู้เพิ่มเติม? ดูที่ [Copilot สำหรับการเขียนโปรแกรมคู่ AI](https://aka.ms/GitHubCopilotAI)
