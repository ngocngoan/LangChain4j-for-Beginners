# सुरुवातीहरूको लागि LangChain4j

LangChain4j र Azure OpenAI GPT-5.2 प्रयोग गरेर AI अनुप्रयोगहरू निर्माण गर्ने कोर्स, आधारभूत च्याटदेखि AI एजेन्टहरू सम्म।

**LangChain4j मा नयाँ हुनुहुन्छ?** मुख्य शब्दहरू र अवधारणाहरूको परिभाषाका लागि [Glossary](docs/GLOSSARY.md) हेर्नुहोस्।

## सामग्रीहरूको तालिका

4. [RAG (पुन:प्राप्ति-संवर्धित उत्पादन)](03-rag/README.md) - बुद्धिमान ज्ञान-आधारित प्रणालीहरू निर्माण गर्नुहोस्
5. [उपकरणहरू](04-tools/README.md) - बाह्य उपकरणहरू र सरल सहायकहरू समाहित गर्नुहोस्

## सिकाइ मार्ग

> **छिटो सुरु**

1. यो रिपोजिटरी तपाईंको GitHub खातामा Fork गर्नुहोस्
2. क्लिक गर्नुहोस् **Code** → **Codespaces** tab → **...** → **New with options...**
3. पूर्वनिर्धारित सेटिङहरू प्रयोग गर्नुहोस् – यसले यस कोर्सका लागि सिर्जना गरिएको विकास कन्टेनर चयन गर्नेछ
4. क्लिक गर्नुहोस् **Create codespace**
5. वातावरण तयार हुन ५-१० मिनेट पर्खनुहोस्
6. सुरु गर्न [छिटो सुरु](./00-quick-start/README.md) सिधा जानुहोस्!

मोड्युलहरू पूरा गरेपछि, LangChain4j परीक्षण अवधारणाहरू क्रियाशील रूपमा हेर्न [Testing Guide](docs/TESTING.md) अन्वेषण गर्नुहोस्।

> **नोट:** यो तालिम GitHub मोडेलहरू र Azure OpenAI दुवै प्रयोग गर्दछ। [छिटो सुरु](00-quick-start/README.md) मोड्युलले GitHub मोडेलहरू प्रयोग गर्दछ (Azure सदस्यता आवश्यक छैन), जबकि १-५ मोड्युलहरू Azure OpenAI प्रयोग गर्दछन्। यदि तपाईंकोसँग छैन भने [निःशुल्क Azure खाता](https://aka.ms/azure-free-account) बाट सुरु गर्नुहोस्।


## GitHub Copilot सँग सिकाइ

छिटो कोडिङ सुरु गर्न, यो प्रोजेक्ट GitHub Codespace वा तपाईंको स्थानीय IDE मा devcontainer सहित खोल्नुहोस्। यस कोर्सले प्रयोग गरेको devcontainer GitHub Copilot लाई AI जोडिएको प्रोग्रामिङको लागि पूर्व-संरचित रूपमा समावेश गर्दछ।

प्रत्येक कोड उदाहरणमा GitHub Copilot लाई सोध्न सकिने सुझाव गरिएको प्रश्नहरू समावेश छन् जसले तपाईंको बुझाइ गहिरो पार्न मद्दत गर्छ। 💡/🤖 संकेतहरूको लागि हेर्नुहोस्:

- **Java फाइल हेडरहरू** - प्रत्येक उदाहरणका लागि विशेष प्रश्नहरू
- **मोड्युल README हरू** - कोड उदाहरणहरू पछि अन्वेषण सुझाबहरू

**कसरी प्रयोग गर्ने:** कुनै पनि कोड फाइल खोल्नुहोस् र Copilot लाई सुझाव गरिएको प्रश्नहरू सोध्नुहोस्। यसले कोडबेसको पूर्ण सन्दर्भ छ र व्याख्या, विस्तार, र वैकल्पिक समाधानहरू सुझाव गर्न सक्छ।

थप सिक्न चाहनुहुन्छ? हेर्नुहोस् [AI जोडिएको प्रोग्रामिङका लागि Copilot](https://aka.ms/GitHubCopilotAI)।
