# LangChain4j सुरु गर्नेहरूको लागि

LangChain4j र Azure OpenAI GPT-5 सँग AI एप्लिकेसनहरू निर्माण गर्ने कोर्स, आधारभूत च्याटदेखि AI एजेन्टहरू सम्म।

**LangChain4j नयाँ हुनुहुन्छ?** मुख्य सर्तहरू र अवधारणाहरूका परिभाषाहरूको लागि [शब्दावली](docs/GLOSSARY.md) हेर्नुहोस्।

## विषय सूची

1. [छिटो सुरु](00-quick-start/README.md) - LangChain4j सँग सुरु गर्नुहोस्
2. [परिचय](01-introduction/README.md) - LangChain4j का आधारभूत कुरा सिक्नुहोस्
3. [प्रॉम्प्ट इन्जिनियरिङ](02-prompt-engineering/README.md) - प्रभावकारी प्रॉम्प्ट डिजाइनमा विशेषज्ञता हासिल गर्नुहोस्
4. [RAG (रिट्रिभल-अग्मेन्टेड जेनेरेसन)](03-rag/README.md) - बुद्धिमानी ज्ञान आधारित प्रणालीहरू बनाउनुहोस्
5. [औजारहरू](04-tools/README.md) - बाह्य उपकरण र सादा सहायकहरू समाहित गर्नुहोस्
6. [MCP (मोडेल कण्टेक्स्ट प्रोटोकल)](05-mcp/README.md) - मोडेल कण्टेक्स्ट प्रोटोकल (MCP) र एजेन्टिक मोड्युलहरूसँग काम गर्नुहोस्

## सिकाइ मार्ग

> **छिटो सुरु**

1. यो रिपोजिटरीलाई आफ्नो GitHub खातामा Fork गर्नुहोस्
2. क्लिक गर्नुहोस् **Code** → **Codespaces** ट्याब → **...** → **New with options...**
3. पूर्वनिर्धारित सेटिङहरू प्रयोग गर्नुहोस् – यसले यस कोर्सको लागि सिर्जना गरिएको Development container चयन गर्दछ
4. क्लिक गर्नुहोस् **Create codespace**
5. वातावरण तयार हुन ५-१० मिनेट पर्खनुहोस्
6. सिधै [छिटो सुरु](./00-quick-start/README.md) मा जानुहोस्!

मोड्युलहरू पूरा गरेपछि, LangChain4j परीक्षण अवधारणाहरू क्रियामा हेर्न [टेस्टिङ गाइड](docs/TESTING.md) अन्वेषण गर्नुहोस्।

> **ध्यान दिनुहोस्:** यो प्रशिक्षण GitHub मोडेलहरू र Azure OpenAI दुवै प्रयोग गर्दछ। [छिटो सुरु](00-quick-start/README.md) मोड्युल GitHub मोडेलहरू प्रयोग गर्छ (Azure सदस्यता आवश्यक छैन), जबकि मोड्युल १-५ Azure OpenAI प्रयोग गर्छ। यदि तपाईंसँग छैन भने [निःशुल्क Azure खाता](https://aka.ms/azure-free-account) बाट सुरु गर्नुहोस्।


## GitHub Copilot सँग सिकाइ

छिटो कोडिङ सुरु गर्न, यस प्रोजेक्टलाई GitHub Codespace वा स्थानीय IDE मा devcontainer प्रयोग गरेर खोल्नुहोस्। यस कोर्समा प्रयोग गरिएको devcontainer GitHub Copilot सँग पूर्व-सेटअप गरिएको छ, जसले AI जोडिएको प्रोग्रामिङलाई सजिलो बनाउँछ।

हरेक कोड उदाहरणमा GitHub Copilot लाई सोध्न सकिने सिफारिस गरिएका प्रश्नहरू छन् जसले तपाईको बुझाइलाई गहिरो बनाउँछ। 💡/🤖 संकेतहरू तल हेर्नुहोस्:

- **Java फाइल शीर्षकहरू** - प्रत्येक उदाहरणका लागि विशेष प्रश्नहरू
- **मोड्युल README हरू** - कोड उदाहरण पछि अन्वेषण संकेतहरू

**कसरी प्रयोग गर्ने:** कुनै पनि कोड फाइल खोल्नुहोस् र Copilot लाई सुझाइएका प्रश्नहरू सोध्नुहोस्। यसले कोडबेसको सम्पूर्ण परिवेश बुझ्छ र व्याख्या, विस्तार र विकल्पहरू सुझाव दिन सक्छ।

थप सिक्न चाहनुहुन्छ? [AI जोडिएको प्रोग्रामिङका लागि Copilot](https://aka.ms/GitHubCopilotAI) हेर्नुहोस्।
