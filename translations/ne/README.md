# LangChain4j सुरु गर्नेलाई

LangChain4j र Azure OpenAI GPT-5.2 सँग AI अनुप्रयोगहरू निर्माण गर्ने कोर्स, आधारभूत च्याटदेखि AI एजेन्टसम्म।

**LangChain4j मा नयाँ?** मुख्य सर्त र अवधारणाहरूको परिभाषाका लागि [शब्दावली](docs/GLOSSARY.md) हेर्नुहोस्।

## सामग्री तालिका

1. [छिटो सुरु](00-quick-start/README.md) - LangChain4j सँग सुरु गर्नुस्
2. [परिचय](01-introduction/README.md) - LangChain4j का आधारभूत कुरा सिक्नुस्
3. [प्रॉम्प्ट इन्जिनियरिङ](02-prompt-engineering/README.md) - प्रभावकारी प्रॉम्प्ट डिजाइनमा दक्ष हुनुस्
4. [RAG (रेट्रिभल-अग्मेन्टेड जेनेरेशन)](03-rag/README.md) - बुद्धिमान ज्ञान-आधारित प्रणालीहरू बनाउनुहोस्
5. [उपकरणहरू](04-tools/README.md) - बाह्य उपकरणहरू र सरल सहायकहरू एकीकृत गर्नुहोस्
6. [MCP (मोडेल कन्टेक्स्ट प्रोटोकल)](05-mcp/README.md) - मोडेल कन्टेक्स्ट प्रोटोकल (MCP) र एजेन्टिक मोड्युलहरूसँग काम गर्नुहोस्

## सिकाइ मार्ग

> **छिटो सुरु**

1. यो रिपोजिटरीलाई आफ्नो GitHub खाता मा Fork गर्नुहोस्
2. क्लिक गर्नुहोस् **Code** → **Codespaces** ट्याब → **...** → **New with options...**
3. पूर्वनिर्धारित सेटिङहरू प्रयोग गर्नुहोस् – यसले यस कोर्सका लागि बनाइएको Development container चयन गर्नेछ
4. क्लिक गर्नुहोस् **Create codespace**
5. वातावरण तयार हुन ५-१० मिनेट कुर्नुहोस्
6. सुरु गर्न [Quick Start](./00-quick-start/README.md) मा सिधै जानुहोस्!

मोड्युलहरू पूरा गरेपछि, LangChain4j परीक्षण अवधारणाहरूलाई क्रियाशील रूपमा हेर्न [Testing Guide](docs/TESTING.md) अन्वेषण गर्नुहोस्।

> **नोट:** यस तालिममा GitHub मोडेलहरू र Azure OpenAI दुवै प्रयोग गरिएको छ। [Quick Start](00-quick-start/README.md) मोड्युलले GitHub मोडेलहरू प्रयोग गर्छ (Azure सदस्यता आवश्यक छैन), जबकि मोड्युल १-५ ले Azure OpenAI प्रयोग गर्छ। Azure खाता नहुँदा [निःशुल्क Azure खाता](https://aka.ms/azure-free-account) बाट सुरु गर्न सक्नुहुन्छ।

## GitHub Copilot सँग सिकाइ

छिटो कोडिङ सुरु गर्न, GitHub Codespace वा दिइएको devcontainer सहित आफ्नो स्थानीय IDE मा यो परियोजना खोल्नुहोस्। यस कोर्समा प्रयोग गरिएको devcontainer पहिले देखि GitHub Copilot एआई जोडी प्रोग्रामिङका लागि पूर्व-सेटअप गरिएको छ।

प्रत्येक कोड उदाहरणमा तपाईं GitHub Copilot लाई सोध्न सक्ने सुझाव दिइएका प्रश्नहरू समावेश छन् जसले तपाइँको बुझाइलाई गहिरो बनाउँछ। हेर्नुहोस् 💡/🤖 संकेतहरू:

- **Java फाइल हेडरहरू** - प्रत्येक उदाहरणसँग सम्बन्धित प्रश्नहरू
- **मोड्युल README हरू** - कोड उदाहरण पछि अन्वेषण प्रश्नहरू

**कसरी प्रयोग गर्ने:** कुनै पनि कोड फाइल खोल्नुहोस् र Copilot लाई सुझाव दिइएको प्रश्नहरू सोध्नुहोस्। यसले कोडबेसको पूर्ण सन्दर्भ छ र व्याख्या, विस्तार, र विकल्पहरू सुझाव गर्न सक्छ।

थप जान्न चाहनुहुन्छ? [AI जोडी प्रोग्रामिङको लागि Copilot](https://aka.ms/GitHubCopilotAI) हेर्नुहोस्।
