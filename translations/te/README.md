## అభ్యాస మార్గం

**LangChain4j కొత్తవారా?** ముఖ్య పదాలు మరియు కాన్సెప్ట్‌ల నిర్వచనాల కోసం [Glossary](docs/GLOSSARY.md) చూడండి.

> **త్వరిత ప్రారంభం**

1. ఈ రిపోజిటరీని మీ GitHub ఖాతాకు ఫార్క్ చెయ్యండి
2. **Code** → **Codespaces** టాబ్ → **...** → **New with options...** పై క్లిక్ చేయండి
3. డిఫాల్ట్స్‌ను వాడండి - ఇది కోర్సు కోసం సృష్టించిన డెవలప్‌మెంట్ కంటైనర్‌ను ఎంచేస్తుంది
4. **Create codespace** పై క్లిక్ చేయండి
5. ఎన్విరాన్‌మెంట్ రెడీ అయ్యేవరకు 5-10 నిమిషాలు వేచుకోండి
6. [త్వరిత ప్రారంభం](./00-quick-start/README.md) అందులోకి నేరుగా వెళ్లి మొదలు పెట్టండి!

మాడ్యూల్స్ పూర్తి చేసిన తర్వాత, LangChain4j టెస్టింగ్ కాన్సెప్ట్‌లను కార్యాచరణలో చూడటానికి [Testing Guide](docs/TESTING.md) అన్వేషించండి.

> **గమనిక:** ఈ శిక్షణలో GitHub మోడల్స్ మరియు అజ్యూర్ ఓపెన్‌ఏఐ రెండూ వాడబడుతున్నాయి. [త్వరిత ప్రారంభం](00-quick-start/README.md) మాడ్యూల్ GitHub మోడల్స్ ఉపయోగిస్తుంది (అజ్యూర్ సబ్‌స్క్రిప్షన్ అవసరం లేదు), మాడ్యూల్స్ 1-5 అజ్యూర్ ఓపెన్‌ఏఐ వాడుతాయ్. మీ వద్ద లేని వారికి [ఉచిత అజ్యూర్ ఖాతా](https://aka.ms/azure-free-account) తో ప్రారంభించండి.


## GitHub Copilotతో అభ్యసించడం

త్వరగా కోడింగ్ చేయడం ప్రారంభించాలంటే, ఈ ప్రాజెక్టును GitHub Codespace లేదా మీ లోకల్ IDEలో అందిన devcontainerతో తెరవండి. ఈ కోర్సులో వాడిన devcontainer GitHub Copilotతో AI జతగా ప్రోగ్రామింగ్ కోసం ముందుగా కలిపి ఉంటుంది.

ప్రతి కోడ్ ఉదాహరణలో GitHub Copilotను మీరు అడగగలిగే ప్రతిప్రశ్నలు సహా ఉంటాయి, ఇవి మీ అర్థం మెరుగుపరుస్తాయి. 💡/🤖 సూచనలను ఇలా చూడండి:

- **జావా ఫైల్ హెడర్లు** - ప్రతి ఉదాహరణకు ప్రత్యేక ప్రశ్నలు
- **మాడ్యూల్ READMEలు** - కోడ్ ఉదాహరణల తర్వాత అన్వేషణా సూచనలు

**ఎలా వాడాలి:** ఏదైనా కోడ్ ఫైల్ తెరిచి Copilotకు సూచించిన ప్రశ్నలు అడగండి. దీని దగ్గర కోడ్ ఆధారాల పూర్తి పరిజ్ఞానం ఉంటుంది మరియు వివరణ, పొడగింపు, లేదా ప్రత్యామ్నాయాలు సూచిస్తుంది.

మరింత తెలుసుకోవాలనుకుంటున్నారా? [AI జతగా ప్రోగ్రామింగ్ కోసం Copilot](https://aka.ms/GitHubCopilotAI) చూడండి.
