##  Learning Path

> **Quick Start**

1. Fork dis repo go your GitHub account
2. Click **Code** → **Codespaces** tab → **...** → **New with options...**
3. Use all the default ones – dis one go select di Development container wey dem create for dis course
4. Click **Create codespace**
5. Wait for 5-10 minutes make environment ready
6. Jump straight go [Quick Start](./00-quick-start/README.md) make you start!

After you complete di modules, explode di [Testing Guide](docs/TESTING.md) to see how LangChain4j testing concepts dey work.

> **Note:** Dis training dey use both GitHub Models and Azure OpenAI. Di [Quick Start](00-quick-start/README.md) module dey use GitHub Models (no need Azure subscription), but modules 1-5 dey use Azure OpenAI. If you no get, start wit [FREE Azure account](https://aka.ms/azure-free-account).


## Learning with GitHub Copilot

If you want start codin quick quick, open dis project inside GitHub Codespace or your local IDE wit di devcontainer wey dem give. Di devcontainer wey dem take use for dis course don already set GitHub Copilot for AI paired programming.

Every code example get suggested questions wey you fit ask GitHub Copilot to understand better. Watch out for di 💡/🤖 prompts for:

- **Java file headers** - Questions wey dey each example
- **Module READMEs** - Questions for exploration after code examples

**How to use:** Open any code file ask Copilot di suggested questions. E sabi everything about di codebase and e fit explain, add more, or suggest other way.

You wan learn more? Check [Copilot for AI Paired Programming](https://aka.ms/GitHubCopilotAI).
