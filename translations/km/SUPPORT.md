# គាំទ្រ

## របៀបដាក់បញ្ហា និងទទួលបានជំនួយ  

គម្រោងនេះប្រើប្រាស់ GitHub Issues ដើម្បីតាមដានកំហុស និងការស្នើរសុំមុខងារ។ សូមស្វែងរកបញ្ហាដែលមានស្រាប់មុនពេលដាក់បញ្ហាថ្មី ដើម្បីជៀសវាងការបញ្ចូលចម្លង។ សម្រាប់បញ្ហាថ្មី សូមដាក់កំហុសរបស់អ្នក ឬស្នើរសុំមុខងារជាបញ្ហាថ្មីមួយ។

## គោលនយោបាយគាំទ្ររបស់ Microsoft
