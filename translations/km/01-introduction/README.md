## бһўбҹ’бһңбһёбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҡбҹҖбһ“

бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҠбһҫбһҳбһӣбһҝбһ“ бһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһҳбҹүбһјбһҢбҹӮбһӣ GitHub бһҠбһҫбһҳбҹ’бһ”бһёбһ•бҹ’бһүбһҫбһҹбҹҶбһҺбһҫбһҡ бһ бҹ…бһ§бһ”бһҖбһҡбһҺбҹҚ бһ”бһ„бҹ’бһҖбһҫбһҸбһ•бҹ’бһӣбһјбһң RAG бһ“бһ·бһ„бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҖбһ¶бһҡбһҖбһ¶бһҡбһ–бһ¶бһҡбҹ” бһ”бһ„бҹ’бһ бһ¶бһүбһ‘бһ¶бҹҶбһ„бһ“бҹ„бҹҮбһ”бһ„бҹ’бһ бһ¶бһүбһўбҹ’бһңбһёбһҠбҹӮбһӣбһўбһ¶бһ…бһ’бҹ’бһңбһҫбһ”бһ¶бһ“ вҖ” бһҘбһӣбһјбһңбһҷбһҫбһ„бһ”бҹ’бһҡбҹӮбһ‘бҹ…бһ”бҹ’бһҡбһҫ Azure OpenAI бһ“бһ·бһ„ GPT-5.2 бһ бһҫбһҷбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҹбҹ’бһ‘бһёбһӣбһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳбҹ” бһҳбҹҒбһҢбһ»бһӣбһ“бҹҒбҹҮбһ•бҹ’бһҠбҹ„бһҸбһӣбһҫ AI бһҹбһ“бҹ’бһ‘бһ“бһ¶бһҠбҹӮбһӣбһ…бһ„бһ…бһ¶бҹҶ context бһ“бһ·бһ„бһҡбһҖбҹ’бһҹбһ¶бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ– вҖ” бһӮбҹҶбһ“бһ·бһҸбһҠбҹӮбһӣбһ”бһ„бҹ’бһ бһ¶бһүбһҖбҹ’бһ“бһ»бһ„бһҹбһҳбҹҗбһҷбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһӣбһҝбһ“ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһҳбһ·бһ“бһ”бһ¶бһ“бһ–бһ“бҹ’бһҷбһӣбҹӢбҹ”

бһҷбһҫбһ„бһ“бһ№бһ„бһ”бҹ’бһҡбһҫ GPT-5.2 бһҡбһ”бһҹбҹӢ Azure OpenAI бһҮбһ¶бһҡбһҷбҹҲбһ–бҹҒбһӣбһҠбһ№бһҖбһ“бһ¶бҹҶбһ“бҹҒбҹҮ бһ–бҹ’бһҡбҹ„бҹҮбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҹбҹҶбһҡбһ¶бһ”бҹӢбһӮбҹҶбһ“бһ·бһҸбҹҲбһ…бҹҶбһӣбҹӮбһҖбһҖбһҳбҹ’бһҡбһ·бһҸбһҒбҹ’бһ–бһҹбҹӢбһ”бһ“бҹ’бһҗбҹӮбһҳбһ’бҹ’бһңбһҫбһўбҹ„бһҷбһўбһ¶бһҖбһ”бҹ’бһ”бһ”бһҡбһҳбһ¶бһ•бҹ’бһӮбһҪбһ”бһӮбҹ’бһ“бһ¶бһ“бҹ…бһӣбһҫбһҡбһ…бһ“бһ¶бһ”бҹҗбһ‘бҹ’бһҳбһ•бҹ’бһҹбҹҒбһ„бҹ—бһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢбһҮбһ¶бһ„бһҳбһ»бһ“бҹ” бһ–бҹҒбһӣбһўбҹ’бһ“бһҖбһ”бһ“бҹ’бһҗбҹӮбһҳбһҳбҹҒбһҳбһәбһҡбһё бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһғбһҫбһүбһ—бһ¶бһ–бһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бһ…бҹ’бһ”бһ¶бһҹбҹӢбҹ” бһңбһ¶бһ’бҹ’бһңбһҫбһўбҹ„бһҷбһҷбһӣбҹӢбһҷбҹүбһ¶бһ„бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһҗбһ¶бһ•бҹ’бһ“бҹӮбһҖбһҺбһ¶бһҳбһҪбһҷбһ“бһ¶бҹҶбһҳбһҖбһўбҹ’бһңбһёбһҠбһӣбҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”

бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҳбһҪбһҷбһҠбҹӮбһӣбһ”бһ„бҹ’бһ бһ¶бһүбһҡбһ…бһ“бһ¶бһ”бҹҗбһ‘бҹ’бһҳбһ‘бһ¶бҹҶбһ„бһ–бһёбһҡбҹ–

**бһҮбһҮбҹӮбһҖбһҳбһ·бһ“бһҳбһ¶бһ“бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–** - бһҖбһ¶бһҡбһҹбҹҶбһҺбһҫбһҡвҖӢбһҹбҹ’бһ‘бһҫбһҡбһҖбҹ’бһ“бһ»бһ„бһҳбһҪбһҷбһҹбҹҶбһҺбһҫбһӮбһәбһҜбһҖбһҡбһ¶бһҮбҹ’бһҷбҹ” бһҳбҹүбһјбһҢбҹӮбһӣбһҳбһ·бһ“бһҳбһ¶бһ“бһҳбҹҒбһҳбһәбһҡбһёбһ“бҹғбһҹбһ¶бһҡбһҳбһ»бһ“бҹ—бһ‘бҹҒбҹ” бһ“бҹҒбҹҮбһӮбһәбһҮбһ¶бһҡбһ…бһ“бһ¶бһ”бҹҗбһ‘бҹ’бһҳбһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһӣбһҝбһ“бҹ”

**бһҮбһҮбҹӮбһҖбһҳбһ¶бһ“бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–** - бһҖбһ¶бһҡбһҹбҹҶбһҺбһҫбһҡвҖӢбһҳбһҪбһҷбһҡбһҪбһҳбһҳбһ¶бһ“бһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҹбһ“бҹ’бһ‘бһ“бһ¶бҹ” бһҳбҹүбһјбһҢбҹӮбһӣбһҡбһҖбҹ’бһҹбһ¶бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ– context бһҡбһҷбҹҲбһ–бҹҒбһӣбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһҮбҹҶбһ“бһҪбһҹбҹ” бһ“бҹҒбҹҮбһӮбһәбһҮбһ¶бһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳбҹ”

## бһ”бһҫбһҖбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡвҖӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёвҖӢбһ“бҹ…вҖӢбһҸбҹҶбһ”бһ“бҹӢбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“

**бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҖбһ¶бһҡбһ…бһ¶бһҸбҹӢбһ…бҹӮбһ„бҹ–**

бһ’бҹ’бһңбһҫбһІбҹ’бһҷбһ”бҹ’бһҡбһ¶бһҖбһҠбһҗбһ¶ бһҜбһҖбһҹбһ¶бһҡ `.env` бһҳбһ¶бһ“бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҗбһҸбһҠбһҫбһҳ бһҮбһ¶бһҳбһҪбһҷбһҹбһүбҹ’бһүбһ¶бһ”бҹҗбһҸбҹ’бһҡбһҡбһ”бһҹбҹӢ Azureбҹ” бһҡбһҸбҹӢбһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶бһ“бҹҒбҹҮбһ–бһёбһҗбһҸбһҳбҹҒбһҢбһ»бһӣ (`01-introduction/`)бҹ–

**бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбҹ–**

**бһҮбһҳбҹ’бһҡбһҫбһҹ бҹЎбҹ– бһ”бҹ’бһҡбһҫ Spring Boot Dashboard (бһ•бҹ’бһҠбһӣбҹӢбһўбһ“бһ»бһҹбһ¶бһҹбһ“бҹҚбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ VS Code)**

Dev container бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһ•бҹ’бһ“бҹӮбһҖбһ”бһ“бҹ’бһҗбҹӮбһҳ Spring Boot Dashboard бһҠбҹӮбһӣбһ•бҹ’бһҠбһӣбҹӢбһ…бҹҶбһҺбһ»бһ…бһ”бҹ’бһҡбһ‘бһ¶бһҖбҹӢбһҠбҹҸбһ‘бһ¶бһҖбҹӢбһ‘бһ¶бһүбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё Spring Boot бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҡбһҖбһғбһҫбһүбһңбһ¶бһ“бҹ… Activity Bar бһ—бһ¶бһӮбһҒбһ¶бһ„бһҶбҹ’бһңбҹҒбһ„ бһ“бҹғ VS Code (бһҹбһјбһҳбһҹбһ„бҹ’бһҖбҹҒбһҸбһҹбһүбҹ’бһүбһ¶бһҡбһ”бһҹбҹӢ Spring Boot)бҹ”

бһ–бһё Spring Boot Dashboard бһўбҹ’бһ“бһҖбһўбһ¶бһ…бҹ–  
- бһҳбһҫбһӣбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё Spring Boot бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҖбҹ’бһ“бһ»бһ„workspace  
- бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ/бһ”бһүбҹ’бһҲбһ”бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҠбҹ„бһҷбһ…бһ»бһ…бһҸбҹӮбһҳбһҪбһҷ  
- бһҳбһҫбһӣбһҖбҹҶбһҺбһҸбҹӢбһ бҹҒбһҸбһ»бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҖбҹ’бһ“бһ»бһ„бһ–бҹҒбһӣбһ–бһ·бһҸ  
- бһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё  

бһӮбҹ’бһҡбһ¶бһ“бҹӢбһҸбҹӮбһ…бһ»бһ…бһ”бҹҠбһјбһҸбһ»бһ„бһӣбҹҒбһ„бһ“бҹ…бһ…бҹҶбһ бҹҖбһ„ "introduction" бһҠбһҫбһҳбҹ’бһ”бһёбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҳбҹҒбһҢбһ»бһӣбһ“бҹҒбҹҮ бһ¬бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҳбҹҒбһҢбһ»бһӣбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҖбҹ’бһ“бһ»бһ„бһ–бҹҒбһӣбһҸбҹӮбһҳбһҪбһҷбҹ”

бһҘбһЎбһјбһңбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһӣбҹҶбһҠбһ¶бһ”бҹӢбһҠбһјбһ…бһӮбҹ’бһ“бһ¶бһ“бҹ…бһ‘бһёбһ“бҹҒбҹҮбҹ” бһҹбһҪбһҡ "бһҲбҹ’бһҳбҹ„бҹҮбһҒбҹ’бһүбһ»бҹҶбһӮбһә John" бһ бһҫбһҷбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖ "бһҲбҹ’бһҳбҹ„бҹҮбһҒбҹ’бһүбһ»бҹҶбһўбҹ’бһңбһё?" бһҳбҹ’бһҸбһ„бһ“бҹҒбҹҮбһңбһ¶бһ…бһ„бһ…бһ¶бҹҶбҹ” бһ—бһ¶бһ–бһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бһӮбһә MessageWindowChatMemory вҖ” бһңбһ¶бһҡбһҖбҹ’бһҹбһ¶бһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҮбһҮбҹӮбһҖ бһ“бһ·бһ„бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ context бһҮбһ¶бһҳбһҪбһҷбһҹбҹҶбһҺбһҫбһҡбҹ” бһ“бҹҒбҹҮбһҮбһ¶бһҡбһ”бҹҖбһ”бһҠбҹӮбһӣ AI бһҮбһҮбҹӮбһҖбһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбҹ”
