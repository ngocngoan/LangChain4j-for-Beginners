# ការរួមចំណែក

អរគុណដែលលោកអ្នកមានចំណាប់អារម្មណ៍ក្នុងការរួមចំណែក! យើងសូមស្វាគមន៍ការស្នើសុំភ្ជាប់ (pull requests) និងការកែលម្អដល់មេរៀននេះ។ សូមអនុវត្តតាមណ័រណីយបណ្ណ័ទាំងនេះពេលប្រគល់ការផ្លាស់ប្តូរ៖

1. **មិនបញ្ចូលសម្ងាត់ក្នុងការបញ្ជូន commit ទេ។** មិនគួរបញ្ជូនកូនសោ API ឬឯកសារសម្ងាត់ណាមួយឡើយ។ ប្រើ `.env` ក្នុងកម្រិតមូលដ្ឋានសម្រាប់ការធ្វើតេស្តផ្ទាល់ខ្លួនរបស់អ្នក។
2. **រក្សាឲ្យការផ្លាស់ប្តូរត្រូវបានផ្ដោត។** Pull request តូចៗដែលមានកំណត់ផ្នែកច្បាស់លាស់ងាយស្រួលត្រួតពិនិត្យ និងថែទាំ។ ប្រសិនបើអ្នកកំពុងជួសជុលបញ្ហាច្រើន សូមផ្តល់អោយពួកវាដាច់ដោយឡែក។
3. **បន្ថែមតេស្តនៅកន្លែងដែលអាចធ្វើបាន។** តេស្តឯកតា ឬតេស្តបញ្ចូលចូលជួយបញ្ចេញកំហុសត្រឡប់ក្រោយ។ សម្រាប់ម៉ូឌុល Spring Boot សូមរួមបញ្ចូលតេស្តនៅក្នុងថត `src/test/java`។
4. **គោរពប្រពៃណីរចនាបថកូដ។** ប្រើរចនាសម្ព័ន្ធគម្រោងដែលមានស្រាប់ជាអ្នកដឹកនាំ។ យើងពឹងផ្អែកលើ Maven ដើម្បីគ្រប់គ្រងសំណុំអាស្រ័យ និងផ្លាក plugin Spring Boot Maven សម្រាប់វេចខ្ចប់កម្មវិធី។

អរគុណដែលជួយធ្វើឲ្យមេរៀននេះកាន់តែប្រសើរឡើង!
