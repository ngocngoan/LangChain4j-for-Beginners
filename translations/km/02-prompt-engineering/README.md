# Module 02: бһңбһ·бһҹбҹ’бһңбһҖбһҳбҹ’бһҳбһ”бһүбҹ’бһ…бһјбһӣбһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһ”бһүбҹ’бһҮбһ¶вҖӢбһҮбһ¶бһҳбһҪбһҷ GPT-5.2

- [бһңбһёбһҠбҹҒбһўбһјбһ”бһ„бҹ’бһ бһ¶бһү](#бһңбһёбһҠбҹҒбһўбһјбһ”бһ„бҹ’бһ бһ¶бһү)
- [бһўбҹ’бһңбһёбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҡбҹҖбһ“](#бһўбҹ’бһңбһёбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҡбҹҖбһ“)
- [бһӣбһҖбҹ’бһҒбһҒбһҺбҹ’бһҢбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“](#бһӣбһҖбҹ’бһҒбһҒбһҺбҹ’бһҢбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“)
- [бһҖбһ¶бһҡбһҷбһӣбҹӢбһҠбһ№бһ„бһўбҹҶбһ–бһёбһңбһ·бһҹбҹ’бһңбһҖбһҳбҹ’бһҳбһ”бһүбҹ’бһ…бһјбһӣбһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһ”бһүбҹ’бһҮбһ¶](#бһҖбһ¶бһҡбһҷбһӣбҹӢбһҠбһ№бһ„бһўбҹҶбһ–бһёбһңбһ·бһҹбҹ’бһңбһҖбһҳбҹ’бһҳбһ”бһүбҹ’бһ…бһјбһӣбһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһ”бһүбҹ’бһҮбһ¶)
- [бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһңбһ·бһҹбҹ’бһңбһҖбһҳбҹ’бһҳбһ”бһүбҹ’бһ…бһјбһӣбһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһ”бһүбҹ’бһҮбһ¶](#бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһңбһ·бһҹбҹ’бһңбһҖбһҳбҹ’бһҳбһ”бһүбҹ’бһ…бһјбһӣбһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһ”бһүбҹ’бһҮбһ¶)
  - [бһ”бһүбҹ’бһ…бһјбһӣбһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһ”бһүбҹ’бһҮбһ¶вҖӢбһ–бһёбһӮбҹ’бһҳбһ¶бһ“бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ](#бһ”бһүбҹ’бһ…бһјбһӣбһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһ”бһүбҹ’бһҮбһ¶вҖӢбһ–бһёбһӮбҹ’бһҳбһ¶бһ“бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ)
  - [бһ”бһүбҹ’бһ…бһјбһӣбһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһ”бһүбҹ’бһҮбһ¶вҖӢбһ–бһёбһҳбһ¶бһ“бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚе°‘ж•°](#бһ”бһүбҹ’бһ…бһјбһӣбһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһ”бһүбҹ’бһҮбһ¶вҖӢбһ–бһёбһҳбһ¶бһ“бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚе°‘ж•°)
  - [бһҒбҹ’бһҹбҹӮбһҹбһ„бҹ’бһңбһ¶бһҖбҹӢбһӮбҹҶбһ“бһ·бһҸ](#бһҒбҹ’бһҹбҹӮбһҹбһ„бҹ’бһңбһ¶бһҖбҹӢбһӮбҹҶбһ“бһ·бһҸ)
  - [бһ”бһүбҹ’бһ…бһјбһӣбһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһ”бһүбҹ’бһҮбһ¶бһ…бһјбһӣбһҸбһҪ](#бһ”бһүбҹ’бһ…бһјбһӣбһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһ”бһүбҹ’бһҮбһ¶бһ…бһјбһӣбһҸбһҪ)
  - [бһӮбҹҶбһҡбһјбһ”бһүбҹ’бһ…бһјбһӣбһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһ”бһүбҹ’бһҮбһ¶](#бһӮбҹҶбһҡбһјбһ”бһүбҹ’бһ…бһјбһӣбһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһ”бһүбҹ’бһҮбһ¶)
- [бһҡбһ…бһ“бһ¶бһ”бҹҗбһ‘бҹ’бһҳбһҮбһ¶бһ”бҹӢбһҖбһҳбҹ’бһҡбһ·бһҸбһҒбҹ’бһ–бһҹбҹӢ](#бһҡбһ…бһ“бһ¶бһ”бҹҗбһ‘бҹ’бһҳбһҮбһ¶бһ”бҹӢбһҖбһҳбҹ’бһҡбһ·бһҸбһҒбҹ’бһ–бһҹбҹӢ)
- [бһҡбһҸбҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё](#бһ”бһҫбһҖбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡвҖӢ-application)
- [бһҡбһјбһ”бһҗбһҸбһўбҹҒбһҖбҹ’бһҡбһ„бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё](#бһҡбһјбһ”бһ—бһ¶бһ–бһ“бҹғбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё)
- [бһҹбҹ’бһ‘бһјбһ…бһҹбҹ’бһ‘бһ„бҹӢбһҡбһ…бһ“бһ¶бһ”бҹҗбһ‘бҹ’бһҳ](#бһҹбҹ’бһңбҹӮбһ„бһҷбһӣбҹӢбһўбҹҶбһ–бһёбһӣбҹҶбһ“бһ¶бҹҶбһ”бһҪбһ“)
  - [бһ‘бҹҶбһ“бһ·бһүбһ‘бһ¶бһ” бһ“бһ·бһ„бһҒбҹ’бһ–бһҹбҹӢбһ“бҹғбһҖбһ¶бһҡбһҒбһ·бһҸбһҒбҹҶ](#бһӣбҹҶбһ“бһ¶бҹҶбһ”бһҪбһ“-low-vs-high-eagerness)
  - [бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ… (бһ”бһүбҹ’бһҮбһёбһ§бһ”бһҖбһҡбһҺбҹҚ)](#бһҖбһ¶бһҡбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…-бһҖбһ¶бһҡбһ–бһ“бҹ’бһҷбһӣбҹӢ-tool)
  - [бһҖбһјбһҠбһҠбҹӮбһӣбһўбһ¶бһ…бһ’бҹ’бһңбһҫбһ…бһ·бһҸбҹ’бһҸбһҒбҹ’бһӣбһҪбһ“бһҜбһ„](#бһҖбһјбһҠбһ”бһүбҹ’бһүбҹ’бһҡбһ–бһ·бһ…бһ¶бһҡбһҺбһ¶бһҠбҹ„бһҷбһҒбҹ’бһӣбһҪбһ“бһҜбһ„)
  - [бһҖбһ¶бһҡбһңбһ·бһ—бһ¶бһӮбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’](#бһңбһ·бһ—бһ¶бһӮбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’)
  - [бһҖбһ¶бһҡбһҮбһҮбҹӮбһҖбһ…бһӣбҹҗбһҸбһ…бҹ’бһҡбһҫбһ“бһҮбһ»бҹҶ](#бһҖбһ¶бһҡбһҹбһ“бҹ’бһ‘бһ“бһ¶бһ…бҹ’бһҡбһҫбһ“бһҮбҹҶбһҡбһҫбһҹ)
  - [бһҖбһ¶бһҡбһҷбһӣбҹӢбһҠбһ№бһ„бһҮбҹҶбһ бһ¶бһ“бһҳбһҪбһҷбҹ—](#бһҖбһ¶бһҡбһ–бһ·бһ…бһ¶бһҡбһҺбһ¶бһҮбһ¶бһҠбҹҶбһҺбһ¶бһҖбҹӢбһҖбһ¶бһӣ)
  - [бһӣбһ‘бҹ’бһ’бһ•бһӣбһҠбҹӮбһӣбһҳбһ¶бһ“бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢ](#бһҖбһ¶бһҡбһ”бһүбҹ’бһ…бҹҒбһүбһӣбһ‘бҹ’бһ’бһ•бһӣбһҠбҹӮбһӣбһҳбһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢ)
- [бһўбҹ’бһңбһёбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ–бһ·бһҸбһҮбһ¶бһҖбҹҶбһ–бһ»бһ„бһҡбҹҖбһ“](#бһўбҹ’бһңбһёбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһҖбҹҶбһ–бһ»бһ„бһҡбҹҖбһ“бһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠ)
- [бһҮбҹҶбһ бһ¶бһ“бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢ](#бһҮбҹҶбһ бһ¶бһ“бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢ)

## бһўбҹ’бһңбһёбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҡбҹҖбһ“

бһҒбҹ’бһҹбҹӮбһ—бһ¶бһ–бһҷбһ“бҹ’бһҸбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһ•бҹ’бһҸбһӣбҹӢбһҮбһ¶бһ‘бһ·бһҠбҹ’бһӢбһ—бһ¶бһ–бһ‘бһјбһӣбҹҶбһ‘бһјбһӣбһ¶бһҷбһўбҹҶбһ–бһёбһ”бҹ’бһҡбһ’бһ¶бһ“бһ”бһ‘бһ“бһ·бһ„бһҮбҹҶбһ“бһ¶бһүбһҹбҹҶбһҒбһ¶бһ“бҹӢбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ“бһ№бһ„бһўwickбһ¶бһЎбһҫбһ„бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҳбҹүбһјбһҢбһ»бһӣбһ“бҹҒбҹҮ вҖ” бһ…бһ¶бһ”бҹӢбһ–бһёбһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶бһҖбҹӮбһӣбҹҶбһўбһ”бһүбҹ’бһ…бһјбһӣбһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһ”бһүбҹ’бһҮбһ¶бһ‘бҹ…бһҠбһӣбҹӢбһӣбҹҶбһ“бһ¶бҹҶбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҮбҹҶбһ бһ¶бһ“бһҳбһҪбһҷбҹ—бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ“бһ№бһ„бһўбһ“бһ»бһңбһҸбҹ’бһҸбҹ”

бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҳбҹүбһјбһҢбһ»бһӣбһҳбһ»бһ“бҹ— бһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһҹбҹ’бһңбҹӮбһ„бһҷбһӣбҹӢбһўбҹҶбһ–бһёбһ”бҹ’бһҡбһҸбһ·бһҖбһҳбҹ’бһҳбһ‘бһҳбҹ’бһҡбһ„бҹӢLangChain4j бһҮбһ¶бһҳбһҪбһҷбһҳбҹүбһјбһҢбҹӮбһӣ GitHub бһ бһҫбһҷбһ”бһ¶бһ“бһғбһҫбһүбһ–бһёбһҡбһ”бҹҖбһ”бһҠбҹӮбһӣбһўбһ„бҹ’бһӮбһ…бһ„бһ…бһ¶бҹҶбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһҮбһҮбҹӮбһҖбһҮбһ¶бһҳбһҪбһҷ AI бһҮбһ¶бһҳбһҪбһҷ Azure OpenAIбҹ” бһ–бҹҒбһӣбһ“бҹҒбҹҮбһ–бһҪбһҖбһҷбһҫбһ„бһ“бһ№бһ„бһ•бҹ’бһҸбҹ„бһҸбһӣбһҫбһҡбһ”бҹҖбһ”бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһҹбһҪбһҡбһҹбҹҶбһ“бһҪбһҡ вҖ” бһӮбһәбһ”бһүбҹ’бһ…бһјбһӣбһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһ”бһүбҹ’бһҮбһ¶бһ•бҹ’бһ‘бһ¶бһӣбҹӢ вҖ” бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ GPT-5.2 бһҡбһ”бһҹбҹӢ Azure OpenAIбҹ” бһҡбһ”бҹҖбһ”бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһҡбҹҖбһ”бһ…бҹҶбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһ”бһүбҹ’бһ…бһјбһӣбһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһ”бһүбҹ’бһҮбһ¶бһўбһ¶бҹ’бһ“бһҖбһ”бҹүбҹҮбһ–бһ¶бһӣбҹӢбһҷбҹүбһ¶бһ„бһҒбҹ’бһӣбһ¶бҹҶбһ„бһҠбһӣбҹӢбһӮбһ»бһҺбһ—бһ¶бһ–бһ“бҹғбһ…бһҳбҹ’бһӣбһҫбһҷбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бҹ” бһҷбһҫбһ„бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһ–бһёбһҖбһ¶бһҡбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһ”бһүбҹ’бһ…бһјбһӣбһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһ”бһүбҹ’бһҮбһ¶бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“ бһ бһҫбһҷбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖбһ‘бҹ…бһҡбһ…бһ“бһ¶бһ”бҹҗбһ‘бҹ’бһҳвҖӢбһҮбһ¶бһ”бҹӢбһҖбһҳбҹ’бһҡбһ·бһҸбһҒбҹ’бһ–бһҹбҹӢбһ”бҹ’бһҡбһ¶бҹҶбһ”бһҪбһ“ бһҠбҹӮбһӣбһ”бҹ’бһҡбһҫбһўбһҸбҹ’бһҗбһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбһ–бҹҒбһүбһӣбҹҒбһүбһ“бҹғбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ– GPT-5.2бҹ”

бһҷбһҫбһ„бһ“бһ№бһ„бһ”бҹ’бһҡбһҫ GPT-5.2 бһ–бҹ’бһҡбҹ„бҹҮбһңбһ¶бһ“бһ¶бҹҶбһҳбһҖбһ“бһјбһңбһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһғбҹ’бһӣбһ¶бһ“бһӮбҹҶбһ“бһ·бһҸ вҖ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бҹ’бһҡбһ¶бһ”бҹӢбһҳбҹүбһјбһҢбҹӮбһӣбһ–бһёбһ…бҹҶбһ“бһҪбһ“бһҖбһ¶бһҡбһӮбһ·бһҸбһҠбҹӮбһӣбһӮбһҪбһҡбһ’бҹ’бһңбһҫбһҳбһ»бһ“бһ–бҹҒбһӣбһҶбҹ’бһӣбһҫбһҷбҹ” бһ“бҹҒбҹҮбһ’бҹ’бһңбһҫбһұбҹ’бһҷбһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҡбҹ’бһҠбһ”бһүбҹ’бһ…бһјбһӣбһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһ”бһүбҹ’бһҮбһ¶бһ—бҹ’бһӣбһ¶бһҸбҹӢбҹ—бһҮбһ¶бһ„бһҳбһ»бһ“ бһ“бһ·бһ„бһҮбһҪбһҷбһўбҹ’бһ“бһҖбһҷбһӣбҹӢбһҗбһ¶бһҸбһҫбһ–бҹҒбһӣбһҺбһ¶бһҸбҹ’бһҡбһјбһңбһ”бҹ’бһҡбһҫбһңбһ·бһ’бһёбһҺбһ¶бһҳбһҪбһҷбҹ” бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһўбһҸбҹ’бһҗбһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбһ–бһёбһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ¶бһӮбһ”бҹӢ Azure бһҸбһ·бһ…бһҮбһ¶бһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ GPT-5.2 бһ”бҹ’бһҡбһҖбҹ’бһҡбҹ„бҹҮбһҳбҹүбһјбһҢбҹӮбһӣ GitHubбҹ”

## бһӣбһҖбҹ’бһҒбһҒбһҺбҹ’бһҢбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“

- бһ”бһ¶бһ“бһ”бһүбҹ’бһ…бһ”бҹӢ Module 01 (бһ’бһ“бһ’бһ¶бһ“ Azure OpenAI бһ”бһ¶бһ“бһҠбһ¶бһҖбҹӢбһ…бҹҒбһү)
- бһҜбһҖбһҹбһ¶бһҡ `.env` бһ“бҹ…бһҗбһҸбһ«бһҹбһҮбһ¶бһҳбһҪбһҷбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһҹбһҳбҹ’бһ—бһ¶бһҡбҹҲ Azure (бһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһҠбҹ„бһҷ `azd up` бһҖбҹ’бһ“бһ»бһ„ Module 01)

> **бһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбҹ–** бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһҳбһ·бһ“бһ‘бһ¶бһ“бҹӢбһ”бһ¶бһ“бһ”бһүбҹ’бһ…бһ”бҹӢ Module 01 бһҹбһјбһҳбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһ…бҹҒбһүбһ“бҹ…бһ‘бһёбһ“бҹ„бҹҮбһҮбһ¶бһҳбһҪбһҷбҹ”

## бһҖбһ¶бһҡбһҷбһӣбҹӢбһҠбһ№бһ„бһўбҹҶбһ–бһёбһңбһ·бһҹбҹ’бһңбһҖбһҳбҹ’бһҳбһ”бһүбҹ’бһ…бһјбһӣбһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһ”бһүбҹ’бһҮбһ¶

бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ“бҹғбһңбһ·бһҹбҹ’бһңбһҖбһҳбҹ’бһҳбһ”бһүбҹ’бһ…бһјбһӣбһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһ”бһүбҹ’бһҮбһ¶бһӮбһәбһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бһҡбһңбһ¶бһ„бһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶбһҳбһ·бһ“бһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢбһ“бһ·бһ„бһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶбһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢ бһҠбһјбһ…бһҠбҹӮбһӣбһҖбһ¶бһҡбһ”бҹ’бһҡбҹҖбһ”бһ’бҹҖбһ”бһ‘бһёбһӣбһҫбһ”бһ¶бһ“бһ”бһ„бҹ’бһ бһ¶бһүбҹ”

<img src="../../../translated_images/km/what-is-prompt-engineering.5c392a228a1f5823.webp" alt="What is Prompt Engineering?" width="800"/>

бһңбһ·бһҹбҹ’бһңбһҖбһҳбҹ’бһҳбһ”бһүбҹ’бһ…бһјбһӣбһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһ”бһүбҹ’бһҮбһ¶бһӮбһәбһҖбһ¶бһҡбһҡбһ…бһ“бһ¶бһўбһҸбҹ’бһҗбһ”бһ‘бһ”бһүбҹ’бһ…бһјбһӣбһҠбҹӮбһӣбһ”бһ¶бһ“бһӣбһҫбһҖбһ‘бһ№бһҖбһ…бһ·бһҸбҹ’бһҸбһұбҹ’бһҷбһўбҹ’бһ“бһҖбһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһӣбһ‘бҹ’бһ’бһ•бһӣбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбҹ” бһңбһ¶бһҳбһ·бһ“бһҳбҹӮбһ“бһ”бһүбҹ’бһ…бһјбһӣбһҹбҹҶбһ“бһҪбһҡбһҸбҹӮбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮбһ‘бҹҒ вҖ” бһңбһ¶бһӮбһәбһҖбһ¶бһҡбһҡбҹҖбһ”бһ…бҹҶбһҖбһ¶бһҡбһҹбҹ’бһ“бһҫбһҡбһҖбҹ’бһ“бһ»бһ„бһҷбҹүбһ¶бһ„бһҠбҹӮбһӣбһҳбҹүбһјбһҢбҹӮбһӣбһ“бһ№бһ„бһҷбһӣбҹӢбһҷбҹүбһ¶бһ„бһ…бҹ’бһ”бһ¶бһҹбҹӢбһҗбһ¶бһўбҹ’бһ“бһҖбһ…бһ„бҹӢбһ”бһ¶бһ“бһўбҹ’бһңбһё бһ“бһ·бһ„бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҡбҹ’бһҠбһҠбһ№бһҖбһҮбһүбҹ’бһҮбһјбһ“бҹ”

бһӮбһ·бһҸбһҗбһ¶бһңбһ¶бһҠбһјбһ…бһҮбһ¶бһҖбһ¶бһҡбһ•бҹ’бһҸбһӣбҹӢбһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶбһ‘бҹ…бһҖбҹ’бһҡбһ»бһҳбһҖбһ¶бһҡбһ„бһ¶бһҡбһҳбҹ’бһ“бһ¶бһҖбҹӢбһҳбһҪбһҷбҹ” "бһҮбһҪбһҹбһҮбһ»бһӣбһҖбҹҶбһ бһ»бһҹ" бһӮбһәбһҳбһ·бһ“бһ…бҹ’бһ”бһ¶бһҹбҹӢбҹ” "бһҮбһҪбһҹбһҮбһ»бһӣбһҖбҹҶбһ бһ»бһҹ null pointer бһ“бҹ… UserService.java бһҮбһҪбһҡбһӣбҹҒбһҒ 45 бһҠбҹ„бһҷбһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷ null" бһӮбһәбһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢбҹ” бһҳбҹүбһјбһҢбҹӮбһӣбһ—бһ¶бһҹбһ¶бһ…бҹҒбҹҮбһ’бҹ’бһңбһҫбһҠбһјбһ…бһӮбҹ’бһ“бһ¶ вҖ” бһҖбһ¶бһҡбһ”бһүбҹ’бһҮбһ¶бһҖбҹӢ бһ“бһ·бһ„бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҳбһ¶бһ“бһҹбһ¶бһҡбҹҲбһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ”

бһҡбһјбһ”бһ—бһ¶бһ–бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһҡбһ”бҹҖбһ”бһҠбҹӮбһӣ LangChain4j бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһ¶бһҡбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢ вҖ”бһ—бҹ’бһҮбһ¶бһ”бҹӢбһӮбҹҶбһҡбһјбһҹбҹҶбһҺбһ„бҹӢбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ‘бҹ…бһҳбҹүбһјбһҢбҹӮбһӣбһҸбһ¶бһҳбһҡбһҷбҹҲбһ”бҹ’бһӣбһ»бһҖбһҹбһ¶бһҡбһҡбһ…бһ“бһ¶бһҠбһјбһ…бһҮбһ¶ SystemMessage бһ“бһ·бһ„ UserMessageбҹ”

<img src="../../../translated_images/km/how-langchain4j-fits.dfff4b0aa5f7812d.webp" alt="How LangChain4j Fits" width="800"/>

LangChain4j бһ•бҹ’бһҸбһӣбҹӢбһўбһӮбҹ’бһӮбһ·бһҹбһ“бһёбһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶ вҖ” бһҖбһ¶бһҡбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҳбҹүбһјбһҢбҹӮбһӣ бһўбһ„бҹ’бһӮбһ…бһ„бһ…бһ¶бҹҶ бһ“бһ·бһ„бһ”бҹ’бһҡбһ—бҹҒбһ‘бһҹбһ¶бһҡ вҖ” бһҒбһҺбҹҲбһ–бҹҒбһӣбһҠбҹӮбһӣбһҡбһ…бһ“бһ¶бһ”бҹҗбһ‘бҹ’бһҳбһ”бһүбҹ’бһ…бһјбһӣбһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһ”бһүбҹ’бһҮбһ¶бһӮбһәбһўбһҸбҹ’бһҗбһ”бһ‘бһҠбҹӮбһӣбһ”бһ¶бһ“бһҡбҹҖбһ”бһ…бҹҶбһҷбҹүбһ¶бһ„бһ”бҹ’бһҡбһ»бһ„бһ”бҹ’бһҡбһҷбҹҗбһҸбҹ’бһ“бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ•бҹ’бһүбһҫбһҸбһ¶бһҳбһҡбһҷбҹҲбһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶бһ“бҹҒбҹҮбҹ” бһ”бҹ’бһӣбһ»бһҖбһҹбҹҶбһҒбһ¶бһ“бҹӢбһӮбһә `SystemMessage` (бһҖбҹҶбһҺбһҸбҹӢбһўбһ¶бһҖбһ”бҹ’бһ”бһҖбһ·бһҡбһ·бһҷбһ¶ бһ“бһ·бһ„бһҸбһҪбһҡбһҸбҹҶбһҺбһ¶бһ„ AI) бһ“бһ·бһ„ `UserMessage` (бһ”бһүбҹ’бһ…бҹҒбһүбһҹбҹҶбһҺбһҫбһҡбһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ)бҹ”

## бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһңбһ·бһҹбҹ’бһңбһҖбһҳбҹ’бһҳбһ”бһүбҹ’бһ…бһјбһӣбһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһ”бһүбҹ’бһҮбһ¶

бһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹбһ…бһҳбҹ’бһ”бһ„бһ”бҹ’бһҡбһ¶бҹҶбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһ”бһ„бҹ’бһҖбһҫбһҸбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ“бҹғбһңбһ·бһҹбҹ’бһңбһҖбһҳбҹ’бһҳбһ”бһүбҹ’бһ…бһјбһӣбһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһ”бһүбҹ’бһҮбһ¶бһҠбҹӮбһӣбһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бҹ” бһҳбһҪбһҷбҹ—бһҹбҹҶбһҠбҹ…бһӣбһҫбһ•бҹ’бһ“бҹӮбһҖбһ•бҹ’бһҹбҹҒбһ„бһӮбҹ’бһ“бһ¶бһ“бҹғбһҡбһ”бҹҖбһ”бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһҮбһ¶бһҳбһҪбһҷбһҳбҹүбһјбһҢбҹӮбһӣбһ—бһ¶бһҹбһ¶бҹ”

<img src="../../../translated_images/km/five-patterns-overview.160f35045ffd2a94.webp" alt="Five Prompt Engineering Patterns Overview" width="800"/>

бһҳбһ»бһ“бһ…бһјбһӣбһ‘бҹ…бһҖбһ¶бһ“бҹӢбһҡбһ…бһ“бһ¶бһ”бҹҗбһ‘бҹ’бһҳбһҮбһ¶бһ”бҹӢбһҖбһҳбҹ’бһҡбһ·бһҸбһҒбҹ’бһ–бһҹбҹӢбһҖбҹ’бһ“бһ»бһ„бһҳбҹүбһјбһҢбһ»бһӣбһ“бҹҒбҹҮ бһҒбһҺбҹҲбһ–бҹҒбһӣбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһ”бһүбҹ’бһ…бһјбһӣбһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһ”бһүбҹ’бһҮбһ¶бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ”бҹ’бһҡбһ¶бҹҶбһңбһ¶бҹ” бһңбһ¶бһҮбһ¶бһ”бҹ’бһӣбһ»бһҖбһҹбҹҶбһ”бһ»бһҸбҹ’бһҡбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһңбһ·бһҹбҹ’бһңбһҖбһҳбҹ’бһҳбһ”бһүбҹ’бһ…бһјбһӣбһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһ”бһүбҹ’бһҮбһ¶бһӮбһҪбһҡбһҸбҹӮбһҹбҹ’бһӮбһ¶бһӣбҹӢбҹ” бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһўбһ“бһ»бһңбһҸбҹ’бһҸбһ–бһё [бһҳбҹүбһјбһҢбһ»бһӣбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҡбһ бҹҗбһҹ](../00-quick-start/README.md#2-prompt-patterns) бһ“бҹ„бҹҮбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһғбһҫбһүбһңбһ¶бһҮбһ¶бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸ вҖ” бһ“бҹҒбҹҮбһӮбһәбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһӮбҹҶбһ“бһ·бһҸбһҸбһјбһ…бҹ—бһ“бҹ…бһ–бһёбһҖбҹ’бһҡбҹ„бһҷбһңбһ¶бҹ”

### бһ”бһүбҹ’бһ…бһјбһӣбһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһ”бһүбҹ’бһҮбһ¶вҖӢбһ–бһёбһӮбҹ’бһҳбһ¶бһ“бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ

бһңбһ·бһ’бһёбһҹбһ¶бһҳбһүбҹ’бһүбһ”бҹҶбһ•бһ»бһҸбҹ– бһ•бҹ’бһҸбһӣбҹӢбһ–бһ·бһ’бһёбһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһ‘бҹ…бһҳбҹүбһјбһҢбҹӮбһӣбһҠбҹ„бһҷбһӮбҹ’бһҳбһ¶бһ“бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҺбһ¶бһҳбһҪбһҷбҹ” бһҳбҹүбһјбһҢбҹӮбһӣбһ–бһ№бһ„бһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһҖбһ¶бһҡбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһҡбһ”бһҹбҹӢбһҒбҹ’бһӣбһҪбһ“бһ‘бһ¶бҹҶбһ„бһҳбһјбһӣбһҠбһҫбһҳбҹ’бһ”бһёбһҷбһӣбҹӢ бһ“бһ·бһ„бһўбһ“бһ»бһңбһҸбҹ’бһҸбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бҹ” бһңбһ¶бһҡбһҖбҹ’бһҹбһ¶бһ”бһ¶бһ“бһӣбҹ’бһўбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҹбҹ’бһ“бһҫбһҹбһјбһ…бһјбһӣбһҹбһ¶бһҳбһүбҹ’бһүбһҠбҹӮбһӣбһўбһ¶бһҖбһ”бҹ’бһ”бһҖбһ·бһҡбһ·бһҷбһ¶бһҠбҹӮбһӣбһҡбҹҶбһ–бһ№бһ„бһ‘бһ»бһҖбһӮбһәбһ…бҹ’бһ”бһ¶бһҹбҹӢбҹ”

*бһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҠбҹ„бһҷбһӮбҹ’бһҳбһ¶бһ“бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ вҖ” бһҳбҹүбһјбһҢбҹӮбһӣбһҷбһӣбҹӢбһ–бһёбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһ–бһёбһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶбһҸбҹӮбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮ*

```java
String prompt = "Classify this sentiment: 'I absolutely loved the movie!'";

```
  
**бһ–бҹҒбһӣбһҺбһ¶бһҸбҹ’бһҡбһјбһңбһ”бҹ’бһҡбһҫбҹ–** бһҖбһ¶бһҡбһ”бҹӮбһ„бһ…бҹӮбһҖбһҹбһ¶бһҳбһүбҹ’бһү бһҹбҹҶбһ“бһҪбһҡбһ•бҹ’бһ‘бһ¶бһӣбҹӢ бһҖбһ¶бһҡбһ”бһҖбһ”бҹ’бһҡбҹӮ бһ¬бһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһҺбһ¶бһҳбһҪбһҷбһҠбҹӮбһӣбһҳбҹүбһјбһҢбҹӮбһӣбһўбһ¶бһ…бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҠбҹ„бһҷбһӮбҹ’бһҳбһ¶бһ“бһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶбһ”бһ“бҹ’бһҗбҹӮбһҳбҹ”

### бһ”бһүбҹ’бһ…бһјбһӣбһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһ”бһүбҹ’бһҮбһ¶вҖӢбһ–бһёбһҳбһ¶бһ“бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚе°‘ж•°

бһ•бҹ’бһҸбһӣбҹӢбһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҠбҹӮбһӣбһ”бһ„бҹ’бһ бһ¶бһүбһӣбҹҶбһ“бһ¶бҹҶбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ…бһ„бҹӢбһұбҹ’бһҷбһҳбҹүбһјбһҢбҹӮбһӣбһўбһ“бһ»бһңбһҸбҹ’бһҸбҹ” бһҳбҹүбһјбһҢбҹӮбһӣбһҡбҹҖбһ“бһ–бһёбһ‘бҹ’бһҡбһ„бҹӢбһ‘бҹ’бһҡбһ¶бһҷбһ”бһүбҹ’бһ…бһјбһӣ-бһӣбһ‘бҹ’бһ’бһ•бһӣбһҠбҹӮбһӣбһҡбҹҶбһ–бһ№бһ„бһ‘бһ»бһҖбһ–бһёбһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ бһҫбһҷбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ‘бҹ…бһӣбһҫбһ”бһүбҹ’бһ…бһјбһӣбһҗбҹ’бһҳбһёбҹ” бһңбһ¶бһҮбҹҶбһ“бһҪбһҷбһҮбһ¶бһ”бҹӢбһӣбһ¶бһ”бҹӢбһҒбҹ’бһӣбһ¶бҹҶбһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһҠбҹӮбһӣбһ‘бҹ’бһҡбһ„бҹӢбһ‘бҹ’бһҡбһ¶бһҷбһ¬бһўбһ¶бһҖбһ”бҹ’бһ”бһҖбһ·бһҡбһ·бһҷбһ¶бһҠбҹӮбһӣбһҳбһ·бһ“бһ…бҹ’бһ”бһ¶бһҹбҹӢбҹ”

```java
String prompt = """
```

### бһҒбҹ’бһҹбҹӮбһҹбһ„бҹ’бһңбһ¶бһҖбҹӢбһӮбҹҶбһ“бһ·бһҸ

бһҹбҹ’бһ“бһҫбһұбҹ’бһҷбһҳбҹүбһјбһҢбҹӮбһӣбһ”бһ„бҹ’бһ бһ¶бһүбһҖбһ¶бһҡбһӮбһ·бһҸбһҡбһ”бһҹбҹӢбһҒбҹ’бһӣбһҪбһ“бһҮбһ¶бһҮбҹҶбһ бһ¶бһ“бҹ—бҹ” бһҮбҹҶбһ“бһҪбһҹбһҖбһ¶бһҡбһҮбһ¶бһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһҶбҹ’бһӣбһҫбһҷбһҸбһ”бһ—бҹ’бһӣбһ¶бһҳбҹ— бһҳбҹүбһјбһҢбҹӮбһӣбһ”бҹҶбһ”бҹӮбһҖбһ”бһүбҹ’бһ бһ¶бһҖбҹ’бһ“бһ»бһ„бһҮбҹҶбһ бһ¶бһ“бһҸбһјбһ…бҹ—бһ“бһ·бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҸбһ¶бһҳбһҮбҹҶбһ бһ¶бһ“бҹ” бһңбһ¶бһҮбһҪбһҷбһҖбҹӮбһӣбһҳбҹ’бһўбһ—бһ¶бһ–бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһӣбһҫбһӮбһҺбһ·бһҸбһңбһ·бһ‘бҹ’бһҷбһ¶ бһҸбһҖбҹ’бһҖбҹ’бһҡбһ»бһңбһ·бһ‘бҹ’бһҷбһ¶ бһ“бһ·бһ„бһҖбһ¶бһҡбһҷбһӣбҹӢбһҠбһ№бһ„бһ–бһ бһ»бһҮбҹҶбһ бһ¶бһ“бҹ”

*бһҖбһ¶бһҡбһӮбһ·бһҸбһҮбһ¶бһҮбҹҶбһ бһ¶бһ“бҹ— вҖ” бһ”бҹҶбһ”бҹӮбһҖбһ”бһүбҹ’бһ бһ¶бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһүбһ‘бҹ…бһҮбһ¶бһҮбҹҶбһ бһ¶бһ“бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢ*

```java
String prompt = """
    Problem: A store has 15 apples. They sell 8 apples and then 
    receive a shipment of 12 more apples. How many apples do they have now?
    
    Let's solve this step-by-step:
    """;
String response = model.chat(prompt);
// бһҳбҹүбһјбһҠбҹӮбһӣбһ”бһ„бҹ’бһ бһ¶бһүбһҗбһ¶бҹ– 15 - 8 = 7, бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖ 7 + 12 = 19 бһ•бҹ’бһӣбҹӮбһ”бҹүбҹ„бһҳ
```
  
**бһ–бҹҒбһӣбһҺбһ¶бһҸбҹ’бһҡбһјбһңбһ”бҹ’бһҡбһҫбҹ–** бһ”бһүбҹ’бһ бһ¶бһӮбһҺбһ·бһҸбһңбһ·бһ‘бҹ’бһҷбһ¶ бһ”бҹ’бһҡбһҠбһ¶бһ”бҹӢбһЎбһјбһ бҹ’бһӮбһ·бһҖ бһҖбһ¶бһҡбһҖбҹӮбһӣбһҳбҹ’бһўбһҖбҹҶбһ бһ»бһҹ бһ¬бһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһҺбһ¶бһҳбһҪбһҷбһҠбҹӮбһӣбһ”бһ„бҹ’бһ бһ¶бһүбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһӮбһ·бһҸбһҮбһҪбһҷбһҖбҹӮбһӣбһҳбҹ’бһўбһ—бһ¶бһ–бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң бһ“бһ·бһ„бһ‘бҹҶбһ“бһ»бһҖбһ…бһ·бһҸбҹ’бһҸбҹ”

### бһ”бһүбҹ’бһ…бһјбһӣбһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһ”бһүбҹ’бһҮбһ¶бһ…бһјбһӣбһҸбһҪ

бһҖбҹҶбһҺбһҸбҹӢбһҸбһҪбһ¬бһ”бһ»бһӮбҹ’бһӮбһӣбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ AI бһҳбһ»бһ“бһ–бҹҒбһӣбһҹбһҪбһҡбһҹбҹҶбһҺбһҪбһҡбҹ” бһңбһ¶бһ•бҹ’бһҸбһӣбҹӢбһ”бһҡбһ·бһ”бһ‘бһҠбҹӮбһӣбһҡбҹҖбһ”бһ…бҹҶбһҹбһҳбҹ’бһӣбҹҒбһ„ бһҮбһҳбҹ’бһҡбҹ… бһ“бһ·бһ„бһҳбһ»бһҒбһ„бһ¶бһҡбһҖбһ¶бһҡбһҸбһ”бһҹбһјбһҳ бһҡбҹҖбһ”бһ…бҹҶбһҠбһјбһ…бһҮбһ¶ "бһңбһ·бһҹбҹ’бһңбһҖбһҡбһ‘бһјбһҡбһӮбһҳбһ“бһ¶бһӮбһҳбһ“бҹҚ" бһ•бҹ’бһҸбһӣбҹӢбһҷбҹ„бһ”бһӣбҹӢбһ•бҹ’бһҹбҹҒбһ„бһ–бһё "бһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһҖбһҳбҹ’бһҡбһ·бһҸбһҮбһ”бҹӢбһӣбҹҖбһ„" бһ¬ "бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–"бҹ”

*бһҖбҹҶбһҺбһҸбҹӢбһ”бһҡбһ·бһ”бһ‘ бһ“бһ·бһ„бһҸбһҪ вҖ” бһҹбҹҶбһҺбһҪбһҡбһҠбһјбһ…бһӮбҹ’бһ“бһ¶бһ”бһ¶бһ“бһ…бһҳбҹ’бһӣбһҫбһҷбһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бһҠбҹ„бһҷбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһҸбһҪбһҠбҹӮбһӣбһ”бһ¶бһ“бһ•бҹ’бһҠбһӣбҹӢ*

```java
String prompt = """
    You are an experienced software architect reviewing code.
    Provide a brief code review for this function:
    
    def calculate_total(items):
        total = 0
        for item in items:
            total = total + item['price']
        return total
    """;
String response = model.chat(prompt);
```
  
**бһ–бҹҒбһӣбһҺбһ¶бһҸбҹ’бһҡбһјбһңбһ”бҹ’бһҡбһҫбҹ–** бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҖбһјбһҠ бһ”бһ„бҹ’бһҡбҹҖбһ“ бһңбһ·бһ—бһ¶бһӮбһ–бһ·бһҹбҹҒбһҹбһҠбҹӮбһ“ бһ¬бһ–бҹҒбһӣбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ…бһҳбҹ’бһӣбһҫбһҷбһ–бһ·бһҹбҹҒбһҹбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҳбҹ’бһҡбһ·бһҸбһҮбҹҶбһ“бһ¶бһү бһ¬бһ‘бһҹбҹ’бһҹбһ“бһңбһ·бһҹбҹҗбһҷбһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢбһҳбһҪбһҷбҹ”

### бһӮбҹҶбһҡбһјбһ”бһүбҹ’бһ…бһјбһӣбһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһ”бһүбҹ’бһҮбһ¶

бһ”бһ„бҹ’бһҖбһҫбһҸбһ”бһүбҹ’бһ…бһјбһӣбһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһ”бһүбҹ’бһҮбһ¶бһҠбҹӮбһӣбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһҮбһ¶бһҗбҹ’бһҳбһёбһҮбһ¶бһҳбһҪбһҷбһ…бҹҶбһҺбһ»бһ…бһ”бҹ’бһҡбҹӮбһ”бҹ’бһҡбһ¶бһҹбҹӢбҹ” бһҮбҹҶбһ“бһҪбһҹбһҖбһ¶бһҡбһҹбһҡбһҹбҹҒбһҡбһ”бһүбҹ’бһ…бһјбһӣбһҗбҹ’бһҳбһёбһҡбҹҖбһ„бһҡбһ¶бһӣбҹӢбһҠбһ„ бһҖбҹҶбһҺбһҸбҹӢбһӮбҹҶбһҡбһјбһҳбһҪбһҷбһ бһҫбһҷбһ”бһүбҹ’бһ…бһјбһӣбһҸбһҳбҹ’бһӣбҹғбһ•бҹ’бһҹбҹҒбһ„бҹ—бҹ” бһҗбҹ’бһ“бһ¶бһҖбҹӢ `PromptTemplate` бһҡбһ”бһҹбҹӢ LangChain4j бһ’бҹ’бһңбһҫбһұбҹ’бһҷбһңбһ¶бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһҮбһ¶бһҳбһҪбһҷбһңбһ…бһ“бһ¶бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡ `{{variable}}`бҹ”

**бһ–бҹҒбһӣбһҺбһ¶бһҸбҹ’бһҡбһјбһңбһ”бҹ’бһҡбһҫбҹ–** бһҖбһ¶бһҡбһҹбҹ’бһ“бһҫбһҹбһ»бҹҶбһ–бһёбһҡбһ”бҹӢбһ”бҹҖбһ”бһ’бҹ’бһңбһҫбһ•бҹ’бһҹбҹҒбһ„бҹ— бһҖбһ¶бһҡбһ”бҹ’бһҡбһҳбһјбһӣбһ•бҹ’бһҸбһ»бҹҶ бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҮбһ¶бһҳбһҪбһҷ AI бһ¬бһҹбҹ’бһҗбһ¶бһ“бһҖбһ¶бһҡбһҺбҹҚбһҺбһ¶бһҳбһҪбһҷбһҠбҹӮбһӣбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһ”бһүбҹ’бһ…бһјбһӣбһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһ”бһүбҹ’бһҮбһ¶бһ“бҹ…бһҹбҹ’бһҗбһ·бһҸбһ“бҹ…бһҸбҹӮбһҳбһ·бһ“бһ”бҹ’бһҡбҹӮбһ”бҹ’бһҡбһҪбһӣ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ”бҹ’бһҸбһјбһҡбһҸбһ¶бһҳбҹ”

бһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹбһ”бһҪбһ“бһ“бҹҒбҹҮбһ•бҹ’бһҸбһӣбҹӢбһұбҹ’бһҷбһўбҹ’бһ“бһҖбһ“бһјбһңбһ§бһ”бһҖбһҡбһҺбҹҚбһҡбһ№бһ„бһҳбһ¶бҹҶбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһ—бһ¶бһӮбһ…бҹ’бһҡбһҫбһ“бһ“бҹғбһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһјбһӣбһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһ”бһүбҹ’бһҮбһ¶бҹ” бһ”бҹҶбһҺбҹӮбһҖбһ“бҹ…бһҹбһӣбҹӢбһ“бҹғбһҳбҹүбһјбһҢбһ»бһӣбһ“бҹҒбҹҮбһ”бһ„бҹ’бһҖбһҫбһҸбһӣбһҫбһңбһ¶бһҮбһ¶бһҳбһҪбһҷ **бһҡбһ…бһ“бһ¶бһ”бҹҗбһ‘бҹ’бһҳбһҮбһ¶бһ”бҹӢбһҖбһҳбҹ’бһҡбһ·бһҸбһҒбҹ’бһ–бһҹбҹӢбһ”бҹ’бһҡбһ¶бҹҶбһ”бһҪбһ“** бһҠбҹӮбһӣбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбһӮбҹ’бһҡбһ”бҹӢбһӣбһҖбҹ’бһҒбһҺбҹҲ reasoning control, бһҖбһ¶бһҡбһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһҒбҹ’бһӣбһҪбһ“бһҜбһ„ бһ“бһ·бһ„бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһӣбһ‘бҹ’бһ’бһ•бһӣбһҠбҹӮбһӣбһҳбһ¶бһ“бһҡбһ…бһ“бһ¶бҹ”

## бһҡбһ…бһ“бһ¶бһ”бҹҗбһ‘бҹ’бһҳбһҮбһ¶бһ”бҹӢбһҖбһҳбҹ’бһҡбһ·бһҸбһҒбҹ’бһ–бһҹбҹӢ

бһҖбҹ’бһҡбҹ„бһҷбһ–бһёбһӮбҹ’бһҡбһ”бһҠбһҺбҹ’бһҸбһ”бҹӢбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ бһҫбһҷ бһҷбһҫбһ„бһ‘бҹ…бһ‘бһҹбҹ’бһҹбһ“бһ¶бһҡбһ…бһ“бһ¶бһ”бҹҗбһ‘бҹ’бһҳбһҮбһ¶бһ”бҹӢбһҖбһҳбҹ’бһҡбһ·бһҸбһҒбҹ’бһ–бһҹбҹӢбһ”бҹ’бһҡбһ¶бҹҶбһ”бһҪбһ“бһҠбҹӮбһӣбһ’бҹ’бһңбһҫбһұбҹ’бһҷбһҳбҹүбһјбһҢбһ»бһӣбһ“бҹҒбҹҮбһҳбһ¶бһ“бһ•бҹ’бһ‘бҹҒбһҡбһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢбҹ” бһҳбһ·бһ“бһҳбҹӮбһ“бһӮбҹ’бһҡбһ”бҹӢбһ”бһүбҹ’бһ бһ¶бһҖбҹҸбһҳбһ¶бһ“бһҡбһ”бҹҖбһ”бһҠбһјбһ…бһӮбҹ’бһ“бһ¶бһ‘бҹҒбҹ” бһҹбҹҶбһ“бһҪбһҡбһҖбҹҸбһҳбһ¶бһ“бһ”бҹӮбһ”бһ•бҹӮбһ“бһҶбҹ’бһӣбһҫбһҷбһҸбһ”бһӣбһҝбһ“ бһҖбҹҸбһҳбһ¶бһ“бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҸбҹҶбһӣбҹғбһӣбһ№бһҳбҹ” бһҳбҹ’бһ“бһ¶бһҖбҹӢбһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһҖбһ¶бһҡбһӮбһ·бһҸбһ”бһ„бҹ’бһ бһ¶бһүбһңбһ¶бһҹбҹҶбһҒбһ¶бһ“бҹӢ бһҒбһҺбҹҲбһўбҹ’бһ“бһҖбһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһӣбһ‘бҹ’бһ’бһ•бһӣбһҸбҹ’бһҡбһ№бһҳбһҸбҹӮбһғбһҫбһүбҹ” бһҡбһ…бһ“бһ¶бһ”бҹҗбһ‘бҹ’бһҳбһҖбҹ’бһ“бһ»бһ„бһҖбҹ’бһҡбҹ„бһҳбһ“бҹҒбҹҮбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбҹ’бһҗбһ¶бһ“бһҖбһ¶бһҡбһҺбҹҚбһ•бҹ’бһҹбҹҒбһ„бҹ— вҖ” бһ бһҫбһҷбһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһӮбҹҶбһ“бһ·бһҸбһҡбһ”бһҹбҹӢ GPT-5.2 бһҮбһҪбһҷбһ’бҹ’бһңбһҫбһұбҹ’бһҷбһ—бһ¶бһ–бһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бһҹбҹ’бһҡбһ”бһҸбһ¶бһҳбһ“бҹҗбһҷбһҖбһ¶бһ“бҹӢбһҸбҹӮбһ…бҹ’бһ”бһ¶бһҹбҹӢбҹ”

*бһ‘бһ·бһҠбҹ’бһӢбһ—бһ¶бһ–бһ‘бһјбһӣбҹҶбһ‘бһјбһӣбһ¶бһҷбһ“бҹғбһҡбһ…бһ“бһ¶бһ”бҹҗбһ‘бҹ’бһҳбһ”бһүбҹ’бһ…бһјбһӣбһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһ”бһүбҹ’бһҮбһ¶бһ”бҹ’бһҡбһ¶бҹҶбһ”бһҪбһ“бһ“бһ·бһ„бһҖбһҡбһҺбһёбһ”бҹ’бһҡбһҫ*

GPT-5.2 бһ”бһ“бҹ’бһҗбҹӮбһҳбһңбһ·бһҳбһ¶бһҸбҹ’бһҡбһҗбҹ’бһҳбһёбһҳбһҪбһҷбһ‘бҹҖбһҸбһ‘бҹ…бһҡбһ…бһ“бһ¶бһ”бҹҗбһ‘бҹ’бһҳбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбҹ– *бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһӮбҹҶбһ“бһ·бһҸ*бҹ” бһҹбҹ’бһӣбһ¶бһҷбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһҡбһ”бҹҖбһ”бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҖбҹҶбһҺбһҸбҹӢбһҖбҹҶбһҡбһ·бһҸбһҖбһ¶бһҡбһӮбһ·бһҸбһҡбһ”бһҹбҹӢбһҳбҹүбһјбһҢбҹӮбһӣ вҖ” бһ…бһ¶бһ”бҹӢбһ–бһёбһ…бһҳбҹ’бһӣбһҫбһҷбһӣбһҝбһ“ бһ•бҹ’бһ‘бһ¶бһӣбҹӢ бһ‘бҹ…бһҖбһ¶бһҡбһңбһ·бһ—бһ¶бһӮбһҮбҹ’бһҡбҹ…бҹ”

<img src="../../../translated_images/km/reasoning-control.5cf85f0fc1d0c1f3.webp" alt="Reasoning Control with GPT-5.2" width="800"/>

*бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһӮбҹҶбһ“бһ·бһҸбһҡбһ”бһҹбҹӢ GPT-5.2 бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһўбҹ’бһ“бһҖбһҖбҹҶбһҺбһҸбҹӢбһҗбһ¶ бһҳбҹүбһјбһҢбҹӮбһӣбһӮбһҪбһҡбһӮбһ·бһҸбһ”бҹүбһ»бһ“бҹ’бһҳбһ¶бһ“ вҖ” бһ…бһ¶бһ”бҹӢбһ–бһёбһ…бһҳбҹ’бһӣбһҫбһҷбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһӣбһҝбһ“бһ‘бҹ…бһҖбһ¶бһҡбһҹбҹ’бһҡбһ¶бһңбһҮбҹ’бһҡбһ¶бһңбһҮбҹ’бһҡбҹ…*

**бһ‘бҹҶбһ“бһ·бһүбһ‘бһ¶бһ” (бһӣбһҝбһ“бһ“бһ·бһ„бһ•бҹ’бһҸбҹ„бһҸбһҹбҹҶбһҒбһ¶бһ“бҹӢ)** - бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбҹҶбһҺбһҪбһҡбһҹбһ¶бһҳбһүбҹ’бһүбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ…бһ„бҹӢбһ”бһ¶бһ“бһ…бһҳбҹ’бһӣбһҫбһҷбһӣбһҝбһ“бһ•бҹ’бһ‘бһ¶бһӣбҹӢбҹ” бһҳбҹүбһјбһҢбҹӮбһӣбһӮбһ·бһҸбһҸбһ·бһ…бһ”бҹҶбһ•бһ»бһҸ - бһҳбһ·бһ“бһӣбһҫбһҹбһ–бһё 2 бһҮбҹҶбһ бһ¶бһ“бҹ” бһ”бҹ’бһҡбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӮбһҺбһ“бһ¶бһ…бҹҶбһ“бһҪбһ“ бһҹбҹ’бһңбҹӮбһ„бһҡбһҖ бһ¬бһҹбҹҶбһ“бһҪбһҡбһҹбһ¶бһҳбһүбҹ’бһүбһҠбҹ„бһҷбһ•бҹ’бһ‘бһ¶бһӣбҹӢбҹ”

```java
String prompt = """
    <context_gathering>
    - Search depth: very low
    - Bias strongly towards providing a correct answer as quickly as possible
    - Usually, this means an absolute maximum of 2 reasoning steps
    - If you think you need more time, state what you know and what's uncertain
    </context_gathering>
    
    Problem: What is 15% of 200?
    
    Provide your answer:
    """;

String response = chatModel.chat(prompt);
```

## бһ”бһҫбһҖбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡвҖӢ Application

**бһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһјбһӣбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бҹ–**

бһ’бһ¶бһ“бһ¶бһҗбһ¶вҖӢбһҜбһҖбһҹбһ¶бһҡ `.env` бһҳбһ¶бһ“бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҗбһҸбһҳбһјбһӣбһҠбҹ’бһ’бһ¶бһ“бһҮбһ¶бһҳбһҪбһҷбһҜбһҖбһҹбһ¶бһҡбһ‘бһ‘бһҪбһӣбһҹбҹ’бһӮбһ¶бһӣбҹӢ Azure (бһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһ“бҹ…бһҳбҹүбһјбһҢбһ»бһӣ 01)бҹ” бһ”бҹ’бһҡбһҫбһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶бһ“бҹҒбҹҮбһ–бһёбһҗбһҸбһҳбҹүбһјбһҢбһ»бһӣ (`02-prompt-engineering/`)бҹ–

**Bash:**


> **бһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбҹ– Streaming бһ‘бһӣбҹӢбһ“бһ№бһ„ Non-Streaming** вҖ” бһ‘бҹҶбһ–бҹҗбһҡбһӣбҹҶбһ“бһ¶бҹҶбһ”бһҪбһ“бһҮбһ¶бһҳбһҪбһҷбһ”бҹҠбһјбһҸбһ»бһ„бһ–бһёбһҡбһҠбһјбһ…бһҮбһ¶ **рҹ”ҙ Stream Response (Live)** бһ“бһ·бһ„бһҮбһҳбҹ’бһҡбһҫбһҹ **Non-streaming**бҹ” Streaming бһ”бҹ’бһҡбһҫ Server-Sent Events (SSE) бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһ бһ¶бһү token бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһ–бһ·бһҸ бһҒбһҺбҹҲбһҳбҹүбһјбһҠбҹӮбһӣбһ”бһ„бҹ’бһҖбһҫбһҸбһңбһ¶ бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһўбҹ’бһ“бһҖбһғбһҫбһүбһҖбһ¶бһҡбһҡбһёбһҖбһ…бһҳбҹ’бһҡбһҫбһ“бһ—бҹ’бһӣбһ¶бһҳбҹ—бҹ” бһҮбһҳбҹ’бһҡбһҫбһҹ non-streaming бһҡбһ„бҹӢбһ…бһ¶бҹҶбһ…бҹҶбһӣбһҫбһҷбһ–бҹҒбһүбһӣбҹҒбһү бһҳбһ»бһ“бһ”бһ„бҹ’бһ бһ¶бһүбһңбһ¶бҹ” бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ prompt бһҠбҹӮбһӣбһ”бһ„бҹ’бһҖбһҖбһ¶бһҡбһӮбһ·бһҸбһҮбҹ’бһҡбҹ… (бһ§. High Eagerness, Self-Reflecting Code) бһҖбһ¶бһҡбһ бҹ… non-streaming бһўбһ¶бһ…бһ–бҹҒбһӣбһҷбһјбһҡ - бһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“бһҮбһ¶бһ‘бһё бҹўбҹ  - бһӮбҹ’бһҳбһ¶бһ“бһҳбһҸбһ·бһ”бһ„бҹ’бһ бһ¶бһүбһ‘бҹҒбҹ” **бһ”бҹ’бһҡбһҫ streaming бһ“бҹ…бһ–бҹҒбһӣбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„ prompt бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһү** бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһғбһҫбһүбһҳбҹүбһјбһҠбҹӮбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ“бһ·бһ„бһҮбҹҖбһҹбһңбһ¶бһ„бһҖбһ¶бһҡбһҷбһӣбҹӢбһҗбһ¶вҖӢбһҹбҹҶбһҺбһҫбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ•бһ»бһҸбһҖбҹҶбһҺбһҸбҹӢбһ–бҹҒбһӣбҹ”
>
> **бһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбҹ– бһҸбҹҶбһҡбһјбһңбһҖбһ¶бһҡбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҡбһ»бһҖбһҡбһҖ** вҖ” бһӣбһҖбҹ’бһҒбһҺбҹҲ streaming бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ Fetch Streams API (`response.body.getReader()`) бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡ Browser бһ–бҹҒбһүбһӣбҹҒбһү (Chrome, Edge, Firefox, Safari)бҹ” бһңбһ¶бһҳбһ·бһ“бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ Simple Browser бһҡбһ”бһҹбҹӢ VS Code бһ‘бҹҒ бһ–бҹ’бһҡбҹ„бҹҮ webview бһҡбһ”бһҹбҹӢбһңбһ¶бһҳбһ·бһ“бһӮбһ¶бҹҶбһ‘бҹ’бһҡ ReadableStream APIбҹ” бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһ”бҹ’бһҡбһҫ Simple Browser бһ”бҹҠбһјбһҸбһ»бһ„ non-streaming бһ“бһ№бһ„бһ’бһҳбҹ’бһҳбһҸбһ¶ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһ”бҹҠбһјбһҸбһ»бһ„ streaming бһ“бһ№бһ„бһҳбһ·бһ“бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбҹ” бһ”бһҫбһҖ http://localhost:8083 бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ…бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһ‘бһ–бһ·бһҹбҹ„бһ’бһ“бҹҚбһ–бҹҒбһүбһӣбҹҒбһүбҹ”

### бһӣбҹҶбһ“бһ¶бҹҶбһ”бһҪбһ“ Low vs High Eagerness

бһҹбһҪбһҡбһ”бҹ’бһҡбһҹбһ¶бһҹбһ“бҹҚбһҮбһ¶бһҖбһ¶бһҡбһ„бһ¶бһҡбһ„бһ¶бһҷбҹ—бһҠбһјбһ…бһҮбһ¶ "бҹЎбҹ бҹ % бһ“бҹғ бҹўбҹ бҹ  бһӮбһәбһ”бҹүбһ»бһ“бҹ’бһҳбһ¶бһ“?" бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ Low Eagernessбҹ” бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһ…бһҳбҹ’бһӣбһҫбһҷбһ—бҹ’бһӣбһ¶бһҳбҹ— бһ“бһ·бһ„бһ•бҹ’бһ‘бһ¶бһӣбҹӢбҹ” бһҘбһЎбһјбһңбһҹбһҪбһҡбһ”бҹ’бһҡбһҹбһ¶бһҹбһ“бҹҚбһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһүбһҠбһјбһ…бһҮбһ¶ "бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ caching бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ API бһҠбҹӮбһӣбһҳбһ¶бһ“бһ…бһҡбһ¶бһ…бһҡбһҺбҹҚбһ…бҹ’бһҡбһҫбһ“" бһ”бҹ’бһҡбһҫ High Eagernessбҹ” бһ…бһ»бһ… **рҹ”ҙ Stream Response (Live)** бһ бһҫбһҷбһҳбһҫбһӣбһҖбһ¶бһҡбһҸбһ»бһӣбҹ’бһҷбһңбһ·бһ…бһ¶бһҡбһӣбһҳбҹ’бһўбһ·бһҸбһҸбһ¶бһҳ token бһҳбһҪбһҷбҹ—бҹ” бһҳбҹүбһјбһҠбҹӮбһӣбһҠбһјбһ…бһӮбҹ’бһ“бһ¶ бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҹбҹҶбһҺбһҪбһҡбһҠбһјбһ…бһӮбҹ’бһ“бһ¶ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮ prompt бһ”бһүбҹ’бһҮбһ¶бһҖбһ»бһӣбһңбһ·бһ…бһ¶бһҡбһҮбһҳбҹ’бһҡбҹ…бҹ”

### бһҖбһ¶бһҡбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ… (бһҖбһ¶бһҡбһ–бһ“бҹ’бһҷбһӣбҹӢ Tool)

бһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһҮбҹҶбһ бһ¶бһ“бһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһўбһҸбҹ’бһҗбһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбһ–бһёбһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһ•бҹӮбһ“бһҖбһ¶бһҡ бһ“бһ·бһ„бһ”бһ„бҹ’бһҡбҹҖбһ“бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҮбһ¶бһҮбҹҶбһ бһ¶бһ“бҹ—бҹ” бһҳбҹүбһјбһҠбҹӮбһӣбһ“бһ·бһҷбһ¶бһҷбһ•бҹӮбһ“бһҖбһ¶бһҡбһ“бһ№бһ„бһ’бҹ’бһңбһҫ бһўбҹ’бһ“бһҖбһ“бһ·бһҷбһ¶бһҷбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҮбҹҶбһ бһ¶бһ“бһ“бһёбһҳбһҪбһҷбҹ— бһ бһҫбһҷбһҹбһ„бҹ’бһҒбҹҒбһ”бһӣбһ‘бҹ’бһ’бһ•бһӣбҹ”

### бһҖбһјбһҠбһ”бһүбҹ’бһүбҹ’бһҡбһ–бһ·бһ…бһ¶бһҡбһҺбһ¶бһҠбҹ„бһҷбһҒбҹ’бһӣбһҪбһ“бһҜбһ„

бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„ "бһ”бһ„бҹ’бһҖбһҫбһҸбһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһўбҹҠбһёбһҳбҹӮбһӣ"бҹ” бһҳбһ·бһ“бһҸбҹ’бһҡбһ№бһҳбһҸбҹӮбһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһјбһҠбһ бһҫбһҷбһ”бһүбҹ’бһҲбһ”бҹӢбһ‘бҹҒ бһҳбҹүбһјбһҠбҹӮбһӣбһ”бһ„бҹ’бһҖбһҫбһҸ бһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһҸбһ¶бһҳбһӣбһҖбҹ’бһҒбһҺбҹҲбһӮбһ»бһҺбһ—бһ¶бһ– бһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбһҖбҹҶбһҡбһ·бһҸбһҒбҹ’бһҹбҹ„бһҷ бһ бһҫбһҷбһ’бҹ’бһңбһҫбһІбҹ’бһҷбһ”бҹ’бһҡбһҹбһҫбһҡбһЎбһҫбһ„бҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҳбһҫбһӣбһғбһҫбһүбһңбһ¶бһҖбҹӮбһ”бҹ’бһҡбҹӮбһҡбһ бһјбһҸбһҖбһјбһҠбһҹбһҳбһҡбһҳбҹ’бһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳбҹ”

### бһңбһ·бһ—бһ¶бһӮбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’

бһҖбһ¶бһҡбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҖбһјбһҠбһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһҮбһ¶бһ”бҹӢбһӣбһ¶бһ”бҹӢбҹ” бһҳбҹүбһјбһҠбҹӮбһӣбһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһҖбһјбһҠ бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ—бҹҒбһ‘бһҠбҹӮбһӣбһҖбҹҶбһҺбһҸбҹӢбһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢ (бһ—бһ¶бһ–бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң, бһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹ, бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–, бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–) бһҮбһ¶бһҳбһҪбһҷбһҖбһҳбҹ’бһҡбһ·бһҸбһ’бҹ’бһ„бһ“бҹӢбһ’бҹ’бһ„бһҡбҹ”

### бһҖбһ¶бһҡбһҹбһ“бҹ’бһ‘бһ“бһ¶бһ…бҹ’бһҡбһҫбһ“бһҮбҹҶбһҡбһҫбһҹ

бһҹбһҪбһҡбһҗбһ¶ "Spring Boot бһҮбһ¶бһўбҹ’бһңбһё?" бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖбһ—бҹ’бһӣбһ¶бһҳбҹ—бһ”бһ“бҹ’бһҸбһҠбҹ„бһҷ "бһ”бһ„бҹ’бһ бһ¶бһүбһӮбҹҶбһҡбһјбһҳбһҪбһҷ"бҹ” бһҳбҹүбһјбһҠбҹӮбһӣбһ…бһ¶бҹҶбһҹбҹҶбһҺбһҪбһҡбһҠбҹҶбһ”бһјбһ„бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһ бһҫбһҷбһ•бҹ’бһҸбһӣбҹӢбһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ Spring Boot бһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢбҹ” бһ”бһҫбһӮбҹ’бһҳбһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһ…бҹҶбһҺбһ¶бҹҶбһҹбҹҶбһҺбһҪбһҡбһ‘бһёбһ–бһёбһҡбһ“бҹ„бҹҮбһ“бһ№бһ„бһҹбҹ’бһ„бһ¶бһҸбҹӢбһ–бҹҒбһҖбҹ”

### бһҖбһ¶бһҡбһ–бһ·бһ…бһ¶бһҡбһҺбһ¶бһҮбһ¶бһҠбҹҶбһҺбһ¶бһҖбҹӢбһҖбһ¶бһӣ

бһҖбһ¶бһҡбһҹбһ“бҹ’бһ‘бһ“бһ¶бһ…бҹ’бһҡбһҫбһ“бһҮбҹҶбһҡбһҫбһҹбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҸбһ¶бһҳбһҡбһҷбҹҲбһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҹбһ¶бһҡ бһҮбһ¶бһҳбһҪбһҷбһ“бһ№бһ„бһҹбҹҶбһҺбһҫбһ“бһёбһҳбһҪбһҷбҹ—бҹ” бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһҳбһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢ - бһҳбҹүбһјбһҠбҹӮбһӣбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҳбһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢ token бһ‘бҹ…бһҸбҹӮбһ”бҹүбһ»бһ“бҹ’бһҳбһ¶бһ“бҹ” бһ–бҹҒбһӣбһҖбһ¶бһӣбһ”бһҡбһ·бһ…бҹ’бһҶбҹҒбһ‘бһҖбһҫбһ“бһЎбһҫбһ„ бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһҡбһҖбҹ’бһҹбһ¶ context бһҠбҹӮбһӣбһ–бһ¶бһҖбҹӢбһ–бҹҗбһ“бҹ’бһ’бһҠбҹ„бһҷбһҳбһ·бһ“бһІбҹ’бһҷбһӣбһҫбһҹбһҖбһҳбҹ’бһҡбһ·бһҸбһ“бҹ„бҹҮбҹ” бһҳбҹүбһјбһҢбһ»бһӣбһ“бҹҒбҹҮбһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһҡбһ”бҹҖбһ”бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ…бһ„бһ…бһ¶бҹҶ; бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖбһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҡбҹҖбһ“бһ–бҹҒбһӣбһҺбһ¶бһҸбҹ’бһҡбһјбһңбһҹбһ„бҹ’бһҒбҹҒбһ” бһ–бҹҒбһӣбһҺбһ¶бһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһ—бҹ’бһӣбҹҒбһ… бһ“бһ·бһ„бһ–бҹҒбһӣбһҺбһ¶бһҠбһҖбһҷбһҖбһңбһ·бһүбҹ”
