# LangChain4j dla początkujących

Kurs tworzenia aplikacji AI z LangChain4j i Azure OpenAI GPT-5.2, od podstawowego czatu po agentów AI.

**Nowy w LangChain4j?** Sprawdź [Słownik](docs/GLOSSARY.md) by poznać definicje kluczowych terminów i pojęć.

## Spis treści

1. [Szybki start](00-quick-start/README.md) - Zacznij pracę z LangChain4j
2. [Wprowadzenie](01-introduction/README.md) - Poznaj podstawy LangChain4j
3. [Projektowanie promptów](02-prompt-engineering/README.md) - Opanuj skuteczne tworzenie promptów
4. [RAG (Generacja wspomagana wyszukiwaniem)](03-rag/README.md) - Buduj inteligentne systemy oparte na wiedzy
5. [Narzędzia](04-tools/README.md) - Integruj narzędzia zewnętrzne i proste asystenty
6. [MCP (Model Context Protocol)](05-mcp/README.md) - Pracuj z protokołem Model Context i modułami Agentic
---

## Ścieżka nauki

> **Szybki start**

1. Zrób fork tego repozytorium do swojego konta na GitHubie
2. Kliknij **Code** → zakładka **Codespaces** → **...** → **New with options...**
3. Użyj domyślnych ustawień – wybierze to kontener deweloperski stworzony na potrzeby tego kursu
4. Kliknij **Create codespace**
5. Poczekaj 5-10 minut na przygotowanie środowiska
6. Przejdź od razu do [Szybkiego startu](./00-quick-start/README.md) i zacznij naukę!

Po ukończeniu modułów, zapoznaj się z [Poradnikiem testowania](docs/TESTING.md), aby zobaczyć w praktyce koncepcje testowania LangChain4j.

> **Uwaga:** Szkolenie wykorzystuje zarówno GitHub Models, jak i Azure OpenAI. Moduł [Szybki start](00-quick-start/README.md) korzysta z GitHub Models (nie wymaga subskrypcji Azure), natomiast moduły 1-5 używają Azure OpenAI. Zacznij od [BEZPŁATNEGO konta Azure](https://aka.ms/azure-free-account), jeśli jeszcze go nie masz.


## Nauka z GitHub Copilot

Aby szybko zacząć pisać kod, otwórz ten projekt w GitHub Codespace lub w swoim lokalnym IDE z dostarczonym devcontainerem. Devcontainer używany w tym kursie jest wstępnie skonfigurowany z GitHub Copilot do programowania wspieranego AI.

Każdy przykład kodu zawiera sugerowane pytania, które możesz zadać GitHub Copilot, by pogłębić swoją wiedzę. Szukaj oznaczeń 💡/🤖 w:

- **Nagłówkach plików Java** - pytania specyficzne dla każdego przykładu
- **README modułów** - pytania do eksploracji po przykładach kodu

**Jak korzystać:** Otwórz dowolny plik z kodem i zadaj Copilotowi podane pytania. Dysponuje on pełnym kontekstem kodu i potrafi go wyjaśnić, rozbudować lub zaproponować alternatywy.

Chcesz dowiedzieć się więcej? Sprawdź [Copilot do programowania wspieranego AI](https://aka.ms/GitHubCopilotAI).
