# LangChain4j dla Początkujących

Kurs budowania aplikacji AI z LangChain4j i Azure OpenAI GPT-5, od podstawowego czatu do agentów AI.

**Nowy w LangChain4j?** Sprawdź [Słownik](docs/GLOSSARY.md) z definicjami kluczowych terminów i koncepcji.

## Spis Treści

1. [Szybki Start](00-quick-start/README.md) - Rozpocznij pracę z LangChain4j
2. [Wprowadzenie](01-introduction/README.md) - Poznaj podstawy LangChain4j
3. [Tworzenie Promptów](02-prompt-engineering/README.md) - Opanuj skuteczne projektowanie promptów
4. [RAG (Retrieval-Augmented Generation)](03-rag/README.md) - Buduj inteligentne systemy oparte na wiedzy
5. [Narzędzia](04-tools/README.md) - Integruj narzędzia zewnętrzne i proste asystenty
6. [MCP (Model Context Protocol)](05-mcp/README.md) - Pracuj z Modelem Kontekstowym i modułami Agentic
---

## Ścieżka Nauki

> **Szybki Start**

1. Zrób fork tego repozytorium na swoje konto GitHub
2. Kliknij **Code** → zakładka **Codespaces** → **...** → **New with options...**
3. Użyj ustawień domyślnych – to wybierze kontener deweloperski stworzony dla tego kursu
4. Kliknij **Create codespace**
5. Poczekaj 5-10 minut na przygotowanie środowiska
6. Przejdź bezpośrednio do [Szybkiego Startu](./00-quick-start/README.md), aby zacząć!

Po ukończeniu modułów poznaj [Przewodnik Testowania](docs/TESTING.md), aby zobaczyć w praktyce koncepcje testowania LangChain4j.

> **Uwaga:** Ten trening korzysta zarówno z modeli GitHub, jak i Azure OpenAI. Moduł [Szybki Start](00-quick-start/README.md) używa modeli GitHub (nie wymaga subskrypcji Azure), natomiast moduły 1-5 korzystają z Azure OpenAI. Zacznij z [DARMOWYM kontem Azure](https://aka.ms/azure-free-account), jeśli jeszcze go nie masz.


## Nauka z GitHub Copilot

Aby szybko rozpocząć pisanie kodu, otwórz ten projekt w GitHub Codespace lub w swoim lokalnym IDE z dostarczonym devcontainer. Devcontainer używany w tym kursie jest wstępnie skonfigurowany z GitHub Copilot do współprogramowania AI.

Każdy przykład kodu zawiera sugerowane pytania, które możesz zadać GitHub Copilot, aby pogłębić zrozumienie. Szukaj podpowiedzi 💡/🤖 w:

- **Nagłówkach plików Java** - pytania specyficzne dla każdego przykładu
- **Plikach README modułów** - pytania exploracyjne po przykładach kodu

**Jak używać:** Otwórz dowolny plik kodu i zadaj Copilot proponowane pytania. Copilot ma pełen kontekst kodu i może wyjaśniać, rozszerzać i sugerować alternatywy.

Chcesz dowiedzieć się więcej? Sprawdź [Copilot do współprogramowania AI](https://aka.ms/GitHubCopilotAI).
