# LangChain4j pradedantiesiems

Kursas apie AI programų kūrimą su LangChain4j ir Azure OpenAI GPT-5.2, nuo pagrindinio pokalbio iki AI agentų.

**Naujas LangChain4j?** Susipažinkite su [Žodynu](docs/GLOSSARY.md) apie pagrindines sąvokas ir terminus.

## Turinys

1. [Greitas pradžia](00-quick-start/README.md) - Pradėkite naudotis LangChain4j
2. [Įvadas](01-introduction/README.md) - Sužinokite pagrindus apie LangChain4j
3. [Užklausų inžinerija](02-prompt-engineering/README.md) - Įvaldykite efektyvų užklausų kūrimą
4. [RAG (Retrieval-Augmented Generation)](03-rag/README.md) - Kurkite intelektualias žinių pagrindu veikiančias sistemas
5. [Įrankiai](04-tools/README.md) - Integruokite išorinius įrankius ir paprastus asistentus
6. [MCP (Model Context Protocol)](05-mcp/README.md) - Dirbkite su Model Context Protocol (MCP) ir agentiškais moduliais
---

## Mokymosi kelias

> **Greitas pradžia**

1. Padarykite šios saugyklos fork'ą į savo GitHub paskyrą
2. Spustelėkite **Code** → skirtuką **Codespaces** → **...** → **New with options...**
3. Naudokite numatytuosius nustatymus – tai pasirinks šiame kurse sukurtą Development konteinerį
4. Spustelėkite **Create codespace**
5. Palaukite 5-10 minučių, kol aplinka bus paruošta
6. Tiesiogiai pereikite į [Greitą pradžią](./00-quick-start/README.md) ir pradėkite!

Baigę modulius, apžvelkite [Testavimo gidą](docs/TESTING.md) ir pamatykite LangChain4j testavimo koncepcijas veikiant.

> **Pastaba:** Ši mokymo programa naudoja tiek GitHub Models, tiek Azure OpenAI. Modulis [Greita pradžia](00-quick-start/README.md) naudoja GitHub Models (nereikalauja Azure prenumeratos), o moduliai 1–5 naudoja Azure OpenAI. Pradėkite su [NEMOKAMA Azure paskyra](https://aka.ms/azure-free-account), jei jos neturite.


## Mokymasis su GitHub Copilot

Norėdami greitai pradėti rašyti kodą, atidarykite šį projektą GitHub Codespace arba savo vietinėje IDE su pateiktu devcontainer. Šiame kurse naudojamas devcontainer yra iš anksto sukonfigūruotas su GitHub Copilot AI poriniam programavimui.

Kiekviename kodo pavyzdyje yra siūlomų klausimų, kuriuos galite užduoti GitHub Copilot, kad gilintumėte supratimą. Ieškokite 💡/🤖 užuominų:

- **Java failų antraštėse** – Klausimai, specifiniai kiekvienam pavyzdžiui
- **Modulių README bylose** – Eksploracijos klausimai po kodų pavyzdžių

**Kaip naudotis:** Atidarykite bet kurį kodo failą ir užduokite Copilot siūlomus klausimus. Jis turi pilną kodo bazės kontekstą ir gali paaiškinti, išplėsti bei pasiūlyti alternatyvas.

Norite sužinoti daugiau? Peržiūrėkite [Copilot AI Poriniam Programavimui](https://aka.ms/GitHubCopilotAI).
