# LangChain4j 新手指南

一門使用 LangChain4j 和 Azure OpenAI GPT-5.2 從基礎聊天到 AI 代理應用開發的課程。

**初次接觸 LangChain4j？** 請參閱[術語表](docs/GLOSSARY.md)了解關鍵術語和概念的定義。

## 目錄

1. [快速入門](00-quick-start/README.md) - 開始使用 LangChain4j
2. [簡介](01-introduction/README.md) - 學習 LangChain4j 基礎
3. [Prompt 工程](02-prompt-engineering/README.md) - 掌握有效的提示設計
4. [RAG (檢索增強生成)](03-rag/README.md) - 建立智慧知識型系統
5. [工具](04-tools/README.md) - 整合外部工具和簡易助理
6. [MCP (模型上下文協定)](05-mcp/README.md) - 使用模型上下文協定 (MCP) 和 Agentic 模組
---

## 學習路徑

> **快速入門**

1. 將此存儲庫分叉到你的 GitHub 帳戶
2. 點擊 **Code** → **Codespaces** 分頁 → **...** → **帶選項的新建...**
3. 使用預設值 – 這會選擇為本課程建立的開發容器
4. 點擊 **Create codespace**
5. 等待 5-10 分鐘，環境即準備完成
6. 直接跳轉到[快速入門](./00-quick-start/README.md)開始吧！

完成模組後，可探索[測試指南](docs/TESTING.md)以了解 LangChain4j 的測試概念實作。

> **注意：** 本訓練同時使用 GitHub 模型和 Azure OpenAI。[快速入門](00-quick-start/README.md)模組使用 GitHub 模型（無需 Azure 訂閱），而模組 1-5 使用 Azure OpenAI。若未有帳戶，請使用[免費 Azure 帳號](https://aka.ms/azure-free-account)開始。

## 與 GitHub Copilot 一起學習

若要快速開始編程，請在 GitHub Codespace 或本地 IDE 中開啟此專案，並使用提供的 devcontainer。本課程所用的 devcontainer 已預先配置 GitHub Copilot，支援 AI 配對程式設計。

每個程式範例都包含建議向 GitHub Copilot 詢問的問題，幫助你深化理解。在以下部份尋找 💡/🤖 提示：

- **Java 文件標頭** - 每個範例特定的問題
- **模組 README** - 程式範例後的探索問題

**使用方法：** 打開任一程式檔案並向 Copilot 詢問建議問題。它擁有完整的程式碼庫上下文，可以解釋、延伸及建議替代方案。

想要了解更多？請參考 [AI 配對程式設計的 Copilot](https://aka.ms/GitHubCopilotAI)。
