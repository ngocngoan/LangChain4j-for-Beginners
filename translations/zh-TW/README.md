# LangChain4j 初學者課程

使用 LangChain4j 與 Azure OpenAI GPT-5 架構 AI 應用，從基礎聊天到 AI 代理的完整課程。

**LangChain4j 新手？** 請參考 [詞彙表](docs/GLOSSARY.md) 以了解主要術語和概念的定義。

## 目錄

1. [快速開始](00-quick-start/README.md) - 快速上手 LangChain4j
2. [介紹](01-introduction/README.md) - 學習 LangChain4j 的基礎知識
3. [提示工程](02-prompt-engineering/README.md) - 精通有效的提示設計
4. [RAG（檢索增強生成）](03-rag/README.md) - 建構智慧型知識系統
5. [工具](04-tools/README.md) - 整合外部工具與簡易助理
6. [MCP（模型上下文協議）](05-mcp/README.md) - 使用模型上下文協議與代理模組
---

## 學習路徑

> **快速開始**

1. 將本倉庫 Fork 到您的 GitHub 帳號
2. 點擊 **Code** → **Codespaces** 分頁 → **...** → **New with options...**
3. 使用預設值 – 這會選擇本課程所採用的開發容器
4. 點擊 **Create codespace**
5. 等待 5-10 分鐘讓環境就緒
6. 直接跳至 [快速開始](./00-quick-start/README.md) 開始學習！

完成所有模組後，可探索 [測試指南](docs/TESTING.md)，瞭解 LangChain4j 的測試概念與應用。

> **注意：** 本訓練使用 GitHub 模型與 Azure OpenAI。快速開始模組使用 GitHub 模型（不需 Azure 訂閱），而 1-5 模組使用 Azure OpenAI。若尚未有 Azure 帳號，可使用 [免費 Azure 帳戶](https://aka.ms/azure-free-account) 立即開始。


## 使用 GitHub Copilot 學習

想迅速開始撰寫程式碼，請在 GitHub Codespace 或本機 IDE 中開啟本專案 (使用課程提供的 devcontainer)。本課程中 devcontainer 預先設定好 GitHub Copilot 以支援 AI 配對程式設計。

每個程式碼示範均包含您可以問 GitHub Copilot 的建議問題以深化理解。在以下位置尋找 💡/🤖 提示：

- **Java 檔案標頭** — 針對該示例的特定問題
- **模組 README** — 程式碼示範後的探索問題

**使用方式：** 開啟任一程式檔案，向 Copilot 詢問建議問題。Copilot 具備完整碼庫上下文，能解釋、擴充並提供替代方案。

想進一步了解？請參考 [Copilot AI 配對程式設計](https://aka.ms/GitHubCopilotAI)。
