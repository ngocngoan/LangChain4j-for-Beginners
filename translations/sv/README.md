# LangChain4j för nybörjare

En kurs för att bygga AI-applikationer med LangChain4j och Azure OpenAI GPT-5.2, från grundläggande chatt till AI-agenter.

**Ny på LangChain4j?** Kolla in [Glossariet](docs/GLOSSARY.md) för definitioner av viktiga termer och begrepp.

## Innehållsförteckning

1. [Snabbstart](00-quick-start/README.md) - Kom igång med LangChain4j
2. [Introduktion](01-introduction/README.md) - Lär dig grunderna i LangChain4j
3. [Promptdesign](02-prompt-engineering/README.md) - Bemästra effektiv promptdesign
4. [RAG (Retrieval-Augmented Generation)](03-rag/README.md) - Bygg intelligenta kunskapsbaserade system
5. [Verktyg](04-tools/README.md) - Integrera externa verktyg och enkla assistenter
6. [MCP (Model Context Protocol)](05-mcp/README.md) - Arbeta med Model Context Protocol (MCP) och Agentiska moduler
---

##  Lärväg

> **Snabbstart**

1. Forka detta arkiv till ditt GitHub-konto
2. Klicka **Code** → fliken **Codespaces** → **...** → **New with options...**
3. Använd standardinställningarna – detta kommer välja utvecklingscontainern skapad för denna kurs
4. Klicka **Create codespace**
5. Vänta 5-10 minuter tills miljön är redo
6. Hoppa direkt till [Snabbstart](./00-quick-start/README.md) för att komma igång!

Efter att ha slutfört modulerna, utforska [Testningsguiden](docs/TESTING.md) för att se LangChain4j:s testkoncept i praktiken.

> **Obs:** Denna träning använder både GitHub-modeller och Azure OpenAI. [Snabbstart](00-quick-start/README.md)-modulen använder GitHub-modeller (ingen Azure-prenumeration krävs), medan modulerna 1-5 använder Azure OpenAI. Kom igång med ett [GRATIS Azure-konto](https://aka.ms/azure-free-account) om du inte har ett.


## Lära med GitHub Copilot

För att snabbt börja koda, öppna detta projekt i en GitHub Codespace eller din lokala IDE med den medföljande devcontainern. Devcontainern som används i denna kurs är förkonfigurerad med GitHub Copilot för AI-parprogrammering.

Varje kodexempel innehåller föreslagna frågor som du kan ställa till GitHub Copilot för att fördjupa din förståelse. Leta efter 💡/🤖-uppmaningar i:

- **Java-filhuvuden** - Frågor som är specifika för varje exempel
- **Modulernas README-filer** - Utforskningsuppmaningar efter kodexempel

**Hur man använder:** Öppna valfri kodfil och ställ de föreslagna frågorna till Copilot. Den har fullständig kontext om kodbasen och kan förklara, utöka och föreslå alternativ.

Vill du lära dig mer? Kolla in [Copilot for AI Paired Programming](https://aka.ms/GitHubCopilotAI).
