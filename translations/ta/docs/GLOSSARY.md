## முக்கியக் கருத்துக்கள்

**ஏ.ஐ எஜென்ட்** - சுயாதீனமாக காரணம்கொண்டு செயல்படும் ஏ.ஐ பயன்படுத்தும் அமைப்பு. [தொகுதி 04](../04-tools/README.md)

**ச்செருக்கு** - ஒருமுறை செயற்பாடுகளின் தொடர், வெளிப்பாடு அடுத்த படிக்கு வழிகாட்டும்.

**துண்டாக்கல்** - ஆவணங்களை சிறிய துண்டுகளாகப் புரண்டல். சாதாரணமாக: 300-500 டோக்கன்கள் ஒட்டிச்சேர்க்கையுடன். [தொகுதி 03](../03-rag/README.md)

**சூழல் ஜன்னல்** - மாதிரி செயல்படுத்தும் கூடிய அதிகபட்ச டோக்கன்கள். GPT-5.2: 400K டோக்கன்கள்.

**மூலாங்கங்கள்** - உரையின் அர்த்தத்தை பிரதிநிதி செய்யும் எண் வெக்டர். [தொகுதி 03](../03-rag/README.md)

**இயக்கம் அழைக்கல்** - மாதிரி வெளிப்பாடு அமைப்போடை கோரிக்கைகளை உருவாக்கி வெளியீடு செய்கிறது. [தொகுதி 04](../04-tools/README.md)

**கற்பனை** - மாதிரிகள் தவறான ஆனால் நம்பத்தகுந்த தகவலை உருவாக்கும் போது.

**வினா** - மொழி மாதிரிக்கு உள்ளீடு வாக்கியம். [தொகுதி 02](../02-prompt-engineering/README.md)

**அர்த்தவித்திரவு தேடல்** - உரை அர்த்தம் கொண்டு தேடல் செய்கிறது, சொல் அடிப்படையில் அல்ல. [தொகுதி 03](../03-rag/README.md)

**நிலைமையுடன் எதிர் நிலைமையற்ற** - நிலைமையற்ற: நினைவில்லை. நிலைமையுடன்: உரையாடல் வரலாற்றை வைத்துக்கொள்கிறது. [தொகுதி 01](../01-introduction/README.md)

**டோக்கன்கள்** - மாதிரிகள் செயலாக்கும் அடிப்படை உரை அலகுகள். செலவுகள் மற்றும் வரம்புகளை பாதிக்கும். [தொகுதி 01](../01-introduction/README.md)

**கருவி ச்செருக்கு** - தொடர் கருவி செயலாக்கம், வெளிப்பாடு அடுத்த அழைப்பை அறிவுறுத்தும். [தொகுதி 04](../04-tools/README.md)

## LangChain4j கூறுகள்

**AiServices** - வகை-பாதுகாப்பான ஏ.ஐ சேவை இடைமுகங்களை உருவாக்குகிறது.

**OpenAiOfficialChatModel** - OpenAI மற்றும் Azure OpenAI மாதிரிகளுக்கான ஒருங்கிணைந்த கிளையண்ட்.

**OpenAiOfficialEmbeddingModel** - OpenAI அதிகாரபூர்வ கிளையண்ட் மூலம் மூலாங்கங்கள் உருவாக்குகிறது (OpenAI மற்றும் Azure OpenAI இரண்டும் ஆதரிக்கிறது).

**ChatModel** - மொழி மாதிரிகளுக்கான மைய இடைமுகம்.

**ChatMemory** - உரையாடல் வரலாற்றை பராமரிக்கும்.

**ContentRetriever** - RAGக்கான தொடர்புடைய ஆவண துண்டுகளை கண்டறியும்.

**DocumentSplitter** - ஆவணங்களை துண்டுகளாகப் பிரிக்கும்.

**EmbeddingModel** - உரையை எண் வெக்டர்களாக மாற்றும்.

**EmbeddingStore** - மூலாங்கங்களை சேமிக்கும் மற்றும் மீட்டெடுக்கிறது.

**MessageWindowChatMemory** - சமீபத்திய செய்திகளின் திரை அழைபதை பராமரிக்கும்.

**PromptTemplate** - `{{variable}}` இடமாற்றிகளுடன் மீண்டும் பயன்படக்கூடிய வினாக்களை உருவாக்குகிறது.

**TextSegment** - மெட்டாடேட்டாவுடன் கூடிய உரை துண்டு. RAG இல் பயன்படும்.

**ToolExecutionRequest** - கருவி செயலாக்க கோரிக்கை பிரதிநிதித்துவம்.

**UserMessage / AiMessage / SystemMessage** - உரையாடல் செய்தி வகைகள்.

## AI/ML கருத்துக்கள்

**சிறு மாதிரிகளுடன் கற்றல்** - வினாக்களில் எடுத்துக்காட்டுகளை வழங்குதல். [தொகுதி 02](../02-prompt-engineering/README.md)

**பெரிய மொழி மாதிரி (LLM)** - பெரும் உரை தரவுக்கு பயிற்சி பெற்ற AI மாதிரிகள்.

**கருத்து முயற்சி** - GPT-5.2 ஆழமான யோசனையை கட்டுப்படுத்தும் அளவுரு. [தொகுதி 02](../02-prompt-engineering/README.md)

**உஷ்ணம்** - வெளிப்பாட்டில் ஆதூசியத்தை கட்டுப்படுத்துகிறது. குறைந்த=தீர்மானமான, அதிக=சிறந்த கற்பனை.

**வெக்டர் தரவுத்தளம்** - மூலாங்கங்களுக்கு சிறப்பான தரவுத்தளம். [தொகுதி 03](../03-rag/README.md)

**பூஜ்ய மாதிரி கற்றல்** - எடுத்துக்காட்டுகள் இல்லாமல் பணிகளை செய்யுதல். [தொகுதி 02](../02-prompt-engineering/README.md)

## குர்ரைல்கள் - [தொகுதி 00](../00-quick-start/README.md)

**ஆழமான பாதுகாப்பு** - பயன்பாட்டுக் குர்ரைல்கள் மற்றும் வழங்குநர் பாதுகாப்பு வடிகட்டிகளோடு கூடிய பல அடுக்கு பாதுகாப்பு.

**கட்டாய தடுப்பு** - கடுமையான உள்ளடக்க மீறல்களுக்கு வழங்குநர் HTTP 400 பிழை ஏற்படுத்தும்.

**InputGuardrail** - LLM-க்கு செல்லும் முன் பயனர் உள்ளீட்டை சரிபார்க்க LangChain4j இடைமுகம். தீங்கு விளைவிக்கும் வினாக்களை ஆரம்பத்தில் தடுக்கும். செலவு மற்றும் தாமதத்தை குறைக்கும்.

**InputGuardrailResult** - குர்ரைல் சரிபார்ப்பின் திரும்பும் வகை: `success()` அல்லது `fatal("reason")`.

**OutputGuardrail** - பயாரர்களுக்கு திருப்பும் முன் ஏ.ஐ பதில்களை சரிபார்க்க இடைமுகம்.

**Provider Safety Filters** - AI வழங்குநர்களின் (உதா. GitHub மாதிரிகள்) API மட்டத்தில் மீறல்களை பிடிக்கும் உள்ளமைந்த உள்ளடக்க வடிகட்டிகள்.

**Soft Refusal** - மாதிரி பிழை காட்டாமை பதிலளிப்பதை மரியாதையுடன் மறுத்தல்.

## அறிமுகக் கையேடு - [தொகுதி 02](../02-prompt-engineering/README.md)

**சிந்தனைச் சங்கிலி** - கூடிய துல்லியத்துக்கான படிநிலை காரணக்கூற்று.

**முறைபடியான வெளிப்பாடு** - குறிப்பிட்ட வடிவமைப்பு அல்லது அமைப்பை கடைப்பிடித்தல்.

**அதிக எண்ணம்** - GPT-5.2 முழு காரணக்கூற்று மாதிரி.

**குறைந்த எண்ணம்** - GPT-5.2 விரைவான பதிலுக்கு மாதிரி.

**பன்முறை உரையாடல்** - பரிமாற்றங்களில் சூழலை பராமரிப்பு.

**பாத்திர அடிப்படையிலான வினாக்கள்** - அமைப்பு செய்திகளால் மாதிரி பாவனையை அமைத்தல்.

**சுய பரிசீலனை** - மாதிரி வெளியீட்டை மதிப்பீடு செய்து மேம்படுத்தல்.

**உருவமைப்புச் செய்தியியல்** - நிலையான மதிப்பீடு கட்டமைப்பு.

**பணிசெயல் மாதிரி** - திட்டமிடு → செயல்படுத்து → சுருக்கு.

## RAG (உதவியுடன் திரட்டல் உருவாக்கம்) - [தொகுதி 03](../03-rag/README.md)

**ஆவணம் செயலாக்க குழாய்** - ஏற்றவும் → துண்டாக்கவும் → மூலாங்கு → சேமி.

**நினைவகத்தில் மூலாங்கு சேமிப்பு** - சோதனைக்கான தற்காலிக சேமிப்பு.

**RAG** - பதிலுக்கு அடிப்படையாக திரட்டலை உட்கொண்டு உருவாக்க.launch.

**தெறிவு மதிப்பீடு** - அர்த்தவித்திரவுத் துல்லியத்தின் அளவு (0-1).

**மூலக் குறிப்பு** - திரட்டிய உள்ளடக்கத்தின் மெட்டாடேட்டா.

## எஜென்ட்கள் மற்றும் கருவிகள் - [தொகுதி 04](../04-tools/README.md)

**@Tool குறிச்சொல்** - ஜாவா முறைகளை AI அழைக்கக்கூடிய கருவிகள் என்று குறிக்கிறது.

**ReAct மாதிரி** - கருதல் → செயல்பாடு → கவனித்தல் → மீண்டும்.

**அமர்வு மேலாண்மை** - வெவ்வேறு பயனர்களுக்கான தனித்தனியான சூழல்கள்.

**கருவி** - AI எஜென்ட் அழைக்கும் செயல்பாடு.

**கருவி விளக்கம்** - கருவி நோக்கம் மற்றும் அளவுருக்கள் ஆவணம்.

## எஜென்டிக் தொகுதி - [தொகுதி 05](../05-mcp/README.md)

**@Agent குறிச்சொல்** - எஜென்ட் பண்பாட்டைக் குறிக்கும் இடைமுகங்களை குறிக்கிறது.

**எஜென்ட் கேட்டுக்கோள்** - `beforeAgentInvocation()` மற்றும் `afterAgentInvocation()` வழியாக செயல்பாட்டை கண்காணிக்கும் ஹூக்.

**எஜென்டிக் பரப்பளவு** - வெளியீடுகளை சேமிக்கும் பகிரங்க நினைவகம். `outputKey` மூலம் தொடர்புடைய எஜென்ட்கள் பயன்படுத்துகின்றன.

**AgenticServices** - `agentBuilder()` மற்றும் `supervisorBuilder()` மூலம் எஜென்ட்களை உருவாக்கும் தொழிற்சாலை.

**பயன்பாடுகளுக்கு மாறுதல் வழி** - விதிகளின் அடிப்படையில் வேறு நிபுணர் எஜென்ட்களுக்கான வழி.

**மனிதர் பங்கு** - ஒப்புதல் அல்லது உள்ளடக்க ஆய்விற்கு மனிதர்கள் உள்ளடக்கிய வேலை நடைமுறை.

**langchain4j-agentic** - அறிவியல் கொள்கையின் அடிப்படையிலான எஜென்ட்கள் கட்டுமானத்திற்கு மைவன் சார்பு (சோதனை கட்டம்).

**முழுமைச் சுழற்சி வேலை** - ஒரு நிபந்தனை (உதா. தரம்கோள் ≥ 0.8) முழுக்கும் வரை எஜென்ட் நடத்தை மீண்டும் செய்யவும்.

**outputKey** - எஜென்ட் குறிச்சொல் அளவுரு; எஜென்டிக் பரப்பளவில் முடிவுகள் சேம்யிடும் இடம்.

**சமமுறை வேலை** - பல எஜென்ட்களை சுதந்திர பணிகளுக்கு ஒரே நேரத்தில் இயக்கு.

**பதில் முதன்மை** - மேற்பார்வையாளர் இறுதி பதிலை அமைக்கும் முறை: கடைசி, தொகுப்பு, மதிப்பிடப்பட்ட.

**வரிசைப்படுத்தப்பட்ட வேலை** - வெளியீடு அடுத்த படிக்கு செல்லும் படி எஜென்ட்களை வரிசைப்படுத்தி செயல்படுத்து.

**மேற்பார்வையாளர் எஜென்ட் மாதிரி** - மேற்பார்வையாளர் LLM எந்த துணை எஜென்ட்களை அழைக்க தீர்மானிக்கும் மேம்பட்ட சக்தி மிக்க மாதிரி.

## மாதிரி சூழல் நெறிமுறை (MCP) - [தொகுதி 05](../05-mcp/README.md)

**langchain4j-mcp** - LangChain4j இல் MCP இணைப்புக்கு மைவன் சார்பு.

**MCP** - மாதிரி சூழல் நெறிமுறை: AI பயன்பாடுகளை வெளிப்புற கருவிகளுடன் இணைக்கும் தரநிலை. ஒருமுறை கட்டி, எங்கும் பயன்படுத்து.

**MCP கிளையண்ட்** - MCP சேவகங்களைச் சேர்ந்து கருவிகளை கண்டறிந்து பயன்படுத்தும் பயன்பாடு.

**MCP சேவகர்** - தெளிவான விளக்கங்கள் மற்றும் அளவுரு திட்டங்களுடன் MCP வழியாக கருவிகளை வெளிப்படுத்தும் சேவை.

**McpToolProvider** - LangChain4j கூறு; MCP கருவிகளை ஏ.ஐ சேவைகள் மற்றும் எஜென்ட்களில் பயன்படத் தருகிறது.

**McpTransport** - MCP தொடர்புக்கான இடைமுகம். நடைமுறைகள் Stdio மற்றும் HTTP அடக்கம்.

**Stdio போக்குவரத்து** - உள்ளக செயலி போக்குவரத்தை stdin/stdout மூலம். கோப்பக அணுகல் அல்லது கட்டளை நடவடிக்கைகளுக்கு பயனுள்ளது.

**StdioMcpTransport** - MCP சேவகத்தை துணைசெயல்முறையாக தோற்றுவிக்கும் LangChain4j நடைமுறை.

**கருவி கண்டறிதல்** - கிளையண்ட் கருவிகள் மற்றும் அவர்களின் விளக்கங்களுடன் சேவகரை விசாரிக்கும்.
