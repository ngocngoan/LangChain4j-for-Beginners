## शिकण्याचा मार्ग

> **जलद प्रारंभ**

1. हा संग्रह आपला GitHub खात्यावर Fork करा
2. क्लिक करा **Code** → **Codespaces** टॅब → **...** → **New with options...**
3. मूळ निवडी वापरा – यामुळे या कोर्ससाठी तयार केलेला विकास कंटेनर निवडला जाईल
4. क्लिक करा **Create codespace**
5. पर्यावरण तयार होईपर्यंत ५-१० मिनिटे थांबा
6. थेट [जलद प्रारंभ](./00-quick-start/README.md) येथे जा आणि प्रारंभी करा!

मॉड्यूल पूर्ण केल्यावर, LangChain4j च्या चाचणी संकल्पनांचा अनुभव घ्या यासाठी [चाचणी मार्गदर्शक](docs/TESTING.md) पहा.

> **टीप:** ही प्रशिक्षण विधी GitHub मॉडेल्स आणि Azure OpenAI दोन्ही वापरते. [जलद प्रारंभ](00-quick-start/README.md) मॉड्यूल GitHub मॉडेल्स वापरते (Azure सदस्यता आवश्यक नाही), तर मॉड्यूल १-५ Azure OpenAI वापरतात. [मुफ्त Azure खाते](https://aka.ms/azure-free-account) नसेल तर सुरुवात करा.


## GitHub Copilot सह शिक्षण

त्वरित कोडिंग सुरू करण्यासाठी, हा प्रकल्प GitHub Codespace मध्ये किंवा स्थानिक IDE मध्ये devcontainer सह उघडा. या कोर्समध्ये वापरलेला devcontainer GitHub Copilot सह AI जोडलेले प्रोग्रामिंगसाठी पूर्व-संयोजित आहे.

प्रत्येक कोड उदाहरणामध्ये GitHub Copilot कडून विचारू शकणारे शिफारसीत प्रश्न समाविष्ट आहेत जे आपली समझ वाढवतील. 💡/🤖 चिह्नांनी पहा:

- **Java फाईल हेडर्स** - प्रत्येक उदाहरणाशी संबंधित प्रश्न
- **मॉड्यूल README फाईल्स** - कोड उदाहरणांनंतर अन्वेषणासाठी प्रश्न

**कसे वापरायचे:** कोणतीही कोड फाईल उघडा आणि Copilot कडे सूचित प्रश्न विचारा. याला कोडबेसचा पूर्ण संदर्भ आहे आणि तो समजावू, विस्तारीत करू, आणि पर्यायी सुचना देऊ शकतो.

अधिक जाणून घ्यायचे आहे? [AI जोडलेले प्रोग्रामिंगसाठी Copilot](https://aka.ms/GitHubCopilotAI) पहा.
