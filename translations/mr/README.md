# नवशिक्यांसाठी LangChain4j

LangChain4j आणि Azure OpenAI GPT-5 सह AI अनुप्रयोग तयार करण्यासाठी एक कोर्स, मूलभूत चॅटपासून AI एजंटपर्यंत.

**LangChain4j मध्ये नवीन आहात?** महत्त्वाच्या संज्ञा आणि संकल्पनांसाठी [शब्दसंग्रह](docs/GLOSSARY.md) बघा.

## अनुक्रमणिका

1. [जलद प्रारंभ](00-quick-start/README.md) - LangChain4j सह सुरू करा
2. [परिचय](01-introduction/README.md) - LangChain4j चे मूलतत्वे समजून घ्या
3. [प्रॉम्प्ट इंजिनीअरिंग](02-prompt-engineering/README.md) - प्रभावी प्रॉम्प्ट डिझाइनमध्ये निपुण व्हा
4. [RAG (Retrieval-Augmented Generation)](03-rag/README.md) - बुद्धिमान ज्ञानाधारित प्रणाली तयार करा
5. [टूल्स](04-tools/README.md) - बाह्य साधने आणि साधे सहाय्यक समाकलित करा
6. [MCP (मॉडेल कॉन्टेक्स्ट प्रोटोकॉल)](05-mcp/README.md) - मॉडेल कॉन्टेक्स्ट प्रोटोकॉल (MCP) आणि एजंट मॉड्यूल्सवर काम करा

## शिकण्याचा मार्ग

> **जलद प्रारंभ**

1. हा रेपॉझिटरी आपल्या GitHub खात्यावर Fork करा
2. **Code** → **Codespaces** टॅब → **...** → **New with options...** क्लिक करा
3. डिफॉल्ट निवडा – या कोर्ससाठी तयार केलेला Development container निवडला जाईल
4. **Create codespace** क्लिक करा
5. वातावरण तयार होईपर्यंत 5-10 मिनिटे थांबा
6. सुरू करण्यासाठी थेट [जलद प्रारंभ](./00-quick-start/README.md) वर जा!

मॉड्यूल पूर्ण केल्यानंतर, LangChain4j च्या चाचणी संकल्पना अंमलात पाहण्यासाठी [चाचणी मार्गदर्शिका](docs/TESTING.md) एक्सप्लोर करा.

> **टीप:** हे प्रशिक्षण GitHub मॉडेल्स आणि Azure OpenAI दोन्ही वापरते. [जलद प्रारंभ](00-quick-start/README.md) मॉड्यूल GitHub मॉडेल्स वापरतो (कुठल्याही Azure सदस्यत्वाशिवाय), तर 1-5 मॉड्यूलसाठी Azure OpenAI वापरला जातो. आपल्याकडे Azure खाते नसेल तर [मुफत Azure खाते](https://aka.ms/azure-free-account) वापरून प्रारंभ करा.


## GitHub Copilot सह शिकणं

त्वरित कोडिंग सुरू करण्यासाठी, हा प्रकल्प GitHub Codespace किंवा आपल्या स्थानिक IDE मध्ये प्रदान केलेले devcontainer वापरून उघडा. या कोर्समध्ये वापरलेला devcontainer GitHub Copilot सह AI जोडणी प्रोग्रामिंगसाठी पूर्व-संरचित आहे.

प्रत्येक कोड उदाहरणास GitHub Copilot कडून विचरता येणाऱ्या प्रश्नांसह दिलेले आहेत जे आपली समज अधिक खोल करू शकतात. 💡/🤖 प्रॉम्प्टसाठी बघा:

- **Java फाइल हेडर्स** - प्रत्येक उदाहरणासाठी विशिष्ट प्रश्न
- **मॉड्यूल रीडमी फायली** - कोड उदाहरणांनंतर तपासणी प्रश्न

**कसे वापरायचे:** कोणतीही कोड फाइल उघडा आणि Copilot कडे सूचित प्रश्न विचारा. त्याच्याकडे कोडबेसचे पूर्ण संदर्भ असतो आणि तो समजावू, विस्तार करू, आणि पर्याय सुचवू शकतो.

अधिक जाणून घ्यायचे आहे? [AI जोडणी प्रोग्रामिंगसाठी Copilot](https://aka.ms/GitHubCopilotAI) बघा.
