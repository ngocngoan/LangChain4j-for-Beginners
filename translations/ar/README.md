# LangChain4j للمبتدئين

دورة لبناء تطبيقات الذكاء الاصطناعي باستخدام LangChain4j و Azure OpenAI GPT-5.2، من الدردشة الأساسية إلى وكلاء الذكاء الاصطناعي.

**جديد على LangChain4j؟** اطلع على [المصطلحات](docs/GLOSSARY.md) لتعريفات المصطلحات والمفاهيم الرئيسية.

## جدول المحتويات

1. [البدء السريع](00-quick-start/README.md) - ابدأ مع LangChain4j
2. [مقدمة](01-introduction/README.md) - تعلّم أساسيات LangChain4j
3. [هندسة المطالبات](02-prompt-engineering/README.md) - اتقن تصميم المطالبات الفعّالة
4. [RAG (الجيل المعزز بالاسترجاع)](03-rag/README.md) - بناء أنظمة معرفية ذكية
5. [الأدوات](04-tools/README.md) - دمج الأدوات الخارجية والمساعدين البسيطين
6. [MCP (بروتوكول سياق النموذج)](05-mcp/README.md) - العمل مع بروتوكول سياق النموذج (MCP) والوحدات الوكلائية

## مسار التعلم

> **البدء السريع**

1. قم بتفريغ هذا المستودع إلى حساب GitHub الخاص بك
2. انقر على **كود** → علامة تبويب **بيئات الشيفرة** → **...** → **جديد مع الخيارات...**
3. استخدم الإعدادات الافتراضية – سيحدد هذا حاوية التطوير التي تم إنشاؤها لهذه الدورة
4. انقر على **إنشاء بيئة الشيفرة**
5. انتظر من 5 إلى 10 دقائق حتى تكون البيئة جاهزة
6. انتقل مباشرة إلى [البدء السريع](./00-quick-start/README.md) للبدء!

بعد إكمال الوحدات، استكشف [دليل الاختبار](docs/TESTING.md) لرؤية مفاهيم اختبار LangChain4j عمليًا.

> **ملاحظة:** يستخدم هذا التدريب كل من نماذج GitHub و Azure OpenAI. تستخدم وحدة [البدء السريع](00-quick-start/README.md) نماذج GitHub (لا حاجة لاشتراك Azure)، بينما تستخدم الوحدات 1-5 Azure OpenAI. ابدأ باستخدام [حساب Azure مجاني](https://aka.ms/azure-free-account) إذا لم يكن لديك واحد.


## التعلم مع GitHub Copilot

لبدء الترميز بسرعة، افتح هذا المشروع في بيئة Codespace لـ GitHub أو محرر IDE المحلي الخاص بك باستخدام devcontainer المزود. يحتوي devcontainer المستخدم في هذه الدورة على GitHub Copilot مهيأ مسبقًا للبرمجة المزدوجة باستخدام الذكاء الاصطناعي.

يشمل كل مثال شفرة أسئلة مقترحة يمكنك طرحها على GitHub Copilot لتعميق فهمك. ابحث عن الإشارات 💡/🤖 في:

- **رؤوس ملفات Java** - أسئلة محددة لكل مثال
- **ملفات README للوحدات** - مقترحات استكشافية بعد أمثلة الشفرة

**كيفية الاستخدام:** افتح أي ملف شفرة واطرح على Copilot الأسئلة المقترحة. لديه سياق كامل لقاعدة الشفرة ويمكنه الشرح، التوسيع، واقتراح البدائل.

هل تريد معرفة المزيد؟ اطلع على [Copilot للبرمجة المزدوجة باستخدام الذكاء الاصطناعي](https://aka.ms/GitHubCopilotAI).
