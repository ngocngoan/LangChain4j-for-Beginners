# LangChain4j pentru Începători

Un curs pentru construirea aplicațiilor AI cu LangChain4j și Azure OpenAI GPT-5.2, de la chat de bază la agenți AI.

**Ești nou la LangChain4j?** Consultă [Glosarul](docs/GLOSSARY.md) pentru definiții ale termenilor și conceptelor cheie.

## Cuprins

1. [Pornire rapidă](00-quick-start/README.md) - Începe cu LangChain4j
2. [Introducere](01-introduction/README.md) - Învață fundamentele LangChain4j
3. [Ingineria prompt-urilor](02-prompt-engineering/README.md) - Stăpânește proiectarea eficientă a prompt-urilor
4. [RAG (Generare augmentată cu căutare)](03-rag/README.md) - Construiește sisteme inteligente bazate pe cunoștințe
5. [Unele Instrumente](04-tools/README.md) - Integrează instrumente externe și asistenți simpli
6. [MCP (Protocolul Contextului Modelului)](05-mcp/README.md) - Lucrează cu Protocolul Contextului Modelului (MCP) și module Agentic
---

## Parcurs de Învățare

> **Pornire rapidă**

1. Fă un fork acestui depozit în contul tău GitHub
2. Apasă pe **Code** → fila **Codespaces** → **...** → **New with options...**
3. Folosește setările implicite – acest lucru va selecta containerul de dezvoltare creat pentru acest curs
4. Apasă **Create codespace**
5. Așteaptă 5-10 minute până când mediul este gata
6. Mergi direct la [Pornire rapidă](./00-quick-start/README.md) pentru a începe!

După ce finalizezi modulele, explorează [Ghidul de Testare](docs/TESTING.md) pentru a vedea concepte legate de testarea LangChain4j în practică.

> **Notă:** Acest antrenament folosește atât Modele GitHub, cât și Azure OpenAI. Modulul [Pornire rapidă](00-quick-start/README.md) folosește Modele GitHub (nu este necesar abonament Azure), în timp ce modulele 1-5 folosesc Azure OpenAI. Începe cu un [cont GRATUIT Azure](https://aka.ms/azure-free-account) dacă nu ai deja unul.


## Învățare cu GitHub Copilot

Pentru a începe rapid să codezi, deschide acest proiect într-un GitHub Codespace sau în IDE-ul tău local cu devcontainer-ul furnizat. Devcontainer-ul folosit în acest curs vine preconfigurat cu GitHub Copilot pentru programare AI în pereche.

Fiecare exemplu de cod include întrebări sugerate care pot fi adresate GitHub Copilot pentru a aprofunda înțelegerea. Caută prompturile 💡/🤖 în:

- **Header-ele fișierelor Java** - Întrebări specifice fiecărui exemplu
- **README-urile modulelor** - Prompturi de explorare după exemplele de cod

**Cum se folosește:** Deschide orice fișier de cod și pune întrebările sugerate lui Copilot. Acesta are context complet al bazei de cod și poate explica, extinde și sugera alternative.

Vrei să afli mai multe? Consultă [Copilot pentru programare AI în pereche](https://aka.ms/GitHubCopilotAI).
